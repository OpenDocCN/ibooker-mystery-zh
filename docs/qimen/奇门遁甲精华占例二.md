## 为杜永汉南行做生意预测实例

1998年12月10日早上8点20分，我的一个名字叫杜永汉的朋友找到我，说求测一件大事。进屋后他坐离位。杜永汉是1973年出生的。我按其前来求测的时间起局如下：

戊寅年 甲子月 辛卯日 壬辰时 阳4局
值符：甲申庚 天心星 值使：开门

| 太 阴 | 六 合 | 白 虎 |
| :---: | :---: | :---: |
| 壬 戊 任生 | 乙 癸 冲伤 | 戊 丙 辅杜 |
| 螣 蛇 | 己 禽 | 玄 武 |
| 丁 乙 蓬休 |  | 癸 辛 英景 |
| 值 符 | 九 天 | 九 地 |
| 庚 壬 心开 | 辛 丁 柱惊 | 丙 庚 芮死 |

杜永汉坐离位，也没有说测什么事，只是告诉我要做一件大事，让我测测吉凶。

我审视了一下奇门局，根据杜永汉所坐离位及其年命所落宫位，十分肯定地对他讲：“你最近要出一趟远门，是往南方走，对吧？”

“你的易经越来越准了，我真的是要去南方，去的是广州，你能测出我去广州做什么吗？”杜有意来考我。

我观察了一下时辰干支壬辰所在的巽宫，对杜讲：“你想去广州赚钱。”“你又测对了！实话跟你讲吧，我已组织好了几车皮水果，两边的差价很大，如果正常运作的话，这次会稳稳当当地赚上一笔。”杜永汉很自信地告诉我。

## 判断思路：

杜坐离位，日干辛落坎宫，坎宫旺相冲动离宫，冲则必动。杜年命癸落兑宫，被日支卯宫所冲，又是动的符号。因此，断杜永汉要出一次远门。时辰为测来意的重要依据，今时辰干支壬辰均在巽宫，巽宫地盘为戊，门盘为生门，都是财的信息，故断杜要外出求财。断杜要去南方：因其本身就坐在离宫，离宫地盘又是其年命癸之故。

“请你给我预测一下，这次去广州能赚多少钱？整个运输过程顺利否？”杜永汉问。

“如果你信我，这次最好不去广州。因为这次去了之后你的货卖不上好价位。搞不好要赔钱的。”我给杜敲警钟。

“不会吧？我已经提前派人前去考察过了，那边价格比这边好多了，而且货供不应求，我这次主要发的货是李子和葡萄，马上就快到元旦了，往年这个时候水果都会买上好价的。”杜疑惑地问。

“如果你非要去的话，谁也拦不住。但这次南行的结果是：李子全部卖掉了，葡萄可就卖得不好，没准有一部分剩了还得拉回来。”我又从奇门局中看出一些细节。

“现在货都备齐了，如果不去广州，就得在家边批发出去，这样赚的就太少了。”杜不甘心。

“那就别怪我没提醒你，总之这次南行你肯定会后悔的。”我坚持自己的意见。

杜永汉没有听我的，当天下午就启程了，走了四车货，二车李子、二车葡萄。到了广州销了几天，很不理想，李子全部卖掉了，赚了些钱，可葡萄卖得很不好，最后舍本卖都出不了手。无奈之下只好拉回来，算算总账，本钱刚刚回来，吃住费用还不算在内。回到瓦房店之后见到我便说：“真后悔当初没听你的劝，不然我不会这么惨的，来回走了半个月时间，一分钱没挣回来，白搭了路费、时间、还有那么多人力。这一趟广州跑得太不值了。”

## 判断思路：

在这个实例中，我取用神的方法与传统的不一样。传统的取用神方法是：日干为卖方，时干为货物。而我在这里把离九宫做为李子来看，把坤二宫做为葡萄来看。

为什么这样取用神呢？按传统的时干为用神的话，现在有二种货：一种李子，一种是葡萄。时干到底代表李子，还是代表葡萄？若再遇上同时运输五种、十种货物，一个时干能代表那么细致吗？

显然不能，所以可以肯定地讲：传统的取用神方法是不够精确的，也很难应付复杂的求测与判断。这里我取离九宫代表李子的理由是：

1. 离九宫的格局为乙十癸，乙为木，癸为水为“子”，而木加子正是“李”字。离九宫为天冲星、伤门，全是木，又离九宫为“九紫”，正是李子的颜色。
2. 坤二宫为天辅星，“辅”与“葡”字有谐音。象形之说。故取坤二宫天辅星代表葡萄。

从全局大象来判断，戊为本金，落宫在坤，生门为利润，落巽宫，巽宫克坤宫，利润克本金，故不宜投资。在这里我以杜的年命癸为求测者，今癸落兑七宫。离九宫为李子，离宫克兑宫，货克人，离九宫又逢空亡，所以断李子全部卖掉，价格也较好（生门宫生离九宫，离宫又生成所落坤二宫）。

坤二宫天辅星为葡萄，今来生杜年命癸所落之兑宫，货恋人，故难以出手。坤宫格局为戊十丙青龙返首，所以卖不出又拉回来（返首主返回）。这个实例的关键点在于：辛卯日午未空亡，离九宫空亡，但坤二宫不空亡。因为未为阴支，而坤宫为戊十丙，都为阳，按《奇门法窍》一书中的规则，阳与阳比，阴与阴比，坤宫并不空亡，不空即葡萄出不了手。

## 为谢先生测官运升迁

2000年11月23日晚间19点40分，一男子前来敲门，因较晚了我本不想开门，他说有急事，无奈只能让他进来。进屋后他坐坤位，经介绍，姓谢，1961年（辛丑）出生。我按他前来求测时间起局如下：

庚辰年 丁亥月 乙酉日 丙戌时 阴5局
值符：甲申庚 天冲星 值使：伤门

| | | |
| :--- | :--- | :--- |
| 六 合 | 太 阴 | 螣 蛇 |
| 乙 己 | 壬 癸 | 丁 辛 |
| 心死 | 蓬惊 | 任开 |
| 白 虎 | 值 符 | 九 天 |
| 丙 庚 | 戊 禽 | 庚 丙 |
| 柱景 | | 冲休 |
| 玄 武 | 九 地 | |
| 辛 戌 丁 | 癸 壬 | 己 乙 |
| 芮禽杜 | 英伤 | 辅生 |

我看奇门局是九星返吟，返吟主快，说明谢的确遇到了急事。正想仔细推敲一下整个奇门局，坐在坤位的谢先生主动递给我一支香烟，平时我不吸烟，于是拒绝，可是谢不肯，并说：“这是中华烟，您就抽一支吧！”随即他已点燃了火。

出于礼貌，我吸了他送的中华烟。当时我坐坎位。谢长长地吐了一口烟，说：“您能看出我来测什么事情吗？”

我已经从他坐的方位及给我递烟点烟的过程，知道了他的来意：“你今天这么晚来预测，主要是为了工作调动之事来求测的。单位里有个空缺位置，你想调去，但这个位置竞争很激烈，很多同事都想去，现在你心里没有把握，想给大领导送礼打点一下，又担心落空，是这样吧？”我单刀直入。

“刘老师您太准了！我现在是副局级，单位正局位置现在正空着，很多人都看中这位置，我已经是40岁的人了，如果这次上不去的话，这一生再没有机会了。您仔细给看看，如果我下点功夫运作一番（送礼），能否上去？”谢先生问。

## 判断思路：

谢先生进屋后坐坤位，正是马星位置（寅午戌马在申），坤宫乘天任星、开门。九星返吟，返吟主变动（相当于六爻纳甲中的六冲卦），开门主工作、位置、官运等符号。又谢先生年命辛丑落艮宫，正与其座位坤宫对冲，亦是调动之信息。奇门中有时辰为事体之说，今时干丙落震三宫，被值符所落之兑七宫冲动，也是调动之象。这么多“动”向，说明谢先生必是为工作调动之事前来预测的。开门主工作，今坤宫值空亡，而谢坐坤宫，说明单位有一空位，其想去这个位置。因谢先生年命辛落坤宫地盘。地盘主内心深处想法。月干丁也落坤宫，月干代表同行竞争者，说明这个空位竞争很激烈，很多人都想去这个位置。

断谢先生想去单位领导家送礼打点一下：谢坐坤宫，生值符兑七宫，其年命辛在艮八宫，与戌同宫来生值符兑七宫，都是要给领导送礼的信息，戌代表钱财等。再者就是谢向我求测，我坐坎位，正是值使门伤门所落宫。谢送给我中华烟，我不吸烟，谢再次送过来，并点燃火，这都是给领导送礼的信息。

其实谢先生的意图很明确：如果有机会升上去，那么就努力一下，给领导送点礼。如果没有上去的可能，那也就不去破费了。他自己信心不足。

我把全局分析了一番，最后给谢先生吃了定心丸：“你放心大胆去运作打点吧，最后这个空位一定非你莫属，别人条件都不够。事成之后你可要请客！”我跟谢先生开玩笑地说着。

“您放心！如果这次我真的升上去的话，一定重重地感谢你，找个最好的饭店请您的客！”

后来的事情发展如我所测，谢先生不久便被提升为正局长。

## 判断思路：

测官运除了传统的开门代表文官，杜门代表武官以外，我重点看值符、值使，因为值符、值使就是权力的象征。在这个局里，现在的值符为甲申庚、天冲星；下一轮值符是甲午辛、天芮星。今谢先生年命辛乘天芮星落艮八宫，其年命支丑亦在艮八宫，这说明待这轮值符值事完毕之后，下一轮必然是“辛”做值符值事掌权了。值符为全局之主宰，当然是正位了。由此十分肯定地断谢先生必升迁。

## 为郑刚先生断终身运

2005年农历正月初九傍晚6点左右，一对夫妇前来求测，经介绍，男的是1962年出生，名字叫郑刚。女的姓曹，1963年出生。

进屋后郑刚坐巽位，曹女士坐离位。我按时间起局：

乙酉年 戊寅月 壬申日 己酉时 阳2局
值符： 甲辰壬 天心星 值使： 开门

| | | |
| :--- | :--- | :--- |
| 螣 蛇 | 太 阴 | 六 合 |
| 乙 庚 | 丁 辛 丙 | 己 戊 |
| 蓬死 | 任禽惊 | 冲开 |
| 值 符 | 值 符 | 白 虎 |
| 壬 己 | 辛 | 庚 癸 |
| 心景 | 禽 | 辅休 |
| 九 天 | 九 地 | 玄 武 |
| 癸 丁 | 戊 乙 | 丙 壬 |
| 柱杜 | 芮伤 | 英生 |

郑刚见我写好了，便说：“今天来这儿，主要是想让你把我这个人一生的运气给测一测，我的出生年月日记得，但时辰父母已记不清楚了，您能否给推出时辰来？”

“我不用生辰八字一样可以为你断一生的，请放心。”我对郑刚说。

分析了一下全局，结合郑刚及曹女士坐宫，我便下断语：“你是官场上做官的，现在是正职。对吧？”

只见郑刚与曹女士相互对视了一下，然后回答：“您测得对，继续往下说。”

## 判断思路：

郑刚年命壬乘天心星加临值符落震三宫，值符代表领导。又其坐巽宫，巽宫天盘乙奇，乙又为太岁（乙酉年），也为当官之符号。

我又接着断：“你小时候8岁那年秋天上的学，9岁就当班干部了。14岁那年，应该是在初中吧，也是当班干部，而且应该是主要班委。”

“太准了！我是69年秋天上的小学，那年正好8虚岁。上小学二年级时，因为我体育好，老师就让我做班里的体育委员。上初二那年，被老师任命为团支部书记，是1975年的事，应该是14岁吧。”郑刚很兴奋地回答着。

## 判断思路：

郑刚是壬寅年生人，年命寅在艮宫起一岁，震宫二岁，巽宫三岁、四岁……兑宫为八岁，兑宫乘天辅文曲星，正冲动郑刚年命壬所落之震三宫，兑宫地盘为甲寅癸，寅正是郑刚年命。故断他8岁（1969年）上学，兑宫主秋天。9岁在乾宫，乾宫地盘为壬，正是郑刚的年命壬，同时壬又落值符宫，壬又为日干，乾主官，所以断郑刚9岁当班干部。体育委员者，应乾宫为天心星武曲星之故。断郑刚14岁那年当班干部，而且是主要班委——14岁流年宫位走在震宫，应在1975乙卯年。震宫天盘壬乘天心星值符，壬为日干，又是值符旬首，且又为郑刚年命。所以断这一年当班委，值符当然是主要班委了。

接下来我又断郑刚父亲有大灾，应该是在郑刚20岁那年。郑刚马上回答：“非常准确！在1981年清明节前后，我父亲遭遇了车祸，后来经抢救无效，去世了。”

## 判断思路：

以日干壬为求测人郑刚，庚为郑刚父亲。今天盘庚落兑宫，为自身羊刃宫（庚羊刃在酉），格局为庚加癸大格，正冲着值符所在之震三宫，震宫又有日干壬；地盘庚在巽四宫，临死门凶门，蓬星凶星。以上这些都是父亲有灾的最明显的信息符号。应1981年者，因1981年是辛酉年，兑宫为酉之故，又兑宫乘白虎凶神，白虎主道路，庚也为道路，震三宫为车辆符号，值符宫一般都是动态的。震兑相冲，故主车灾。再者，81年年干辛落离宫，惊门亦在离宫，惊门为兑，主酉，离宫克庚所落兑宫，也主辛酉年父亲有灾。

接下来我又继续下断语：“你父亲是个有学问的人，但是很清高，比较抗上，和领导处不来，但群众对他还是很拥护的。”

郑刚验证道：“你说的一点没错，我父亲是工程师，只是和单位领导总有摩擦，原因很简单，单位领导不懂技术，总是乱指挥，父亲就和领导火起来。但是领导最终还得给我父亲赔理道歉。工人们对我父亲非常尊敬。”

## 判断思路：

断郑刚父亲是个有学问之人：庚乘天辅星之故；断郑刚父亲清高、抗上：庚在兑宫直接冲克值符震三宫；断父亲受群众拥护：时干己落坤二宫乘六合来生庚之兑宫。

我又对郑刚说：“尽管你在学校里成绩不错，但最终你没有读大学。”郑刚回答：“因为当时我家里经济条件不是很好，在高一时因身体突然患病而不得不休学一年，待到重新返回学校想好好学习一番参加高考的时候，父亲又突然出了车祸，家里整个塌了天，这时的我根本没有精力再去参加高考了。”

## 判断思路：

郑刚年命壬在震三宫，临值符，但天辅星临庚在兑七宫冲克震宫。兑七宫是今年太岁酉的位置。太岁主高等学府，酉又为亚魁星，也主高等学府、考学之类的信息。这些信息都来冲克壬之震宫，故断郑刚没能读大学。断郑在校学习成绩不错——震宫乘景门生丁奇所落之离宫，景门为考试题目，丁奇为考生答卷。

观察了一番乾宫的信息，我对郑刚讲：“1982年，你参加了工作。同年有人给介绍了女朋友，女朋友家经济条件很好，但你们没有相处成功，分手了。”

郑刚回答道：“我是1982年秋天参加工作的。上班三个月左右，便有人上门提亲。女方家经济条件很好。我们相处了二个多月时间，因性格不合分手了。刘老师测得太准了！”

## 判断思路：

1982年是壬戌年，乾宫地支有戌，乾宫地盘又临壬，壬既是1982年年干，同时又是郑刚年命壬，乾宫为开门本位，开门主工作，开门所乘暗干壬是日干，也是郑刚年命，故断1982年参加工作。乾宫之生门所乘暗干为丁，丁与壬合，故断这一年处女朋友。因当时为寅月，寅月生门旺相，故断女方经济条件很好。壬申日戌亥空亡，即乾宫空亡，故双方没有谈成，分手了。又郑刚坐在巽位，正被乾宫所冲克，故乾宫的信息对郑刚不利。

接下来我又从艮宫的组合找出了新的断语：“在你24岁那一年（1985年），经人介绍又相处一个女朋友，这个女人性格比较稳重、内向，戴了一副眼镜。你们俩相处了二年又分手了，是你主动提出分手的。”

“活灵活现！刘老师测得太准了！1985年年末，单位同事给我介绍一位1963年出生的女朋友，属兔的，她戴一副近视镜，人也稳重，话语不多，当我们相处到1987年时，正商量结婚的事情，女方父母在经济上对我们家要求很苛刻，双方僵持起来，最后终因我们家经济条件不是很好，难以满足对方的过分要求，无奈只能分手。”郑刚十分感慨地讲述着。

“苦尽甜来，1987年虽然失恋了，但工作上你却另有收获，这一年你被提升了，对吧！”我从震三宫的组合又读取出更新的信息。

## 关于十干克应的解析

古籍中有关奇门遁甲十干克应的解释多牵强，令人费解。尤其在实际预测中运用起来极不方便，使很多奇门爱好者见到众多的克应断语难以适从，因为这些断语太多，根本记不住，故多数易友认为奇门遁甲太难学，诸多的克应背不住，从而望而却步。

其实十干克应并非如古人所造的那样，下了一大堆的断语，然后死背下来，每次占测时套用一番，得出某种吉或凶的结论。这都是研究奇门的误区。

十干克应无非就是天盘干与地盘干的一种组合，凡是组合都表达一种具体的象。我们没有必要去死记古人搞的那些固定断语，只要我们掌握了这些格局形成的基本原理，便可举一反三地从一种格局中派生出更多的附合易理的断语来。现举一例子来阐明。

如天盘甲子戊加地丙奇，名为青龙返首，吉格，古书中这样讲：

> > 六甲加丙名青龙返首，凡阴阳二遁此时可造。百事吉。若更合奇门，利攻战，出行最为良。

现代社会并非战争年代，如果预测时遇上这种格局，总不能给人说“利攻战，出行最为良”吧！

其实这个格局的组合很简单：天盘甲子戊加地盘丙奇，这种组合之所以名曰青龙返首，主要因为甲为青龙，丙为甲木之子，甲木生丙火，为父母回首顾盼子女之象。这种组合在现代生活中可以有多种释义：

-   1. 父母在为子女的生存忙个不停，或父母在为子女的工作跑来跑去。为什么这样来解释呢？因为甲在天盘，天盘主动，丙在地盘，地盘主静，甲生丙，故有此断语，在这个组合里，甲是付出者，丙是受益者。因甲木要生丙火，甲木生丙火之后自身能量受到耗泄，而丙火得到甲木的生助后能量增加，故这种格局是对子女有利的格局。甲为丙之父母，父母代表工作，故有为子女的工作跑来跑去之象。

这个组合利主不利客，如果一个人前者来占问子女的状况得此格局，则轻易不要断吉，很可能是在为子女的事操心、忙碌、付出。

如果是一个中学生来求测，则可以断这个学生有个很有能力的父亲或母亲在帮助他（她）搞好学习。如果是一个单位的领导前来预测，得到青龙返首这个格局，则可以断这个领导对手下的员工非常照顾体贴，重视员工的切身利益，经常为员工们做些福利事业等等。

通过上边对青龙返首这个格局的详解，相信奇门爱好者们对传统的格局会有一个新的认识和理解，至于其它的常用吉凶格局的具体详解，将在下一章做详细解析。

## 奇门问答

> >问：如何看待奇门局中的值符、值使？它们在预测过程中有何具体作用和意义？
>
> 答：值符是占事时辰的旬首，统领十个时辰内的生杀大权。也就是说，在值符所管辖的十个时辰内，它的权力是最大的，不论旺衰，值符所落之宫是最有权力的宫位，它类似于六爻纳甲中的动爻，可以克制一切的静爻，而静爻再旺相，也难以制服动爻。在一个起好的奇门局中，值符所落之宫是动态的，而其它宫位（除值使门所落之宫）都是相对静态的。值符所落之宫可以对九宫中任何一宫产生生、克、制、耗、泄等作用。而其它任何一宫（包括值使门所在之宫）都不能对值符宫产生生、克、制、耗、泄的作用。即使旺相的宫位，也难以克制休囚的值符所在之宫。或者说，任何一个宫来生、克、制、耗、泄值符之宫都是徒劳的，只能消耗自身宫位的能量，值符宫的能量是不受损的。

值使是一个时辰内的主宰，但值使必须在值符的控制下行使自身权力，值使也是动态的，值使门所在之宫位可以对除了值符宫之外的任何一宫行使生、克、制、耗、泄的权力。其它任何一宫（除值符宫）都不能对值使宫行使生、克、制、耗、泄的权力。即使旺相宫也不行。

值符宫可以生、克、制、耗、泄值使宫，而值使宫不能生、克、制、耗、泄值符宫。因值符宫的权力大于值使宫之故。

局中最喜值符宫生扶值使宫，或值符宫与值使宫比和。

最忌值符宫克值使宫、值符宫耗泄值使宫、值使宫克值符宫。
