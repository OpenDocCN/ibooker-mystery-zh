# 曾子南的龍脈

曾子南是地理宗師楊救貧仙師第一弟子，唐朝國師贛州曾文迪大明師的直系第五房第五十一代後裔。大家知道地理明師出在江西，但江西的明師就是贛州曾家。因曾家的地理自曾文迪受傳楊公三元地理及三元命理後，歷代秘傳迄今，從不傳授真訣給外人，父傳子，子傳孫，傳授自己子孫，都要焚香立誓，方予傳授。

曾子南九歲開始接受名聞全國的大明師祖父曾連山的指導研究三元地理；為了學習地理，隨祖父跑遍大江南北各省，見多識廣，經驗豐富，故人稱之為「百貨公司」，工夫到家，是當今堪輿界唯一權威，有獨無偶。曾子南工夫超人一等，精通尋龍點穴，卜宅葬墳，尤精陽宅，能使人人速發福澤，求財得財，求子得子，有病治病，有訟治訟，且家學淵源，著書二十余部（書名另刊內），工夫之高明，由此可知。

曾子南為了擁護總統復興中華文化，發揚堪輿學術，先後創立堪輿學會總會、臺灣省中國堪輿學會、臺北市萬象，且多社會賢達聞人，歡迎大家共結善緣，保證有很大益處。

仁人孝子卜地造宅葬墳，既相信地理，且欲求福得福，就要聘請明師主庚，方能如願以償。如果聘請一知半解的時師主庚，就不能求福得福的，那祇有碰運氣了。明師卜宅葬墳，速發福澤；時師卜宅葬墳，禍福無憑，全靠東家運氣而定。

曾子南尋龍尋福龍，點地點吉地，定向定旺向，擇日即擇富貴日課，為人造福，能使人速發福澤，人家不平安得財，神乎其技，到處都做有見證，鐵的事實，人人知道。

陳故副總統墓園，前國防部長四星上將顧祝同的父墳及岳父墳，前臺灣省主席四星上將周至柔的母墳，今臺灣省主席四星上將陳大慶的岳母墳，陳立夫的父墳，前外貿會主任，前委員尹仲容墳，及改宅後大發福澤的不知其數，難以枚舉！

曾子南為當今大明師，是人人公認的，但從來都不擺明師的架子，和藹可親，祇要有事去求見他，無分貴賤富貴，他一視同仁，優禮接待，並竭誠為人服務，故有「救貧師」的稱號，「菩薩心腸」的美名。

國外人士都不惜千里而來禮聘曾子南作福，國內同胞更要聘曾子南作福，纔是求福得福的正途捷徑。對嗎？！
