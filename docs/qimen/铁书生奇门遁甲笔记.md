

# 铁书生奇门遁甲笔记



三十六计的反间计、离间计是惊门手段。

反间计：三国时周瑜巧用反间计杀了曹操手下精通水战的大将：蔡瑁张允。

历史上反间计如韩世忠反间计，皇太极施反间计除袁崇焕，岳飞施反间灭魏齐。

离间计：三国演义第五十九回 “许褚裸身斗马超，曹操抹书间韩遂”

“奇门遁甲”的含义是由“奇”“门”“遁甲”三个概念组成。“奇”就是乙、丙、丁三奇；“门”就是休、生、伤、杜、景、死、惊、开八门；“遁”即隐藏，“甲”指六甲，即甲子、甲戌、甲申、甲午、甲辰、甲寅，“甲”是在十干中最为尊贵，它藏而不现，隐遁于六仪之下。“六仪”就是戊、己、庚、辛、壬、癸。隐遁原则是甲子同六戊，甲戌同六己，甲申同六庚，甲午同六辛，甲辰同六壬，甲寅同六癸。另外还配合蓬、任、冲、辅、英、芮、柱、心、禽九星。奇门遁的占测主要分为天、门、地三盘，象征三才。天盘的九宫有九星，中盘的八宫（中宫寄二宫）布八门，地盘的八宫代表八个方位，静止不动，同时天盘地盘上，每宫都分配着特定的奇（乙、丙、丁）仪（戊、己、庚、辛、壬、癸六仪），这样，根据具体时日，以六仪、三奇、八门、九星排局，以占测事物关系、性状、动向，选择吉时吉方，就构成了《》《》《》《》《》《》《》《》《》《》《》《》《》《》《》《》《》《》《》神秘文化中一个特有的门类——奇门遁甲。

# 六仪用神与忌神性格特点

奇门十千克应的全部内涵，不真正的弄明白十千克应，你根本就成不了奇门高手。奇门的十千克应，可以说是奇门遁甲爱好者一见都头疼的事情。不知道里面说的到底是吉还是凶，基本都是模棱两可之间，其实十千克应是古人总结出来的精华，一直流传了几千年，自然有她的道理。这里面包含了古人的智慧，只是没有人把她弄明白而已。比如：戊加丙 青龙返首，动作大利，若逢迫墓击刑，吉事成凶。大家看看，这句话就是模棱两可了，开始在我的体系里，其实很简单的，也就是用神和忌神的关系。

现在世上流传的奇门遁甲，也讲十千克应格局的运用，就是不管是什么格局组合也不管是落在什么宫位里，所出来结果都是一样的，那么大家想想是不是错误的。

十干克应严格要求每个格局在每个宫位所处的环境及所处的状态。如这个格局：戊加戊凡事闭塞静守为吉，那么这个格局在八个宫位里面所处的环境状态都是一样的吗？

那是不可能的。因为每个宫位都有每个宫位的五行性质，乾宫是金，兑宫也是金，但金与金的大小能量是不同的。

巽卦是木，震木也是木，里面的木气能量根本是不同的。在本体系里，戊加戊，凡事闭塞，静守为吉。在艮宫的时候要静守为吉。

在兑宫的时候，怎么也不可能静守的。因为戊土在艮宫旺，所处的能量很强的，就像家里有很多的财产一样，不出去挣钱也照样生活的。在兑宫是不是泄气了，是不是很弱呀，当然要去挣钱了。大家想想是不是有道理。

读像是建立在旺衰、组合、喜忌、宫位的基础上读的，九星、八门、八神、三奇六仪在不同的宫位，不同的环境，不同的组合，不同用神和忌神的强弱的状态下，把信息锁定，读出信息象义。否则不按照这样的去读，那无疑是古人的刻舟求剑，《刻舟求剑》
