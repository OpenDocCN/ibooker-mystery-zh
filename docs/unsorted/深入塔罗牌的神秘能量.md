前言  

① 塔羅牌的運行原理  

在正式學會占卜後，許多人對於塔羅的準度嘖嘖稱奇，但也有許多人提出疑問，塔羅牌運作的原理到底是什麼？有一種很常見的說法是潛意識，人們的內心的想法以及過去的經驗，會反映在抽牌上，會不自覺抽到與現況和內心想法相同的牌。只不過多數的占卜都不會讓問卜者先看到牌的畫面，依照我的習慣也不會讓問卜者親自抽牌，問卜者的潛意識似乎很難發揮在牌上。  

幾個月前我讀了一本有關於宗教的書籍，裡面提到要是人們對於一件沒有生命的物件做了儀式性的動作，就會有靈附著在上頭。這時我想起一副塔羅牌開牌之後，一定要先抽主牌，占卜時也重複著默念、洗牌、切牌、發牌等規律的儀式化流程，而且塔羅牌總是有越用越準的趨勢，讓我不禁懷疑是否真的有靈附在塔羅上。但不管是潛意識亦或是靈都不在本書討論的範圍內，這裡沒有必要深究，這裡要教導各位初學者如何快速、有效的學習塔羅。就像是你到駕訓班學開車，你不需要了解引擎構造以及傳動原理，只要懂得踩油門和控制方向盤，你也能考取駕照。  

② 謂主牌？  

主牌象徵一副塔羅牌的精神、性格，通常在新牌拆封後，會將二十二張大牌獨立抽出，經過洗牌、切牌後抽出一張牌來代表主牌。我的第一副塔羅牌，主牌抽中了教皇，在占卜的時候完全顯露出教皇的個性，很喜歡給予建議、鼓勵，很多人在占卜後都能跳脫現狀找到新方向。而我的第二副塔羅牌主牌是月亮，在占卜時很常揭發當事人想隱瞞的祕密，或是不願意面對的事實，甚至當事人完全不了解的現況。  

或許把祕密赤裸裸的攤在牌面上會令人不舒服，但有些問卜者偏愛我這副塔羅，他們認爲這種表達的方式能幫助他們提早認清現實。我的第三副塔羅主牌是女祭司，她能很理性的分析任何事，尤其是學業以及工作，但對於感情的事總顯得有些害羞，老是用迂迴的方式呈現在牌面，光是要摸透她的表達方式還真是吃足了苦頭。  

不論主牌是那張牌，都不會影響到塔羅牌的準度，只不過面對同一個問題，會有不同的看法以及表達方式。即便是外行人，告訴他占卜流程後，所翻出來的牌也都是準確的，之所以會出現哪些占卜師特別神準，哪些則功力不足，完全是解讀上的問題。所以學習塔羅牌的人發現自己的占卜準確度不被認同時，有可能是對方不願認清、面對現實，也有可能是自己解讀的方向錯誤，需要多花點功夫和牌培養默契。  

〣 卜的結果有可能被改變嗎？  

身為一位占卜師，我不主張命運百分之百掌握在人的手上，人生活在世上總帶有一些宿命。但也不是說人生已經被定型，每個階段不過是按照劇本演出。  

對我來說人生就像一張空白的申論題，考試的方向已經決定，但你可以決定答題方向、內容，你的努力與否也決定了考試的分數。算命是種考前猜題，但它無法代替你做答。就好比戀愛，如果抽中死神牌，早已註定和對方緣分已盡，分手的事實不會被改變，但你可以決定要陷入哀傷、自怨自艾，或是重新振作，找尋更適合的對象，也有些人痛苦分手之後才體認到過去的感情有多愚蠢，反而對於出現死神牌感到慶幸。  

課業方面，或許已經難逃被當的命運，但你可以選擇擺爛，明年再見，也可以選擇放手一搏，雖然還是難逃被當的命運，但也有可能得到補考的機會，這樣就不用再多花一年的光陰。  

就占卜的經驗來說，如果結果是大牌，通常是宿命，很難扭轉。要是結果出現的是小牌，可能有機會透過努力改變現狀。只不過人生本來就要面對許多課題，與其強行改變命運，不如順著命運，從中學習寶貴的經驗，體驗命運要教會你些什麼。而且牌面的吉凶禍福是以世俗的價值觀去衡量，實際上不論哪張牌都沒有絕對的好壞，任何人世間的變局都是有意義的，只是很少有人能夠欣然接受。

# 占卜陣法說明  

塔羅牌占卜解牌時，最重要的是一個平靜的心。

無論是占卜解牌者或是詢問者，都要專注精神在塔羅牌上，若是無法靜下心或是帶著雜亂的想法，對於各方面來說都是不好的；無論牌面的意思好或是壞，也都只能當作是參考，絕對不要受制於塔羅牌上的牌義而不努力乃至於自暴自棄，那麼就失去了解牌的意義了。

陣法或是牌義只能當作當下的參考，真正掌握命運的還是在自己手中；有了好的牌義固然可喜，但是要更加的努力達成目標；出現不好的牌義也不要灰心，只要審視自己的內心和冷靜的對應接下來的挑戰，未來仍然會出現無限的可能性。

雖然有時候人們會覺得『一切都是命運使然』，但是筆者遇過一位研究非常透澈的命理老師和筆者說過：『影響命運的有三種東西：『健康』、『信仰』和『愛』。』

宗教解析也好、命理風水解析也好，塔羅牌反映出來的未來，只是建言、只是預告，沒有一定、更沒有絕對。

切記，命運還是掌握在自己手中。

## 占卜的關連性  

占卜時要注意的是，牌與牌之間的互相關連性之外，還有各種牌屬性的影響，特別是大阿爾克納牌在各種牌陣中出現時，所代表的通常是全面性的影響，利用出現的大阿爾克納牌為中心來解牌，通常會受到更詳細的啟示。

逆位牌的出現在各個學派中有不同的解釋，筆者這邊也是比較傾向『能量解釋』：一張牌逆位的呈現是由於能量不足所導致，例如：『魔術師』這張牌有著四大元素的呈現，同時也有暗示著學習的意味，通常是給予占卜者正面的評價；假如逆位呈現時，同樣暗示著占卜者在學習上稍微的欠缺，或許是環境影響、也或者是學習態度所影響，在逆位呈現上就顯得有些差強人意；也因為能量不足被隱蔽的結果，逆位呈現的情況就出現了。

筆者這邊建議：在逆位牌出現時，好好觀察、體會能量不足的原因和現實狀況，才能充分的解釋每次解牌的涵義。

除了各種牌之中的聯結和正逆位的觀察，更重要的是能夠依照解牌占卜師的感性、對於占卜者間的對談內容、內心想法去作充分的理解和傾聽，才能夠完成每一次的解牌任務。

### 陣法解說  

#### 一、縱列聖三角：三張牌  

###### 1. 牌陣說明  

縱列聖三角是占卜師最基本的陣法，可以說是基本中的基本。根據各個學派有著各種不同的說法和名稱，這邊依照最基礎也最常用由左至右三張牌的縱列聖三角來解釋。

###### 2. 牌陣效用  

縱列聖三角的實用性非常的高，幾乎所有問題都可以從這最基本的牌組從中獲得啟發；由左至右放上三張牌的牌組，可以充分的知道各種狀況。

###### 3. 牌陣用法  

縱列聖三角有兩種基本的解釋。

（1）時間之流  

從左至右分別為『過去』『現在』『未來』。

因為時間流動的關係，因為過去的『行動』出現了現在的『現況』，未來則有可能出現的『結果』。

時間之流牌陣可以用在各種問題占卜上，也可以從中瞭解到因果之間的關係。

（2）現況占卜  

從左至右分別為『現況』『行動』『結果』。

從現況開始分析，之後的行動後所預期的結果；和時間之流不同的方式來解釋，針對現況感到困惑的占卜者所使用會有更好的預期效果。

###### 4. 注意事項  

聖三角有各種各樣的陣法和變化，筆者認為從一開始的縱列聖三角開始學起會比較容易入門，對於占卜初學者來說也是容易解釋的牌陣。

這邊要特別提醒的是：根據筆者解牌多年的狀況來看，許多占卜者容易陷入看到未來牌出現不好的牌義時會感到不安，在這邊提醒各位占卜師，能夠開導或是讓占卜者感到安心是很重要的。

#### 二、瓊哈里窗：四張牌  

###### 1. 牌陣說明  

筆者結合了心理學中的『Johari window』所研究出的牌陣，適合用於心理測驗與心理分析上。

Johari window為美國心理學家Joseph Luft和Harry Ingham兩位於1955年發表的潛意識和公眾間的交流所提出的心理學理論；『Johari window』由四個區塊所組成，分別為左上『開放之窗』、右上『盲點之窗』、左下『隱蔽之窗』、右下『未知之窗』。

中文翻譯有『周哈里窗』或是『瓊哈里窗』等音譯，在健康心理學和交流分析領域中時常會用來舉例和解說。

※ Johari window說明  
開放之窗：open self，開放給公眾並且不隱藏的部分。  
盲點之窗：blind self，自己並不知道而其他人知道的盲點部分。  
隱蔽之窗：hidden self，只有自己知道並且不開放的祕密部分。  
未知之窗：unknown self，自己不知道而公眾也不知道的未知部分。

###### 2. 牌陣效用  

對於占卜者的心理測驗和分析上有很大的幫助，對於茫然無助或是對於自己內心想法有所疑問時推薦使用。

###### 3. 牌陣用法  

由一左上、二右上、三左下、四右下排成『田』字型，並依照順序進行解牌：『開放之窗』、『盲點之窗』、『隱蔽之窗』、『未知之窗』的順序進行占卜者的心理分析。

###### 4. 注意事項  

筆者在進行這項Johari window解牌時，曾經多次因為沒有顧及到對方的心情，而導致占卜者當場哭出來的情況，這讓筆者的指導老師多次嚴厲的指正我的解牌過於直接而被迫停止這個陣法的使用。

這邊會提出這個陣法，是希望各個占卜者可以用在更好的方面，若是可以用在開導對方和讓對方進而獲得心靈上的啟示，那也算是好事一件。

需要注意的地方是，根據筆者以往在進行『未知』這一張牌的解釋案例上，許多占卜者都會出現非常迷惑的表情，之所以如此也是因為這張牌背後的涵義或許並不是當下占卜者可以理解的，強行解釋也有可能出現反效果，因此各位占卜師可以選擇是否要婉轉的解釋這張牌，依照情況不進行解釋也是可行的。

- 遠古的過去
- 近年的事項
- 這一星期至昨日的過去
- 今日的狀況
- 明日至一星期後的未來

不一定要從一月一日開始，從占卜的當下開始算起一年的時間也可以；占卜過去的年運也是可行，也可當作參考。

## 3. 牌陣用法

靈魂數字的計算方式如下：

出生年+月+日，再將數字相加等於0-22的大阿爾克牌。

例如1982/01/23的生日，將1982+1+23=2006

2+0+0+6=8

對應大阿爾克牌第八張牌『力量』。

和牌義好壞並無特別的影響，就算是牌義不好的牌也有可能因為本身的行為和模式而成為助力。

特別說明，19雖然是1+9=10，但是10還是要用1+0=1來計算，這時就會由19和10以及1來對應，這些牌也都會所影響；若是出現22，除了2+2=4對應大阿爾克四號牌『皇帝』之外，也會對應零號牌『愚者』。

# 十二、生命之樹：十張牌

# 1. 牌陣說明

『耶和華 上帝在東方的伊甸栽了一個園子，把他所造的人安置在那裡。耶和華 上帝使各樣的樹木從地上長起來，既悅人眼目，又好作食物。園子中間還有一顆生命樹，和一顆辨識善惡樹。』《舊約聖經·創世記》第二章第8-9節

生命之樹在聖經中創世紀篇就記載著通往神之國的路徑，卡巴拉學者也將創世的符號用生命之樹來代表。

生命之樹分為十個直點，也就成為了塔羅牌中經典的生命之樹牌陣。

# 2. 牌陣效用

這個生命之樹牌陣可以解讀出一個人的個性構成，同時通過牌陣瞭解占卜者更多內心的訊息。

以下為生命之樹對應的牌陣意義：

第一張牌：Kether，王冠，Crown。理想與目標。

第二張牌：Chochmah，智慧，Wisdom。智慧與自發性。

第三張牌：Binah，理解，Understanding。理解及應變。

第四張牌：Chesed，慈悲，Mercy。善良與援助。

第五張牌：Geburah，力量，Fortitude。力量與領導、障礙與困難。

第六張牌：Tiphareth，平衡，Balance。特質與健康、調和與機會。

第七張牌：Netzach，勝利，Victory。愛與靈感、熱情與慾望。

第八張牌：Hod，威嚴，Splendour。榮耀與技能、評價與成績。

第九張牌：IX Yesod，基礎，the Foundation。靈魂中心、潛在能力。

第十張牌：X Malkuth，王國，the Kingdom。實際生活、最終結果。

生命之樹的第二種用法，是用來占卜第三方的人事物，這和戀愛占卜十分類似，但是可以用生命之樹牌陣占卜出和占卜者有親密關係的戀人，就算被占卜的第三方不在現場也可以獲得一定的訊息。

第一張牌：心理狀態。對方的性格和心情，外在顯示出的氣質。

第二張牌：智慧。對方的想法和行動原則。

+   第四張牌：相性。和占卜者之間的相性層度。
第五張牌：不合。和占卜者之間的不合層度。
第六張牌：平衡。領導能力與協調性、調整能力等影響。
第七張牌：積極性。對方的積極性表現。
第八張牌：消極性。對方的消極性表現。
第九張牌：本質。對方的本能、本質的顯示。
第十張牌：綜合結果。最終顯示出來的評價結果。

# 3. 牌陣用法

依照生命之樹的構成，由上至下的排出順序：在各個學派間略有不同，但是本質的涵義是不改變的。

有的學派很注重取牌的順序，由第一張到第十張都有順序要遵守，有的學派還會在最後解牌完後會再取一張牌當作達卡牌；這邊筆者的建議是可以依照占卜師自行判斷是否要遵守取牌的順序，以及最後的達卡牌當作結論，這些都可以由現場狀況自行決定，無論要不要遵守取牌順序或是抽達卡牌，都不影響生命之樹的牌陣。

# 4. 注意事項

生命之樹的牌陣雖然有趣，但是本身的涵義非常的深遠，建議在占卜時可以反覆推敲以及研究牌義和各個牌的關連性，以及對應的牌陣位置。

在使用生命之樹牌陣占卜現場不在的第三方時，要注意是否會因為關連性不足而有誤差，一般而言若是第三方和占卜者並不親近的情況下，是不建議使用生命之樹去研究解牌的；除了誤差會大之外，通常而言也不會具有什麼意義。
