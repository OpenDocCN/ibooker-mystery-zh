## 前言

为什么要写这本书？

1.  紫微四化星很好用，任何人的本命四化星于其一生中，都会陆续或同时展现力量，不需要命盘也可随时解读。

A. 把四化星背熟，可以生年干、各大运、流年、流月、流日时，及时运用。有下列几种方法：

只看命主出生年干，不需命盘亦可以四化星解读一生重点。只要对方生活经验丰富，年过四十，通常有六成的命中率。

例如丙年出生者，可有下列重点解读：

- 天同化禄：注重精神层面及生活品质。
- 天机化权：做事积极有效率，懂得安排有效率的生活。
- 文昌化科：于文艺大范围相关事务有兴趣或有机会接触，聪明好学。
- 廉贞化忌：注意身体或心情上的过敏。

以上于命主，是一生可参考的四个重点，也可交互运用，亦可提醒命主，有哪些优点可加强，其化忌所代表的负面现象，可由化禄、化权、化科几颗星曜协助解决。例如，多以文昌化科的优点，寻找资讯，处理过敏现象，或以天同化禄加上文昌化科，安排悠闲时间，解除压力。

2.  四化星很适合用来趋吉避凶。有下列三种方法：

- A. 在命盘中找出四化星，回顾过去四化星所发生的种种，于精神、心灵方面的负面，可即时调整。例如以前的居住环境很吵杂，心情不定，现在回顾，有新的正面思考与实际解决方法。也可用于未来相同四化出现时，面对并处理嘈杂问题。
- B. 找出未来四化，预先安排计划防范。例如，未来逢武曲化忌，在观念上了解，武曲化忌有哪些正面意义及效果，先有心理准备，也开始练习正面思维，并预先存放生活必须的储蓄。
- C. 观察每个流月、流日、流时。以流日为重点。

B. 如果有命盘，除了以出生年四化星当主要参考，也可以配合各大运，流年、流月、流日、流时共同解读，于任何年龄层皆适用。

3.  四化星很有趣，每天观察自己及他人的四化星，仿佛天天都有新鲜事，愈观察愈开心，不但放每日纪录当天的精神、心灵及实际发生状况，做为日后参考。

得下烦恼。还会消解烦恼。

4.  还有些好处，待我们实际观察、练习后，一起讨论。

我写《紫微四化星——如何改变一生的秘笈》为何敢说改变你一生？研习紫微斗数五十年来，我在四化星上着力颇深。在这本书中采用了古代贤人和自己的精心归纳整理。例如壬年生的人，可以参看壬天干四化看看自己符合哪几点，作为终生参考。壬月干、壬日干的时候，自己又有什么感受？另外壬年生人，在甲年至癸年的循环之中又会有何观念变化？学好《紫微四化星——如何改变一生的秘笈》，你就可以自己算自己的命，在新一年、新一月，乃至新一日来临前，先了解未来的你，可能会如何！当读者抓住方法，甚至能帮助身边的人时，这样人生自然迎来一片不一样的景象。

# 第一节：什么是四化星

“四”是数字，“化”是一种变化，但不离本质，例如，将固体的糖与液体的水加在一起，糖与水结合后会化成糖水，糖水是糖化出来的，仍属于糖，也独立为“糖水”，有其作用。

古贤以诸星不同性质，安排某些星曜逐以推算时空的四种化星，分别是化禄星、化权星、化科星与化忌星，合称四化星。

## 化禄星

### 精神及心灵现象：

1.  心情顺畅，尤其与未化禄时相比较。
2.  觉得其实没有什么事情是困难的。
3.  觉得困难也可以解决，但不知道为什么以前没有想到，以大运、流年、流月、流日遇到时较为明显。本命遇到，一生多顺遂。
4.  不易发怒，或可以把负面情绪理性转换掉。

### 正面生活现象：

1.  各方面知识皆有机会涉略，有时可以多样互相通用。
2.  原先受阻的事情可以顺利推展进行。
3.  寿命、体力增加。
4.  机会增加。
5.  只要自己愿意，通常都会进步。
6.  所期待的事情得以实现。

## 化权星

### 精神或心灵现象：

1.  相信自己能够掌握，通常也可以做到。
2.  稳定、沉着。
3.  可修正以前的错误。

### 正面生活中现象：

1.  大部分时候能够把握自己该做的事，并符合自己的意思。
2.  让人产生信赖。
3.  在生活及命运的变化中，大多时候是成功者，不怕失败。
4.  不易被人牵着鼻子走，保持一定正面原则及努力方向。

## 化科星

### 精神或心灵现象：

1.  心情开朗，与未化科时相比，多有云开见月之感。
2.  觉得充满希望。
3.  有体悟。

### 正面生活中现象：

1.  有知名度。
2.  生活中处处有生机、希望，多能顺利推展计划。
3.  予人良好的印象，让别人乐于替他说好话。
4.  使人有愉快的感觉。
5.  充满希望，也会因有希望而努力。
6.  把所体悟的发挥在生活中。

## 化忌星

### 精神或心灵现象：

1.  觉得有阻碍。
2.  觉得自己心里打了很多结，有说不出的难过。
3.  害怕阻碍，因此不敢向前，有暂时畏缩、逃避的心理。
4.  有时觉得老天或别人刻意在跟自己做对，跟自己过不去。
5.  以上的精神心灵层面，大多可以转为正面思考。

例如：

1.  知道阻碍有时只是使人暂时停止，所有的阻碍都有其意义存在。
2.  能够随机应变。
3.  相信人生没有真正的绝路。

### 正面生活中的现象：

1.  觉得有阻碍。
2.  不顺利、旁生枝节。
3.  害怕阻碍，因此不敢前进，有暂时畏缩、逃避的心理。
4.  有时觉得老天或别人刻意在跟自己做对，跟自己过不去。

但也可以转成下列现象：

1.  抽丝剥茧、观察及分析挫折产生的原因，重新处理，或改变心态，乐观、客观对待。
2.  面对经验之后，可以克服困难，亦不怕困难。
3.  能够随机改变计划，继续努力。
4.  遭遇困难时，会停下来思考，反而增加定力，将事情看得更清楚，又增加信心，有良性连锁反应。
5.  愿意找寻方法及他人协助。

# 第三节：四化星与天罗地网

天罗、地网宫，也就是辰、戌二宫，是紫微斗数命盘中较特殊的两个宫位，除七杀、破军、贪狼、太阳、太阴、天府、天相较不受其限制外，大部分的星曜坐天罗地网宫时，有不同的讨论方向，善用者，皆向正面发展。

## 化禄星

任何星曜在天罗地网宫化禄时，都有下列现象：

1.  都容易使当事人变得较为谨慎，而不强求，或不逞强，有些人懂得随顺正面心态及当时生活状况。也有收获。
2.  一生多从事同一种性质的工作（但无碍兼职），及仲介、贸易。

如有煞星同宫，也可减少其缺点，无三颗以上煞星会照，且田宅宫、福德宫皆佳者，也能在保守中进步，累积钱财或不动产。

## 化权星

化权星在天罗地网有下列现象：

- 一、使原本无力的星曜，增加勇气，且有突破现状的想法及计划，一路向前。
- 二、一生总有不停奋斗的阶段，也可因自己的努力而有成就，有时也会处于种种具有竞争性的环境中，但仍仍有个人或团队表现。

## 化科星

化科星在命盘中任何宫位皆较能表现其力量，尤其是天罗、地网宫。

- 一、天罗、地网，无法束缚代表亮度的化科星，因此任何星曜在天罗地网化科坐命，皆自然能有表现，只要再加努力，大限、流年配合得好，才华与能力是不会被掩盖的。
- 二、当事人的精神、心灵不易受困，有些人对提升自我、创造设计、文艺、求新知识、宇宙观等，别有心得或兴趣，也有表现。

终其一生，向客观、大度前进。

## 化忌星

- 化忌星在天罗地网中，有下列几种现象：

一、其不吉，因受限制，反而可以减少于人生的种种破坏力。

二、化忌星有时可使人在困难中努力，并于大运遇六吉、化禄、化权、化科时有收获。

- 化忌星与六煞星之一同坐天罗地网宫时，也可使得凶星之间的煞力互相抵制，而减少损害。

# 第五节 四化的运用

本节分为下列三个部分：

## 一、各天干的链接与融合。

每组天干四化在互相链接之后，产生的整体现象。
每一个天干与其他天干的链接关系，例如癸天干与甲天干、甲天干与乙天干的链接关系。

## 二、各天干四化星的现象。

每个天干化禄、化权、化科、化忌，各有一组解释，例如，甲天干，廉贞化禄、破军化权、武曲化科及太阳化忌，皆各列举几条现象。
每一组现象，都有其独一无二的代表，例如，乙天干，天机化禄，代表多机会及其带动的顺遂，居各星化禄之冠，又如丙天干，文昌化科，其所代表着观察力及领悟力居诸星之冠。
每一个化星，也有各自的最高境界，例如，天同化权的最高境界是，不“强求”掌握，大器、拿捏得宜、随缘分享、自有成就，是天同化权的最高境界。

每个化星，从最基本的代表，至最高境界，都可能也可以在自己一生中因事件出现，或在某个时间阶段表达出来，例如太阴化科本有光亮或也没有，但被大家认同月亮是有光的，为希望、光明、安慰、感动、大爱。最高境界也是因其所代表的意义而产生，通常也是在生活中体验，向正面调整改进之后，了解也成就了最高境界。

本书描述各四化的文句有些重复，是考虑到大家平时惯用的言词语句不尽相同，例如2020年有三倍券也有敬老金，但三倍券与敬老金共有“政府照顾”的核心心意义，三倍券、敬老金，与古时的增发薪饷或赈灾都可以“太阳化禄”来表达。所以，我们研读各四化现象，也需要考虑核心心意义，运用于其引申出来的古今社会现象。

本节所有的化忌星，没有直接写出很严重的现象，例如，凶险不断、心术不正等，因为：

1.  任何化忌都有正负面。先了解化忌星所代表的负面现象，在内心转负面想法、观念，在行为上正面处理，可以化解。（有赖于大家日日练习体会）
2.  保持客观，感恩、祝福也能找出新方向，解决些内心的困扰，困扰一旦解决，生活中也会陆续出现正面改变。
3.  有些化忌具有在磨练翻转体会之后，负面转正面的基因，成大器大爱。例如太阳、太阴二星化忌，落陷，皆有此机会。（二星本质如此，逢化禄、化权、化科、化忌皆然。）

我格外重視流日，因爲方便觀察，利於紀錄，把一日四化看做所有未來還會出現流日四化的參考，暫不管地支宮位，只以四化爲標準，長期紀錄，以個人內心想法，外在行爲，以及實際發生的事情，逐一對照未來及過去每個相同天干，爲自己統計歸納出一套屬於自己的四化星核心意義

此外，流日四化適合逐日連續觀察，以甲、乙、丙至壬、癸，每十天干爲一組的相續觀察，跟未來每一組十天干對照比較。

### 6. 流時四化

時辰有其重要性，例如；中樂透在某一時辰，從此影響後半生。拿到高考錄取通知書的時辰也很重要，關乎未來工作。時辰也屬於本天干的劇本內。有時面試、或開刀，也會以時辰搭上年、月、日，排出命盤，面試的命盤解讀重點放在福德、遷移、事業宮，開刀的則放在福德、疾厄宮，有化祿、化權、化科及六吉星出現皆吉。

## 第七節 生活中創造化祿、化權、化科星的建議

# 第九節 四化星的過去現在與未來

我們對「過去、現在、未來」的體會各有不同。

## 過去：

- 1. 以時間單位表達，例如，2019年以前，1983年10月18日，上個月，昨天等。
- 2. 已經發生的時序，例如，五年前曾計劃到高雄旅遊，但未實現，此計畫也「存在大腦中」。

## 現在：

- 1. 以時間單位表達，例如現在是2020年10月18日上午12點。
- 2. 正在發生的事情，例如正在喝咖啡，也在大腦中。
- 3. 現在，我們可以說是「此時此刻」或「當下」，但又有兩種現象：
   a. 如果心是安定，靜止沒有時間空間，大腦存在的是清楚明白，自然具有觀察的能力，脫離也超越過去、現在、未來。
   b. 如果現在大腦有想些事情，時間前進以後就變成過去，例如2020年10月18日上午12點在喝咖啡，到13點時間過去了，喝咖啡事件也結束了。

## 未來：

- 1. 可以時間單位表達，例如2025年、下個月、下週五、再過幾天、將來等。
- 2. 沒有事情發生。
- 3. 有預先計畫的事情，例如，2023年大學畢業、已預約訂房下週去台南旅遊，不一定如期發生，但也會留在大腦中。

- 太陽化祿：有點多、有點熱、快或急、蔓延。有瘀塞或痰的感覺。大家都有的現象，錢財、資源、互助、眼界、愛心增加。
- 武曲化權：迅速做決定，即時掌握，有勇氣、能擔當。
- 天同化科：以開放的心胸過生活，為自己安排悠適的空間，互助分享。
- 太陰化忌：為六親關係，及全世界人類努力。有時情緒複雜、起落。

## 辛天干

- 巨門化祿：以各種方式使所有的不平衡現象，得以通暢。運用語言、文字、傳媒，不失為一種方式。
- 太陽化權：代表權威的感覺及現象，制定某些方法制度，適合大眾，有大愛及遠見。
- 文曲化科：心裡有話會說清楚，也利於以與文藝有關的其他方式表達，例如音樂、舞蹈等。
- 文昌化忌：留意所有溝通表達方式，注意文字、數字、作品。成敗皆有，也許敗中求勝。

## 癸天干

天梁化祿：承祖上或自己修養而來的福蔭，有財物、名聲，也有得到保佑、保護的感覺或實際現象。

紫微化權：在意自己的形象，面子、表現，也具有某些權威感、權威性專業。

左輔化科：願改進、適應、幫助自己以及他人。

武曲化忌：做決定之前三思，確定來日不會後悔，不管有無三思，做了決定也不後悔。從擔心是否會中斷，到會運用中斷。而不介意中斷。體會到置之死地而後生。

破軍化祿：因保留、儲備、後援有繼，而造成流通及未來的動力。

巨門化權：言語有力，能說服他人，或發號施令、或帶動風氣，適合通用大眾傳播、文藝方式表達訴求。

太陰化科：心中有光，有被撫慰的感覺，也能傳遞光、愛的感覺給他人。利考試、比賽、進修。

貪狼化忌：不強求、不逞能，順著現有狀況過日子。有個人才藝。身體上或有胎記、傷疤，或不平衡處。
