# 從個人到政治——改變自己，改變世界

每個人生命來都是一個獨立的個體，但當他逐漸長大成人以後，慢慢就變成了一群人的集合體。然而，大多數人都沒有意識到這一點。 如果你只是靜靜地坐著，傾聽你的頭腦，你會發現裡面充滿了很多聲音。你會驚 訝自己很輕易就能辨識出這些聲音。有些聲音來自祖父，有些聲音來自祖母，有些來自 你的父親，有些來自你的母親。有些聲音來自神父、老師、鄰居、朋友或是你的敵人。 所有的聲音都混雜在一起，它們在你的內在形成一個群體，你幾乎不可能找到自己的聲音；這群人密密麻麻的擠在一起。事實上，你早在很久以前就忘了自己的聲音。你從來沒有表達意見的自由。你被教導要服從，要聽從長輩的教誨，追隨老師或神職人員的行止。沒有人要你去尋找自己的聲音，沒有人問過你：“你有没有聽過自己的聲音？”所以，你的聲音越來越微弱，其他人的聲音很大聲，很權威，它們是發號司令的 人，你身不由己的跟隨他們。你不是故意的，你知道這麼做不對。但是為了受尊敬、被接受、被愛，你不得不乖乖順從。你的內在只少了一種聲音，只少了一個人，那就是你自己；除此之外，總是一整群人。這群人老是讓你抓狂，一個聲音說：“就這麼做吧。”另一個聲音說：“絕不能那麼做！別聽他的！你被四分五裂。”這整群人都得退下；你必須對他們說：“請別來煩我！”那些退隱山林的人們並不是真的脫離了這個社會；他們只是想要找個地方讓自己內在的那群人消失，但你內在的那些人顯然還不怎麼想離開。

我的內心有一個部分想坐下來休息、靜心，專注自己的內在成長；但 是，另一部分卻想工作、奔波、規劃組織、抗爭、與新聞記者和政界人 士高談闊論。我如何解決這兩股衝動之間的矛盾呢？

—人有內外兩面—，這是一個老掉牙的謬論，一個方便人們去譴責另外一面的謬論。

# 尋找改變的方向——沉溺過去的黃金歲月，或者創造金色的未來？

當今人類最迫切需要的，就是意識到人類的過去已經把人類帶入歧途。延續過去沒有任何意義，這是一條不歸路。～新人類～的誕生是絕對迫切需要的。

新人類將不再延續舊有的觀念，不再認為一個體只是社會的一部分。新人類將會是一個個體與個體的紮合：個體是主人，社會則為他們提供服務。這個社會將有許多不同的面向。它不會有許多宗教，只會有一個宗教意識。它不會有一個以造物主自居的專 制上帝，因為這暗示著人類的奴隸身分。新人類擁有神性，那是一種終極的品質、一種 開悟的品質。`神`將遍及所有的人，所有的事，所有的物。 有史以來，個人第一次不用按照規定行事；他會得到協助讓他去做自己。他不會被灌輸任何理念、紀律，以及既定模式。他只會得到無限的愛去追求自由。他犧牲一切，連自己的命都可以不要，但他不能犧牲自由。新的個體不會被壓抑；他能夠自然的、無拘無束的表達他所

現代人會有這樣的感受，並不奇怪；古代人也是一樣的。你翻閱古老的記載；所有古老的經文都說：未來是黯淡無光的。「它的過去必然是一段黃金歲月。過去是美好的，是「satyug」，真理的年代；未來是「kalyug」，是死亡與黑暗的年代。每個人內心深處都有這種心態；它與時間或你周遭的現實無關。你得丟棄這種悲觀的心態。這完全取決於你想怎麼做。

在過去，兩個世界強權國家不停的交戰，那樣的戰爭沒有效率，沒有危險性，不會把所有的人類都毀滅掉。那沒有問題，它只是一個遊戲。男性自我的頭腦非常享受遊戰，它們非常需要遊戲。那就是爲什麼三千年以來，我們已經發動了五千場戰爭。如果沒有原子彈，這樣的戰爭會持續下去。原子彈的存在意味著：你現在如果在決定開戰，將是宇宙性的自殺行為。誰想冒這個險？沒有人會贏，所有人都會死。如果沒有人贏，這場遊戲還有什麼意義？

戰爭表示有人會勝利，有人會落敗。如果沒有人贏，雙方同歸於盡，那戰爭就太愚蠢了。原子彈的存在使得世界強權國家牽制彼此發動戰爭。發動戰爭是荒謬的。所有的黨派都同歸於盡，一切還有什麼意義？原子彈讓戰爭變得毫無意義。

我一想到原子彈，就看見無限的希望。我絕不是一個悲觀的人。我相信所有的事情會一天比一天更好。你或許覺得不可思議，但如果你明白我的話，就是這麼簡單。

有了原子彈，戰爭成為全面性的。過去炸彈只是戰爭的一部分——它會殺死一些人——但是現在，整個地球都會被毀滅。我們有這麼多炸彈，足以讓每個人死一千次，我們也可以用它們摧毀地球一千次。面對我們摧毀的力量，地球現在非常渺小；相較於我們破壞的力量，地球更是微不足道。現在，誰想冒這個險，為什麼要這麼做？你就算打贏了也沒什麼好得意的，因為所有的人全會死。

第三次世界大戰不會爆發，並不是受到佛陀與耶穌非暴力以及愛的教誨，不是這樣！是因為原子彈，因為那樣的戰爭，死亡是必然的，自殺行為是全面性的。不只人類，還有鳥禽、動物、樹木，地球上所有的生命都將全部被毀滅。

## 社區與個人——新形態的生活
整體的存在中，只有「人」需要規則。沒有任何動物需要規則。

問題在於，人需要規則、法律、政府、法院、軍隊、警察部隊，因為他失去了動物自然的狀態，又還沒有達到另一個自然的狀態。他夾在中間。他還沒有找到自己的定位，他還一片混亂。所有的規則和法律，就是用來控制這團混亂的。這個問題變得更複雜，它發展出控制人的勢力，讓宗教、國家、法院變得非常強大。他們必須獲得權力，否則他們要怎麼控制呢？因此，我們掉進了奴役制度當中。現在他們已經變得太強大了，他們不想放棄自己的既得利益。他們不想讓人們進化。

當你問到，個人與社會要怎樣才能進化，這表示你還不太了解這個問題。如果個體進化，意味著社會得瓦解。社會的存在，只是因為個體不被允許進化。幾個世紀以來，社會所有的權力階層都一直在控制人，同時也在享受他們的權勢。他們還不準備讓人進化，他們不想讓人繼續發展，他們怕自己最後會落得毫無用處。
