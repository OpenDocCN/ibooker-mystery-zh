# 制作时应注意事项

- 使用树脂液以及硅胶时，请务必在空气流通良好的地方进行。
- 使用树脂液以及硅胶时，请不要接触到皮肤。本书建议制作时穿戴手套及口罩。
- 请穿着弄脏也无所谓的衣物并在桌子、地板上铺上垫子后再进行作业，避免树脂液弄脏衣物、家具等等。
- 孕妇以及健康上有疑虑者，请考量自身的身体状况，避免长时间作业，并应适当的休息。
- 家中若有年幼的孩童，请务必将树脂液等材料收纳至适当场所，以避免孩童误食。
- 硬化后的树脂作品若接触到紫外线及空气，有变色的可能。
- 使用树脂液以及硅胶时，请务必先确认制造厂商提醒的注意事项。

## 树脂之基本

### 树脂是什么

树脂，在英文中称为「Resin」。虽然松树、漆树所产的天然树脂，以及人工生产的合成树脂都称为树脂，但作为手工艺材料所称的树脂，大多为“UV胶树脂液”或“双液型环氧树脂”。
其中 UV 胶由于百元商店就有贩售的关系，是种相对容易入手的材料。
UV 胶树脂液与双液型环氧树脂之不同，会在下个项目解说。
于 2017，日本国内开始贩售的“UV-LED 树脂液”，其不仅透明度高，就算经长时间放置也不容易产生黄变（颜色变黄），且持有只需经短时间照射 UV 灯就能硬化的优点。虽然价格上比起一般 UV 胶稍微贵了一点，但之后应该会变得比较普及。

### UV 胶树脂液与双液型环氧树脂之不同

- **UV 胶树脂液**
是种经紫外线照射后会硬化的液体。虽然以太阳光照射也能硬化，但用 UV 灯照射能让硬化的时间缩得比较短。也因要经过照射的步骤，所以只能使用透明的模具。另外，若作品形状的厚度过厚，会造成紫外线难以穿透，使得硬化困难，所以 UV 胶比较适合制作形状较薄的作品。

- **双液型环氧树脂**
需使用主剂与硬化剂两种液体混合才能硬化的一种树脂液。由于价格上比 UV 胶还便宜，因此适合用来大量制作和贩卖的作品。与 UV 胶不同之处为可以使用不透明的模具。虽然拥有硬化时间较长、液体混合比例需正确到以 0.1g 为单位等不便之处，但比起 UV 胶，能制作出的作品还是比较广阔。
※ 使用时请在通风良好之处并穿戴防毒面具进行。
※ 为了防止树脂液接触到皮肤，请穿戴丁腈手套。

| 价格 | 硬化时间 | 适合制作的形状 | 能使用的模具 |
| :--- | :--- | :--- | :--- |
| UV 胶 | 比环氧树脂贵 | 短时间内就可硬化 | 比较小的作品、比较薄的作品 | 透明模具 |
| 环氧树脂 | 比 UV 胶便宜 | 硬化需要时间 | 较大的作品、较厚的作品 | 任何模具皆可 |

  将树脂液染色时会使用，也可用牙签或竹签代替。

  用来将树脂液染色或戳破气泡。
- ⑦ 模具
  将树脂液倒入后使用。一般为硅胶制或聚丙烯制。
- ⑧ 脱模剂
  只要事先涂在模具上，再将成品从模具中取出时会变得比较容易。

## 使用模具

这里将用市面上所贩卖的模具示范，加入 UV 胶至硬化完成的制作过程。第一次制作树脂作品的人，建议使用材质较软且硬化后容易取出的硅胶模具。

- 材料
  - UV 胶
  - 染色剂
- 用具
  - 调色盘
  - 调色棒
  - 硅胶模具
  - 牙签（或是竹签）
  - 热风枪
  - UV 灯
  - 剪刀（或是斜口钳）

※ 使用完的用具，请用树脂清洁剂擦拭。
※ 就算只有一丁点的树脂液，也绝对不能倒入排水沟（因为会凝固变硬）。

# 1

![](img/9bdaa75ac2feafabda94224f5e4a40c3_12_0.png)

将 UV 胶倒入调色盘中，并染上自己喜欢的颜色。在染色时要慢慢的混合颜料，这样才不容易产生气泡。

# 2

![](img/9bdaa75ac2feafabda94224f5e4a40c3_12_1.png)

若产生气泡的话，可以用牙签、竹签戳破，或是用热风枪将气泡消除。由于热风枪会产生高温，所以使用时请注意距离，不要烫伤了。

# 3

![](img/9bdaa75ac2feafabda94224f5e4a40c3_12_2.png)

接下来要将 UV 胶加入模具中。加入时不要一次全部加入，而是要分层加入，这样成品才会比较漂亮。使用调色棒沾取 UV 胶后慢慢加入，成品比较不容产生气泡。

# 4

![](img/9bdaa75ac2feafabda94224f5e4a40c3_12_3.png)

若是产生气泡的话，重复步骤 2 的动作。

# 5

![](img/9bdaa75ac2feafabda94224f5e4a40c3_13_0.png)

使用UV灯照射约4～5分钟。

# 6

![](img/9bdaa75ac2feafabda94224f5e4a40c3_13_1.png)

加入大量的UV胶。若产生气泡的话，重复步骤2的行动。

#### 7

![](img/9bdaa75ac2feafabda94224f5e4a40c3_13_2.png)

使用UV灯照射约4～5分钟。以牙签确认表面的触感，若感觉硬化不全还有点黏，可以将模具转方向或是翻到背面照射。

#### 8

![](img/9bdaa75ac2feafabda94224f5e4a40c3_13_3.png)

待硬化冷却后，从模具中取出。使用剪刀跟镊子，去掉毛刺（突起的部分）。

> **Point**
若放置气泡不管的话，有可能会产生表面凹凸不平，中间有洞等情况，使作品无法漂亮的完成。所以在用UV灯照射硬化之前，先耐心的将气泡消除吧！

![](img/9bdaa75ac2feafabda94224f5e4a40c3_13_4.png)

## 使用 OYUMARU 翻模土

想要制作更有原创性作品的话，可以使用塑胶黏土“OYUMARU”制作模具。若使用矿石等物品翻模，就能表现出与市面贩卖模具所做作品不同的风格。

- OYUMARU 翻模土（单色透明）
- 用来翻模的矿物（小石头或是水糖皆可）
- UV 胶
- 染色剂

![](img/9bdaa75ac2feafabda94224f5e4a40c3_14_0.png)

- 耐热容器
- 镊子
- 厨房纸巾
- 调色盘
- 调色棒
- 牙签
- 热风枪
- UV 灯
- 剪刀（或是斜口钳）

![](img/9bdaa75ac2feafabda94224f5e4a40c3_14_1.png)

将“OYUMARU 翻模土”放入耐热容器中，并注入80度以上的热水。小心不要烫到了。

![](img/9bdaa75ac2feafabda94224f5e4a40c3_14_2.png)

待翻模土变软后，用镊子取出并用厨房纸巾擦拭水气。若用卫生纸擦的话，可能会残有纤维，这点需注意。

![](img/9bdaa75ac2feafabda94224f5e4a40c3_14_3.png)

趁翻模土还软的时候揉圆压扁。

![](img/9bdaa75ac2feafabda94224f5e4a40c3_14_4.png)

压扁后，包住用来翻模的矿物。若翻模土压得太薄，取出矿石时可能会裂开，这点还请注意。

![](img/9bdaa75ac2feafabda94224f5e4a40c3_14_5.png)

为了让倒入UV胶时能稳定不摇晃，可以从上方向下压，让底部变平。

![](img/9bdaa75ac2feafabda94224f5e4a40c3_14_6.png)

若过程中翻模土变得太硬，很难翻模的话，可以再一次将翻模土放入热水中，或是使用热风枪加热，使其软化。

![](img/9bdaa75ac2feafabda94224f5e4a40c3_15_0.png)

放入冷水中，让翻模土变硬。

![](img/9bdaa75ac2feafabda94224f5e4a40c3_15_1.png)

从翻模土制成的模具取出矿物，并确实用厨房纸巾擦干水气。若残有水气，可能会导致加入 UV 胶时硬化不完全，这点还请注意。

![](img/9bdaa75ac2feafabda94224f5e4a40c3_15_2.png)

将 UV 胶加入调色盘，并染上自己喜欢的颜色。慢慢的拌匀混色就能避免气泡产生。

![](img/9bdaa75ac2feafabda94224f5e4a40c3_15_3.png)

若产生气泡，可以用牙签、竹签戳破，或是用热风枪消除。

![](img/9bdaa75ac2feafabda94224f5e4a40c3_15_4.png)

# 染色的基本

染色劑一般分為液態染色劑以及粉狀染色劑。不管是哪種類型的染色劑，都只需要非常少的量就能將樹脂液染色，所以在使用時請以一點一點的方式加入染色。要注意的是，若加入過多染色劑會導致難以硬化，請小心注意。混合時若動作太大很容易混入氣泡，所以請慢慢地、一點一點地混合。

## 使用液態染色劑染色

將透明 UV 膠倒入調色盤中，並加入少量染色劑。

使用調色棒慢慢的混合，並注意動作不要太大。若想要讓顏色更濃，請以一次加入一點點的方式補充染劑。

樹脂液用的液態染色劑容易混色，適合初學者使用。

## 不同染色劑呈現出的效果

依據染色劑的不同，樹脂作品最終呈現的效果也會有所不同。若想讓作品呈現出透明感，樹脂液用的液態染色劑是最合適的。若想稍微抑制住一些透明感並加上一點霧面顆粒感，可以使用粉狀染色劑。最後，若是使用壓克力顏料作為染色劑，就能呈現出明顯的霧面感。

## 製作漸層

1. 將透明 UV 膠加入模具中。
2. 加入極少量的染色劑。也可以加入已經染色的 UV 膠。
3. 將竹籤垂直插入後，慢慢攪拌。待漸層混的差不多時，照射 UV 燈約5～6分鐘使其硬化。照射時可以改變光的方向，使整個模具都可照射到。

## 做成雙層顏色

1. 將透明 UV 膠加入模具中。
2. 照射 UV 燈約3分鐘，使其硬化。因為之後的步驟還會再照一次 UV 燈，所以這個步驟沒有完全硬化也沒關係。
3. 將有顏色的 UV 膠加入。加入時，從邊緣一點一點地加入比較不容易產生氣泡。
4. 照射 UV 燈約5分鐘，使其硬化。照射時可以改變光的方向，使整個模具都可照射到。

## 常見的樹脂作品失敗案例

### 作品表面有皺紋

UV 膠硬化時會有收縮的性質，而這也是產生皺紋的根本原因。在將 UV 膠加入模具時，不要一次全部加入待硬化，而是要分層加入後分次硬化，這樣就比較不容易產生皺紋。若這麼做還是無法避免皺紋的話，可以如同右圖般，用畫筆在整個作品塗上一層薄薄的 UV 膠保護層後再次硬化，皺紋就會變得比較不明顯。

### 作品表面沾有髒污或灰塵

最根本的原因為使用沾有髒污或灰塵的模具所造成。在使用模具前，事先用沾有樹脂清潔劑的面紙、酒精濕紙巾擦拭，或以封箱膠帶的黏著面去除灰塵髒污，就能避免此情形發生。另外，在作業途中，也要當心灰塵不小心飄入。

### 產生氣泡

若是比較小的氣泡，可以用熱風槍消除；若是較大的氣泡，可以用竹籤戳破。另外，使用較難產生氣泡的樹脂液也是另一種方法。

### 容器被熱風槍融化

熱風槍所吹出的風，溫度約 200 ~ 250 度，非常的高溫。若為了消除氣泡而長時間用熱風槍加熱加入樹脂液的容器或模具，則可能會有融化的情形產生。想避免此情況，可以分批多次使用熱風槍加熱。另外，也請小心熱風槍產生的高溫，避免燙傷。

### 在製作途中樹脂液就凝固了

UV 膠擁有被紫外線照射後就硬化的特性。所以在有陽光照射的房間作業的話，很有可能途中就開始硬化。若想保留染色後的樹脂液，以便日後使用，可以先用鋁箔紙包覆，避免紫外線照射，暫時保存一段時間。

将记载有魔法阵与咒语的卷轴，以及作为术式媒介的橄榄石置于试管内的魔法道具。在打开试管时，魔法便会发动，只能使用一次。
因发动魔法的不同，记载在卷轴上的魔法阵与所需媒介石也会有所不同。

**「龙血结晶」**
由于接触到空气而结晶化的龙血。
虽然也有强化身体能力的效果，但比起龙血，效果可是差了好几个档次，不仅如此，服用龙血结晶所强化的部分与药效时间都是不固定的，所以不适合在实战时使用。

用毛刷將深色系的褐色或青銅色眼影刷上去。

將天然石（圓凸面型磨琢）壓上去。
* 照片中用的是孔雀石。

取適量的軟陶，揉成細繩狀。之後會將這些軟陶用來裝飾藥盒，可先多做一點。

將細繩塗上粉末顏料（青銅色）。塗抹時可以多塗一點，不要塗太薄。

用牙刷輕輕地將多餘的粉末刷掉。

# 16

將步驟 13 ～ 15 做的細繩圍在天然石（圓凸面型磨琢）周圍並固定。可以先將少量的液態軟陶塗在細繩上，這樣會比較好固定。

# 17

將剩下的細繩做成裝飾。這裡將以「凱爾特結」為主體，打一個沒有收尾的結。

# 18

將凱爾特結配置在天然石（圓凸面型磨琢）的周圍。在配置前可以先在凱爾特結或軟陶上塗上液態軟陶。

# 19

切掉凱爾特結的收尾處，使首尾自然相連。

# 20

將 2 條細繩擰成麻花狀。

# 21

配置好步驟 20 後，放入烤箱燒烤加熱。（溫度與時間隨著製造商而有所不同）

# 22

從烤箱取出，待冷卻後，塗上保護漆。

# 23

保護漆乾了之後塗上黏著劑。

# 24

貼上藥盒固定。作品完成。

1. 取适量软陶，揉成球状。

2. 将长柱状的天然石原石插入软陶并使其与软陶密合。*照片中使用的是水光水晶。

3. 观察整体的比例，将多余的软陶去除，尽可能让原石露出表面。

4. 修整形状。

5. 使用牙签，将原石与软陶的接合处刻出裂纹。这个步骤可以使整个作品更像天然产物，并使原石固定在软陶上。

6. 将天然石（圆凸面型磨琢）嵌入。*照片中使用的是蓝色月光石以及堇青石。

暂时将天然石（圆凸面型磨琢）移除，并在凹槽铺上金属箔。

再将步骤7移除的天然石（圆凸面型磨琢）放到金属箔上。

取适量软陶，揉成细绳状。这些细绳状的软陶要用来做装饰花纹，可以先多做一些。

将步骤9做好的细绳状软陶固定在天然石（圆凸面型磨琢）周围。只要能固定住，不管围成什么形状都是OK的。

接下来可以自由装饰花纹。将剩下的细绳状软陶随意贴上。

做好翅膀形状的软陶并贴上。

将软陶揉成小球状后贴上。

装饰花纹完成后，使用褐色系的眼影上色。

用牙刷轻轻地将多余的眼影粉末刷掉。

**16**

用针补足花纹。

**17**

用圆铁笔在左右两侧各开2个洞。这些洞之后会装上要挂上链子的把手。

**18**

取将步骤9做好的细绳状软陶2条，并在切短后涂上眼影。这些软陶之后会作为把手。

**19**

在步骤17所开的洞涂上极少量的液态软陶。

**20**

将步骤18做好的把手插入洞中固定后，使用烤箱烘烤（烘烤时间与温度依据软陶制造商而有所差异）。

**21**

待从烤箱取出的成品冷却后，涂上保护漆。

**22**

将2条项链链子分别用C圈固定问号扣头与收尾帽后，再用O圈将链子固定在作品的把手上。

### Point

使用稍微有点裂痕或有内包物的宝石做出的作品，会比使用完美无瑕的宝石更有天然形成的感觉。
