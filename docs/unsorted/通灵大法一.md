# 序言

中华大地乃世界文明之邦。

历经五千年沧桑锤炼与洗礼的中华文明将以无与伦比的慧光普泽人类美好的未来。

中华文明的瑰宝之一《易经》更在当今的世界上掀起一个又一个的研究热潮。

易学是中国预测学和信息科学的起源和基本理论。

易学是具有中国特色的辩证哲学。是天人合一，超越时空，多方位，多角度，立体交叉的全息宇宙信息源。

易学的内容极其丰富，它涵盖了哲学、社会学、神学等多种领域，它是关于探讨事物发展变化的法则和规律，并有控制、调理事物变化过程的理论。

人们面对着宇宙、社会、环境、事业、生活等诸多问题，无不期待着和谐通达，事事如意。但天有阴晴，月有圆缺，人也难免不如意事常八九。因而化解伤灾、病灾、官灾、破财之灾，调理人的命运就成了易道的终极目的，成为造福人类社会的具体办法。

易经教授、冲天居士李纯文老师酷爱祖国传统文化和易学文化，三十余年不辍努力，数次登上泰山、九华山、五台山，拜师访友，深得道家真传，深谙易学精髓。李老师笔耕不辍，在推出一系列易学著作之后，又著述了此部《通灵大法》，公开了其运用灵验的福人之法。

本书内容多为民间流传和道家秘传的预测鬼怪灵异的方法，其中的破解方法更是鲜为人知的秘传绝学，读后令人大开眼界。读者可在本书指导下深入实践，以期更好的服务百姓，造福社会。

## 星相山法易学研究会简介

大庆市星相山法易学研究会会长李纯文老师研易多年，从师于五台山、泰山、九华山等道家长者，而后游历天下，遍访高人，融会贯通，自成一家。在当今易界鱼龙混杂互相争雄的情势下，李老师不忍看学易之人走许多弯路，甚至上当受骗，决意要公开古圣先贤和江湖民间的千金不传之秘，让中国数千年的易文化遗产得以发掘研究与应用。以无比坦荡之胸怀，奉于世人，乃功德昭著之事业，有志于易文化者得之，实乃有缘也。
