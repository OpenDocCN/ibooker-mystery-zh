# 圓方立極

「天圓地方」是傳統中國的宇宙觀，象徵天地萬物，及其背後任運自然、生生不息、無窮無盡之大道。早在魏晉南北朝時代，何晏、王弼等名士更開創了清談玄學之先河，主旨 在於透過思辨及辯論以探求天地萬物之道，當時是以《老子》、《莊子》、《易經》這三部著作 為主，號稱「三玄」。東晉以後因為佛學的流行，佛法便也融匯在玄學中。故知，古代玄學實在是探索人生智慧及天地萬物之道的大學問。

可惜，近代之所謂玄學，卻被誤認為只局限於「山醫卜命相」五術及民間對鬼神的迷信，故坊間便泛濫各式各樣導人迷信之玄學書籍，而原來玄學作為探索人生智慧及天地萬物之道的本質便完全被遺忘了。

有見及此，我們成立了「圓方出版社」（簡稱「圓方」）。《孟子》曰：「不以規矩、不成方圓」。所以，「圓方」的宗旨，是以「破除迷信、重人生智慧」為規，藉以撥亂反正，回復玄學作為智慧之學的光芒；以「重理性、重科學精神」為矩，希望能帶領玄學進入一個新紀元。「破除迷信、重人生智慧」即「圓而神」，「重理性、重科學精神」即「方以智」，既圓且方，故名「圓方」。

出版方面，「圓方」擬定四個系列如下：

- 1. 「智慧經典系列」：讓經典因智慧而傳世；讓智慧因經典而普傳。
- 2. 「生活智慧系列」：藉生活智慧，破除迷信；藉破除迷信，活出生活智慧。
- 3. 「五術研究系列」：用理性及科學精神研究玄學；以研究玄學體驗理性、科學精神。
- 4. 「流年運程系列」：「不離日夜尋常用，方為無上妙法門。」不帶迷信的流年運程書，能導人向善、積極樂觀、得失隨順，即是以智慧趨吉避凶之大道理。

在未來，「圓方」將會成立「正玄會」，藉以集結一群熱愛「破除迷信、重人生智慧」及「重理性、重科學精神」這種新玄學的有識之士，並效法古人「清談玄學」之風，藉以把玄學帶進理性及科學化的研究態度，更可廣納新的玄學研究家，集思廣益，使玄學有另一突破。

# 自序

「八字秘法」可稱為「江湖訣」，起點是在八三、八四年間我跟一位從事算命行業的師父，花了一點錢學回來的，然後再花了三十年的時間去不斷鑽研，把秘法累積下來。可以找到答案的我都盡力去找，找不到的便唯有放棄。

這些秘法可說是千金難得，即使憑一生努力去研究，也可能得不出答案來。我在此書把一生所學公開，望有志學習八字的後學者能一窺八字門檻，進而把八字的境界提升。

能把這些秘法記得愈多便愈好，因為實際應用時可能在八字中只出現三數個條件，若記得愈多，在判斷八字時可以用於套在被算者的程式之秘法便愈多，愈容易讓對方信服。相反，如八字全無秘法的條件，仍可以用八字寒熱命去判斷運氣、用六神去判斷性格、用妻財子祿去判斷六親，雖不致失去其準確性，然卻不能讓被算者拍案叫絕矣。

## 婚期

婚期並無一定的判斷辦法，尤其現代社會，有些註冊不擺酒，又有擺酒不註冊，有暗中在外地結婚無人知道，有些同居十多年而沒有結婚，有些則已有子女但未註冊結婚……形形色色大不同，故現代判斷婚期比古代難得多。

### 結婚定義

結婚的定義是要按照當地社會風俗習慣而定，在一般發達社會，一定要簽名註冊才算是結婚。另外，已與對方生了孩子也算是婚緣的一種。

### 斷婚期

判斷婚期的最直接辦法是：男命流年財星透干，女命流年官殺透干。其次還有日柱天剋地沖；月柱天剋地沖（一喜擋三災，無喜長輩防）；時柱天剋地沖（一喜擋三災，無喜損傷防）；重桃花年；大運天剋地沖；大運行財、官殺；月柱、日柱、時柱伏刑及轉大運時，都是有可能結婚的，故要準確判斷婚期是不容易的，反而有時判斷失身之期比結婚會更容易。

一般來說，結婚年齡會根據不同社會的習慣，有些較早而有些較遲，好像香港，女性在二十五歲前都不太想結婚，男性很多年過四十歲仍不想結婚。但古代婚期普遍都在二十歲前，故現代人很多會錯過姻緣，因重婚緣運一般十年中只有兩年，就是男命流年天干透財，女命流年天干透官殺。男性還好，五十歲才結婚也不遲，年齡限制較小，有些人姻緣運在二十六、二十七歲，但很多女性會覺得太早而不想結婚，卻原來下一個重姻緣運要待至三十六、三十七歲，在那年紀要找尋合適對象便不容易，於是就這樣錯過了婚緣。

又夫妻星在年柱，則早有婚緣，在時柱、日柱則宜遲婚。
