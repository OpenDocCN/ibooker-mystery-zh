## 前言

《幻星塔罗》是我们推出的一款原创塔罗牌。它继承了传统塔罗牌内容，又加上了新的设计，画风清新独特，简洁可爱。

塔罗牌是发源于西方的一种推测工具，到了今天，已经在世界范围内普及。而眼下流传的推测方法也由最初的繁杂简化到了今天这样浅显易懂的程度。

塔罗牌是在西方文化中生长的，所以，其中包含了西方的哲学、宗教、天文等各种文化成分。从某种意义上可以说，塔罗牌本身就包含了一种人生哲学，塔罗牌里的78张牌，分成不同的组，按顺序影射了人生的各个阶段，你可以从中看到一个充满智慧的人生观。

运用塔罗牌进行推测，可以在一定程度上帮你对生活、工作和学习情况有更清晰的认识，让你更了解自己在某件事情上的立场。

“幻星塔罗”，这个命名来源于塔罗牌主牌中的第十七张“星”。“星”是其中一张含义非常深刻的牌，同时也是作者最喜欢的一张，它代表了内心的希望。所以，“幻星”，即意为点燃梦幻中的希望之星。这正是塔罗牌体现的人生哲学中重要的一环，它是为生命之旅指引方向的星盏，让你到梦想的彼岸。

本书由中国最早引进塔罗书系的北京梵天文化传播中心推出，其一贯精美的出版品质定能使本书增色不少，相信能得到广大读者的喜爱。

正满

### 正位

放下过去已有，开始新事物：

- 1. 积极地去开始，并且享受现在的过程。

- 5. 形容一个天真、不受约束的人。

拒绝拥有新的开始：

- 1. 逃避现实中正等待你去完成的事情。

- 3. 正处在一种困惑状态中，使自己不能开始新的自我，这种状态有待去解决。
- 4. 不敢大胆尝试或者依靠他人的经验，放弃自己的判断和意志。

# 塔罗牌的组成

塔罗牌由22张主牌（大阿卡那）与56张副牌（小阿卡那）组成，共有78张。Arcana（阿卡那）是Arcanum的复数，意思是“丰富的秘密”。

# 1. 大阿卡那

塔罗牌里的22张主牌，称为“大阿卡那”。

大阿卡那牌是用来解释事物的发展趋势和内在状况的，它们告诉我们事情的原因（WHY）；相对地，小阿卡那牌则是用来补足大阿卡那牌的，它们表示的是事物的具体表现方式，是大阿卡那的具象化，它们告诉我们事情的状况（WHAT）。很多情况下只用大阿卡那牌来做推测就可以了。但是想具体了解事情的细节时，可以加上小阿卡那牌。

22张大阿卡那牌是整副牌的核心，它们代表内心最深层的含义，大阿卡那的每一张牌有一个名字和数字编号。有些牌的名字直接指出牌的意义，比如节制、正义和力量，有的牌代表一种特别的身份，比如魔术师、隐士、皇帝和教皇，他们体现了这种身份的内涵，有些牌是天体的名字，如星、太阳和月亮，它们体现了和这些天体相关的意义。

22张主牌可以理解为是一种个人内在旅程的不同阶段的展现——称之为愚者之程（Fool's Journey）。从0到21号牌，按照顺序，体现了我们人生当中一个特定的阶段，它由最初的愚者作为起点，向你描述了一个不断成长、追求生命意义的人生历程，并且从中启迪人生的智慧，到了最后的一张“世界”，以获得生命的真实作为结局。

# 2. 小阿卡那

小阿卡那有 56 张牌，分成权杖、圣杯、宝剑、星币四个种类，它们分别由侍从、骑士、王后、国王，以及一（第一张牌即首牌，也称为 ACE 牌）到十的牌面所构成。

小阿卡那将大阿卡那牌的含义带入实用的领域来具体表示出日常生活中的运作。每一组的每一张牌代表一种具有自己含义的生活状态。

小阿卡那是直接与四大元素（火水土风）相关的，四个元素对应四个牌组：

- 1. **权杖 (Wands)** —— 火 (Fire)
  - 创造力、行动力、自信、活力
- 2. **圣杯 (Cups)** —— 水 (Water)
  - 情感、精神、内在、直觉、亲密
- 3. **宝剑 (Swords)** —— 风 (Wind)
  - 知性、思想、正义、真相、原则
- 4. **星币 (Pentacles)** —— 土 (Soil)
  - 实际、安全感、物质、繁荣、才干

## 首牌

每个首牌 (ACE 牌) 显示了这一组牌的主题，带有这组牌的掌控力量。

## 数字牌

2—9 的牌代表的是这一组牌的不同状况，是将这组牌主题的各种状态详细展开。每张牌都有其独特含义。

![](img/d25d018b83958c4b01864d808f4efcb0_116_0.png)

个方面以具体形式表现出的牌，并且是发展过程展开。

- 第一张：开端、基础
- 第二张：计划或者准备
- 第三张：开始进行实施
- 第四张：巩固当前状况
- 第五张：转变矛盾冲突
- 第六张：从第五张显示的冲突中得到释放
- 第七张：坚持、不放弃
- 第八张：超越当前，开始更深层次的追求
- 第九张：得到收获
- 第十张：最终的结果

## 宫廷牌

宫廷牌是指那些代表牌组的人格特质的牌。分为国王牌、王后牌、骑士牌、侍从牌。

## 国王牌

表示通过将这组牌代表的特质正确发挥得到成功。在人格上他是一个王者，是成熟男性的象征。他在许多领域展现权威、控制和掌握。国王的风格是强大的、具有创造性的、具有力量的。

## 王后牌

表示通过对这组牌所代表的特质的深层理解而取得成功。她是一位成熟的女性。她将这一组牌的特质具象化。她不像国王那样充分展现强大的力量，她的重点是在内在层面，她是乐观和自然的。

# 骑士牌

表示将这组牌的特质付之行动。

他是一个热血的青年。所以他是行动派的，他是精力充沛的，他还无法平衡地掌握自己。他将这组牌的特质表现得非常极端，他是处在一个刚开始的过程阶段上。

# 侍从牌

表示在这组牌的特质下的开端，通常代表计划和消息。

他是一个天真的小孩。他表现的是这组牌快乐和放纵状态下的特质。他的方式是简单而自然的，他也代表着潜力和可能性。

小阿卡那牌也是扑克牌的前身，四组牌对应了四个花色：

- 权杖（Wands）——梅花（Clubs）
- 圣杯（Cups）——红桃（Hearts）
- 宝剑（Swords）——黑桃（Spades）
- 星币（Pentacles）——方块（Diamonds）

扑克牌中也保留了宫廷牌等：

- J——侍从牌
- Q——王后牌
- K——国王牌
- A——首牌

# 塔罗牌的推测方法

## 准备

选择一个合适的环境来做推测。这个环境必须安静舒适，能让你的心情平静。例如一个安静而光线祥和、布置舒适的小房间——比如自己的房间。但是只要你自己觉得合适，能够安心地推测，什么样的地方都行。

另外就是推测者要有一个良好的状态。首先，要抱着正确的态度来进行推测，要相信和尊重塔罗牌，能够客观看待即将推测到的结果，千万不要拿它当游戏一样玩，最忌讳纯属无聊来推测一些毫不正经的东西，以求一些自己根本不相信的结果消遣。

接着，要强调的是，请在头脑很清晰的时间推测，这也就是为什么很多人喜欢在晚上推测的原因，晚上人的精神是很放松的，当然，不能在打瞌睡的时候进行。推测要冷静、专心和警觉。只有当你冷静和集中精神时，才能使你的思路清晰。

接下来，找一张足够宽的干净的桌子，还可以准备一块干净的布铺在桌面上，这样让牌更不容易变脏。推测用的桌布最好是黑色，黑色是传统推测者采用的颜色，因为黑色最能聚集能量，而且至少黑色会使你的牌被衬托得更加清楚而且不像一些耀眼的颜色那样有强烈的反光，其他颜色的桌布也可以，只要你自己觉得舒服。

## 提问

首先，当然是确定自己要推测什么，问一个问题，而这个问题在进行下一个步骤——洗牌之后就不能更改了。

塔罗牌将告诉我们的是一件事情的各个方面状态和发展状态，而不是告诉我们如何去做。所以，对于我们想要了解的东西，怎样提出问题，也需要技巧。

首先，提问越具体，答案也就越具体。对于太过笼统的问题，塔罗牌是不能给你具体答案的。如：我的命运如何？也许有人会说:那是我最想推测的问题啊！但是命运是掌握在自己手中的，不是吗?或者问:我的事业如何？塔罗牌也是可以为你解答的，只是它给的答案跟你的问题…一样的笼统。所以，如果想知道具体的答案，那就请把问题问得更具体一些吧。比如说:我现在进行的工作状况如何?我想要转业的事情是否有着落等。再比如说：我跟恋人之间的关系会如何进展？我的学习计划会实施得怎样？
不要推测很久以后的事情，塔罗牌最适合的是帮你了解一件正在发生中的事情的详细状况和前因后果。
不要问一些时间方面的问题。特别是对于初学者来说，这是很难的。塔罗牌一般不会给你一个明确的时间。
不要问一些让塔罗牌做选择的事情，塔罗牌只告诉你事情的状态。它不能代替你的主观分析。
最后，不要希望塔罗牌去帮你解决问题，它只是帮你分析事情，让你可以更好地解决而已。比如说，不要问：我要怎样做才可以解决这个问题？应该问：我在解决这个问题时会有什么需要我注意的事情？

# 洗牌

洗牌是很重要的，这直接关系到你将选出的牌。

洗牌的方式有许多种，没有所谓最好与最坏的方法。应该选择一种自己觉得最顺手的方式。

洗牌之前要先确定自己将要使用的牌阵。洗牌时集中注意力，想着你所要问的问题。

下面介绍几种常用的洗牌方法：

1. 这是最常用的方法，对一般的初学者很适合。首先将牌在桌子上摊开，全部牌面向下，打开双手，以顺时针方向将牌搅动，使牌充分地被打乱，这时候，倾听自己心里的声音，直到自己觉得可以了，就可以停止，将牌收放成一叠。注意整理的时候也是要以顺时针的方向转动，让牌顺着转动慢慢地收回成叠，注意是横着的一叠。
2. 将牌拿在平时习惯用的那只手上，用另一只手从那叠牌中抽出一叠，然后可以凭感觉随机将抽出的牌旋转180度，再放回那叠牌的上方，反复重复这个动作，直到自己觉得可以结束。
3. 可以像洗扑克一样，将牌分成两叠，都是牌面向下。将两叠牌的两端相向提起，再落下，牌就可以充分混合了。反复进行几次直到自己认为可以为止。
4. 你可以根据自己的喜好，在合理的情况下更改这些洗牌方式而变成自己独有的方式，也可以完全自己新创造一种。但无论如何，不要忘记了洗牌的前提，要充分打乱、混合，并且要能随机打乱正逆位。

# 切牌

洗完牌之后是切牌。你帮别人推测时，切牌可以由求问者进行。先介绍一下最常用的切牌方法。
首先，从牌组中随意取出上方的一叠，放在下方。
再从这下方的一叠牌中取出上部的一部分，放到这两叠牌的上方。
再将中间的那叠牌重叠放到下方的那叠牌上。
最后将这下方的牌放到上方的牌组上。
再将这叠牌顺时针旋转90度就完成了。
如果是给别人推测，因为一般来说被推测者是坐在推测者的对面，所以最后转牌时要逆时针旋转90度。
也可以将先洗完的牌旋转90度使其成竖直向，再按上述方法切牌。

# 抽牌

切完牌之后，就可以将所需要张数的牌抽出了，抽牌的方法也是有很多种的，并且可以不必拘泥于形式。

下面介绍几种方法：

1. 凭感觉随意抽出。
2. 将牌旋转开，可以展开成一圈，选一个号码，依照号码点牌，将点到的牌抽出。
3. 依照顺序直接将牌自上方或下方抽出。
4. 其他（如书中附注的方法或自己觉得适合的方法）。

# 摆牌阵

将抽出的牌按顺序照牌阵摆好。

# 解牌

将牌一张张打开之后，接下来就是解读了。
这个很关键。
首先，整体地看一遍所有的牌，感觉一下它们给你的第一印象，感受一下这些牌带给你的特殊感觉。然后，再具体地去分析牌与牌之间的联系和每一张牌的含义。

# 加深对塔罗牌的认识

## 1. 塔罗牌的正逆位

洗牌的时候，牌的方向是混乱的。而后在摆牌时，面对推测者，就出现了正放的牌和倒放的牌。实际上，最初的塔罗牌是没有正逆位之分的，都以正位的意义去解释。后来有推测者独创了包含逆位意义的方法，逐渐成为塔罗牌推测的通用规则。到了现在的塔罗牌推测，牌义都是有正逆位之分。

逆位牌表现了正位牌能量的潜在性，是因受到限制没有得到充分发挥，或者处在没有完全发挥的时期，或为时尚早，或已经过了那段时期。

### 逆位牌的解释一般有几种情况

- 缺乏正位的能量
  - 如节制的正位意义为平衡和健康，而逆位意义是不融合、不平衡和浪费。
- 时间差距导致的转变
  - 如宝剑八正位代表着束缚，而逆位代表着挣脱束缚。
- 抵抗正位牌的能量
  - 如死神牌，它的正位代表一件旧事物的消亡，而逆位牌则可以解释为抗拒将被消亡的潮流，抓住过时的事物不放。宝剑三的正位意义为接受悲伤，而逆位则是抗拒悲伤，自欺欺人，麻痹心灵。
- 缺乏前一张牌的意义
  - 在小阿卡那的四组牌，以及大阿卡那中，都是以数字顺序排列的，而按照这个顺序，每一个牌组都表达着人生某个方面的成长和发展，后一张牌的牌义是建立在前一张的基础之上的。所以大阿卡那牌和小阿卡那牌的倒立，都暗示求问者需要从前一张牌的正立意义中去寻求答案。

# 欢迎来北京

# 1. 牌阵类型

塔罗的推测方法是以牌阵的形式将选出的牌排列好，然后再根据牌的位置及牌的内容来做分析解读，从而得知推测结果。塔罗牌是告诉我们事物发展状况的工具，通过牌面的信息来表达。

塔罗牌的牌阵里有只使用大阿卡那的牌阵，有只使用小阿卡那的牌阵，也有大小阿卡那混合的牌阵。

一般只使用大阿卡那的牌阵有：凯尔特十字、圣三角、二选一、时间之流、六芒星、吉普赛十字等只用少量牌的牌阵。

一般只使用小阿卡那的牌阵，下面介绍到的有维纳斯之爱。

一般大小阿卡那混合使用的牌阵有：七行星、生命之树、黄道十二宫等要用到较多牌的牌阵。

实际上，可以根据具体情况和需要，或者根据推测者的习惯等，同一个牌阵可以使用大阿卡那也可以使用小阿卡那或者混合使用。大阿卡那着重表现的是事情的内在状况和原因，而小阿卡那则着重表现事情的具体状况。

# 2. 指示牌

在我们开始介绍牌阵之前，先介绍一下这个名词。

很多推测中都要用到指示牌，我们之后要介绍的经典牌阵中也会用到指示牌。那什么是指示牌呢？所谓的指示牌，就是在塔罗推测中用来代表提问的人或所要问的问题的牌，整个牌阵最后的结果配合指示牌作出更加准确的解读。

在早期的塔罗推测中，指示牌的使用具有举足轻重的地位。而在今天我们习惯使用的推测方法中，很多时候并不用到指示牌，特别是对于初学者来说，指示牌的使用需要一定的经验。但是，掌握指示牌的使用方法，对塔罗牌推测的准确性很有帮助。

我们推测一件事情，很多时候并不是只为了了解这件事情的客观事实，我们更希望知道的是事情对我们的影响。而事情对人的影响取决于人的身份和立场。所以，指示牌就在这里指出了你的身份。

下面我们的问题是，如何挑选指示牌？

在洗牌之前，从16张宫廷牌中选出一张代表求问者的牌作指示牌。可以由推测者挑，也可以由求问者挑，根据具体情况而定。

下面说几种常见的确定方法：

### 1. 根据求问者的身份

- 成熟的男性——国王牌
- 成熟的女性——王后牌
- 年轻的男性——骑士牌
- 年轻的女性和未成年人——侍从牌

### 2. 根据性格

- 大胆创新，热情活泼，勇敢外向——权杖组
宽容慈善，感情丰富，亲切，敏感——圣杯组
坦诚直率，理智，机警，知识渊博——宝剑组
沉稳实际，谨慎细心，值得信任，忠诚——星币组
（在用这种方法选择时，要熟悉各个牌组所代表的人物性格。但是，这个方法只对推测者了解的人或自己适用，对大多数我们不了解的人，则很难选择。）

### 3. 根据太阳星座

- 白羊座、狮子座、射手座（火象星座）——权杖组
- 金牛座、处女座、摩羯座（土象星座）——星币组
- 双子座、天秤座、水瓶座（风象星座）——宝剑组
- 巨蟹座、天蝎座、双鱼座（水象星座）——圣杯组

### 4. 由求问者自己抽取

先将16张宫廷牌放在一起，让求问者洗牌后抽出一张。

还可以先把16张宫廷牌翻开，让求问者从中选出自己认为合适的部分牌，洗牌后抽出一张。

或者把16张宫廷牌都翻开，让求问者自己凭感觉挑选。但这种方法容易产生主观意志导致的偏差。

最后，补充一下切牌：

在较早的塔罗牌推测里，指示牌的选择范围限于16张宫廷牌。后来，又扩大到大阿卡那牌，因为大阿卡那牌也可以具体地代表人物的性格和立场。而指示牌也不仅仅代表求问者，也用来代表所要推测的事情。所以指示牌的选择范围和代表范围都扩大了，只用宫廷牌和大阿卡那表示已经难以满足，所以，又出现了切牌。在今天的推测中，指示牌是可以用切牌来代替的。切牌的选择范围是所有的78张塔罗牌。任意一张都可以来代表求问者或所要推测的问题，而指示牌的范围是22张大阿卡那与16张宫廷牌。

# 3. 经典牌阵

## 塔罗单张推测法

这是所有的推测法中最简单的一种。
这种推测方法非常自由，用途也非常广，可以用于各方面的问题，但是相对地解释也比较笼统。
通常情况下，单张牌的推测只需要使用 22 张大阿卡那。
推测步骤跟其他推测一样，要先洗牌、切牌后，将牌面向下摊开，由求问者任意抽出一张。
解读时，直接参照原本的牌义跟正逆位方向即可。
这种方法最常见的就是用于了解每日运势。
此外，还可以根据牌的正逆位来判断一个问题 YES/NO 的答案，正即 YES，逆即 NO。

## 时间之流

这个牌阵是个运用非常广泛的牌阵，适用于了解一件事情在不同时期的不同情况，可以预测事情的未来发展。

- 1: 代表过去，已经发生的事情
- 2: 代表现在，事情的客观现状
- 3: 代表将来，如果事情自由发展将会出现的结果

切牌或指示牌位于下方。切牌表示求问者的心态，属于主观状况。指示牌则代表了求问者的特质，属于客观状况。

解读时，首先解释切牌或指示牌的意义和影响，然后依照时间的顺序逐张翻开解读，最后，根据所有的牌作出总结，得出结论，提出建议。

## 钻石

这个牌阵是对问题的过去、现在及未来的情况作解释的，也是一个运用非常广泛的牌阵。

- 1: 代表过去的状况
- 2、3: 代表目前的情形，两张牌需综合解释
- 4: 代表最后结果

比上一个牌阵多了一张表示现在的牌，要注意结合理解。

## 身心灵

身心灵牌阵适用范围广，而且非常简单，容易掌握，所以成为了使用最频繁的经典牌阵之一。它常被用来预算运势。

- 1: 身体状况，即身
- 2: 心理状况，即心
- 3: 精神状况，即灵

这个牌阵也可以用于推测两人之间的关系，身表示生活和相处情况，心表示两人之间的内心与交流，灵代表两人之间的精神与感情。

## 平安扇

平安扇牌阵适用于对人际关系，特别是与某一具体的人之间的关系进行分析。这个牌阵简单易学，所以被广泛应用。一般可以只使用大阿卡那来进行推测。

- 1. 表示当事人目前的人际关系状况
- 2. 表示当事人与对方关系存在的因素
- 3. 表示双方关系未来的发展趋势
- 4. 表示双方关系的结果

解读这个牌阵时，因为每一张牌代表的因素间都是息息相关的，所以要注意牌之间的结合理解。

## 四要素

四要素牌阵的适用范围也非常广，生活中，很多事情都要我们预先准备。四要素牌阵就是这样一个用来分析在未来可能会出现的某个状况的经典牌阵。

西方所谓的四大元素，即水、火、土、风。每种元素都有属于自己的特质。

- 1：火元素，代表在将来特定时期出现的努力方向及精神追求
- 2：土元素，代表这时的物质方面的状况
- 3：风元素，代表这时的障碍、可能受到的伤害
- 4：水元素，代表这时的人际关系与感情状况

## 二选一

- 1：求问者的当前状况
- 2—3：不同的选择
- 4—5：不同的选择导致的不同结果

## 三选一

与二选一牌阵类似，在其基础上做了增加与补充。

- 1: 事情的现状
- 2: 事情发展遇到的障碍
- 3: 周围的环境影响
- 4—6: 不同的选择
- 7—9: 不同选择导致的不同结果

在解读时，前三张牌和指示牌的解读对后面牌的解释尤为重要。

## 恋人三角

这是个适用于推测爱情问题的牌阵。

- 1: 求问者的本身情况及内心想法
- 2: 对方的情况及内心想法
- 3: 两人的感情现状
- 4: 与对方的关系及感情的发展情况

## 六芒星

六芒星牌阵的布局呈六芒星状。摆牌时，也是按照六芒星的形成顺序：先用前三张牌摆成倒三角形，再用后三张摆成正三角形与前一个重合。

- 1. 已经发生的过去
- 2. 正在发生的现在
- 3. 尚未发生的未来
- 4: 事情发生的原因
- 5: 影响事情的环境
- 6: 应当采取的对策

切牌表示推测当事人的主观想法，指示牌，表示当事人的客观特质。也可以在放指示牌或切牌的位置以第七张牌代替，第七张牌表示事情的最终结果。

## 圣花猫

抽牌时，不随机抽出，而是将牌组从上向下数：

- 第7张牌放在位置1。
- 第8张牌放在位置2。
- 第9张牌放在位置3。
- 第10张牌放在位置4。
- 第11张牌放在位置5。
- 第12张牌放在位置6。

- 1、2：从过去到现在的状况
- 3、4：从现在到未来的状况
- 5、6：从未来的发展到最后的结果# 开创杯

# 向日葵

![](img/d25d018b83958c4b01864d808f4efcb0_147_0.png)

当我们想解答一个并不是很具体的问题时，可以采用这个牌阵，它可以让你对这件事情有一个大致上的全方位了解。

牌阵呈一朵葵花的形状，由左至右展开。

+   1. 求问者在这个问题上的特性

# 时间金字塔

![](img/d25d018b83958c4b01864d808f4efcb0_148_0.png)

![](img/d25d018b83958c4b01864d808f4efcb0_148_1.png)

这个牌阵是针对事情的发展状况使用的，可以帮我们了解一件事情在一段时间内的不同情况和发展趋势。解读比较复杂，需要结合多张牌解释一个意思。

时间金字塔牌阵的布局呈三角形，就像一座金字塔，它由十张牌组成，分为四个层次。每一个层次内的顺序是由左至右。

+   - 第一层 (1—4): 事情的过去状况
    - 第二层 (5—7): 事情的现在状况
    - 第三层 (8—9): 事情的未来情况
    - 第四层 (10): 事情的最终结果

切牌的位置习惯上位于金字塔牌阵的右侧，略高于第二层。

# 七行星

![](img/d25d018b83958c4b01864d808f4efcb0_150_0.png)

这个牌阵适用于我们对一件事情进行综合分析，了解各个方面情况。

七行星是古代星象学里的七大行星，这七大行星代表着特有的含义。

七行星牌阵中的七张牌等距离地围为一圈，切牌或指示牌位于牌阵的中央，切牌代表的是求问者的立场，指示牌则表示求问者的特质。

+   1: 月亮——代表家庭状况、安全感
    2: 水星——代表商业贸易或者沟通交流的情况
    3: 金星——代表人与人之间的关系和内心感情
    4: 太阳——代表社会地位和人际交往中的表现价值
    5: 火星——代表冲突、竞争或行动力
    6: 木星——代表机会、精神方面与物质方面的获取
    7: 土星——代表精神状况

# 吉普赛十字

![](img/d25d018b83958c4b01864d808f4efcb0_151_0.png)

吉普赛十字阵是专门针对爱情问题进行分析的牌阵。爱情问题在塔罗牌推测中出现非常频繁，所以这个经典的牌阵运用得非常广泛。

这个牌阵不要用到切牌或指示牌，也不采用抽牌的方式，而是将切好的牌摆成一个方阵，方阵的大小可随意，没有规定。然后只将中央呈十字的五张牌翻开进行解读，做出最终判断。

揭牌时，首先是十字的最上一张，然后是十字的最下面一张，接着分别是左边和右边，最后是正中央的那张。解牌时，翻开一张，解释完毕，再翻开下一张进行解读。这样完全解读后，再综合全部信息作出总结。

+   1: 求问者的心态
    2: 现在的状况
    3: 采取的举措
    4: 周围的环境
    5: 未来的状况

# 凯尔特十字

凯尔特十字阵是最经典的牌阵之一，被广泛地运用。它可以让我们对一件事情的各种因素透彻地分析。

+   1: (纵向摆放，压在指示牌之上) 表示问题的客观现状
    2: (横压在第一张牌之上) 表示影响现状的各种积极或消极的因素（将这张牌顺时针旋转90度后的方位即是牌的正逆位）
    3: 对将来的理想、求问者的主观愿望
    4: 事情的过去状况，形成这种局面的基础
    5: 事情的过去情况
    6: 事情的未来发展
    （7—10位置的牌如图微微倾斜，但不影响方位。）
    7: 求问者的自我状况，即指示牌和左面呈十字图案的六张牌的综合信息
    8: 代表环境对事情的影响，或者周围的人群对事情的看法
    9: 希望或恐惧
    10: 最终结果

![](img/d25d018b83958c4b01864d808f4efcb0_153_0.png)

解牌时，要遵循时间顺序。首先要解读代表过去的基础牌，再解读代表现状和影响的牌，然后是代表理想和未来的牌，再然后以代表自我状况的牌对前面的分析进行概括和补充，之后是代表环境因素和希望的牌，最后根据前面所有的分析结合第10张牌得出结论。

# 維納斯之愛

![](img/d25d018b83958c4b01864d808f4efcb0_155_0.png)

维纳斯之爱牌阵适用于爱情方面的推测。对恋人双方的情况描述比较细致。

在西方的传说中，代表金星的维纳斯是爱的女神，维纳斯的代表符号是♀，如维纳斯之爱牌阵的阵形。

维纳斯之爱牌阵只用56张小阿卡那牌进行推测，不使用切牌或指示牌，也不使用大阿卡那牌。

+   1. 求问者的主观看法
    2. 对方对求问者的态度
    3. 求问者对对方的影响
    4. 对方对求问者的影响
    5. 两人之间的障碍
    6. 感情的最终结果
    7. 求问者最终将对此抱以的态度
    8. 对方最终将对此抱以的态度

解牌时，一般不完全按照抽牌的顺序，在按顺序解读完前四张时，应先解读第七和第八张，最后再是第五和第六张，因为这样可以根据先解读的牌来作推导。

# 生命之树

在生命之树牌阵上10张牌对应10个天体。生命之树牌阵是经典的牌阵，但相对的，解读比较抽象、深奥，展现的是事物各个方面的因素和状况。这里还要用到一张指示牌或切牌，牌洗好切好后，将10张随意抽出，放在1—10的位置上，翻开解读。这时，解读的顺序是从下到上，从第十张至第一张。最后，再在没有用到的牌中抽出一张，放到11的位置。

+   - 1：王冠——海王星
    象征最高理念，代表事件的理想或本质以及问题的起因
    - 2：智慧——天王星
    代表创造力、知识与潜力
    - 3：理解——土星
    代表心灵的智慧与悟性，表示智慧在现实中的地位和作用
    - 4：仁慈——木星
    表示机会、事件中的有利因素、别人给予的援助
    - 5：严肃——火星

    - 7：胜利——金星

# 黄道十二宫

![](img/d25d018b83958c4b01864d808f4efcb0_159_0.png)

这个牌阵非常适用于推测一个人在特定时期各个方面的状况和运势。这个牌阵与星象学相结合，星象学中解释道：太阳以一年为周期在黄道上自西向东运行，在宇宙中，黄道经过的这个面上分布着十二星座，就是白羊座、金牛座、双子座、巨蟹座、狮子座、处女座、天秤座、天蝎座、射手座、摩羯座、水瓶座、双鱼座。

在星象学中，黄道十二宫本来就具有各自的代表含义，这个牌阵直接引用了这些含义。牌阵的形状是一个圆，十二张牌环列指示牌之外。每一张牌对应一个宫位。

+   1. **第一宫（命宫）——白羊座**
    代表特质、性格表现以及给人的第一印象

+   2. **第二宫（财帛宫）——金牛座**
    代表物质财富

+   3. **第三宫（兄弟宫）——双子座**
    代表环境影响、学习能力、沟通表达能力

+   4. **第四宫（田宅宫）——巨蟹座**
    代表事物的根源、家庭状况

+   5. **第五宫（子女宫）——狮子座**
    代表对感情的态度、自我表现与娱乐

+   6. **第六宫（奴仆宫）——处女座**
    代表工作、劳动和健康

+   7. **第七宫（婚姻宫）——天秤座**
    代表对婚姻、社交、合作等关系的态度

+   8. **第八宫（疾厄宫）——天蝎座**
    代表“再生能力”，具体显示财产、投资和性等方面

+   9. **第九宫（迁移宫）——射手座**
    代表对深层心智能力的追求，具体显示在对信仰、深造、事业等方面的追求上

+   10. **第十宫（官禄宫）——摩羯座**
    代表自我价值、事业、社会地位的实现

![](img/d25d018b83958c4b01864d808f4efcb0_160_0.png)

![](img/d25d018b83958c4b01864d808f4efcb0_161_0.png)

+   11: 第十一宫 (福德宫) ——水瓶座
    代表对他人的态度、人际关系、理想、精神追求

+   12: 第十二宫 (玄秘宫) ——双鱼座
    代表未知的力量或弱点、潜意识和牺牲奉献

在这个牌阵中，第一、五、九宫代表火元素，是一个象征行动的组合。第二、六、十宫代表土元素，是象征物质的组合。第三、七、十一宫代表风元素，是象征精神的组合。第四、八、十二宫代表水元素，是象征内在的组合。所以，解读时这样分组，有利于更好地理解。
