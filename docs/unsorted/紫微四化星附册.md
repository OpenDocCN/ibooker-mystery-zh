# 前言

这本日记簿是《紫微四化星——如何改变一生的秘笈》的随书赠品，虽是赠品，也希望是有威力和有帮助的，紫微四化星，一旦能融入你的日常生活，你更能发现其中奥妙无穷的乐趣，以及自我灵敏度的提升。

如果你能持续写完一整本，那你将更容易理解、活用「紫微四化星」所带来的改变的力量。

日记簿中除了由读者们自己练习外，我也提供自己十天干的练习。从甲日到癸日，一年当中可能各出现三十次甲、乙、丙、丁至癸天干。虽然周而复始，但每一次各天干出现的时候，都有不同的现象，或因流日地支不同而不同。也有可能遇到相同于本命、流月的天干，但是在一日之末，回看一天总总，藉四化星的意义参考而写出的「当日状况」，有下列几种现象：

- 一、有「原来如此」的体会，因此好的坏的都接受。
- 二、下一次逢相同天干之前，自然想到让自己过得更好的方法。
- 三、自省是件快乐的事情。
- 四、写四化日记，除了帮助自己，也能帮助别人理解四化星，帮助他人。

接下来我从2021年1月6日开始写了一些如何使用的范例。

1月6日那一天是庚子年己丑月甲寅日，正是甲日，到1月15日，正好是一个完整的十天干。我以这十天示范如何每天写四化星日记簿。

所以，读者写自己的四化星日记时，请先查网络万年历，找出当天的日干。

例如庚子年戊子月己酉日，这个「己」是己日，那就请在这个日记本里找出左上角注明「己天干」那一天的页面，在右上角空白处填上当天日期以及流年流月流日，然后就在那一天的页面上写四化星的日记。
