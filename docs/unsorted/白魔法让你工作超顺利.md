# 作者序

一開始想要寫這本書，是因為多年來我在身心靈領域，獲得相當大的療癒力，進而想要跟讀者分享我所學過的「心靈自癒方式」。讀者將可以藉由書中的問題，找到與自身對應的魔法配方，可以說是從遇到的「問題」中尋找「解答」。藉由改變對事情的想法產生的意念，再加上觀想宇宙高能量、大自然元素的幫忙，從「想」促進所祈求的願望在現實中發生！

如果能從每天的生活中，尤其是我們一天中最重要的場域，職場，找到個人內在真實的力量，就可以將美好感受帶進我們的人生中。

我個人認為，我們在書中使用的魔法比較像「自然魔法」，也就是與大自然合作，進而改變自身想法甚至是未來。我們將會使用到石頭、花朵、水晶、植物與精油等隨手可得的物質，再加上自己內在的力量與意念，如此一來，就可以執行「正向能量」的簡單魔法了。

書中參考了現代巫師們會用的「魔導書」相關的資料，再使用其他占卜工具，透過我的解讀及設計，屬於我個人的專屬魔法配方就誕生了。相較於一些巫師，在本書通稱為「魔法師」，他們喜歡古代繁複的魔法儀式，而我個人則傾向簡單且能隨手取得的自然界物質，將繁雜的儀式簡化，讓讀者運用簡單的儀式，就能達到相同效果。

而且我是女性，所以我的魔法較能呼應古代的原始「女神力量」，使用女性的超敏銳直覺，搭配「地球母親(註)」的神聖自然能量，運用以產生能量。女性和男性頭腦運作不大相同，女性天生具有超凡的直覺與想像力，並擁有母性與美感，男性則是具有邏輯、推理與雄性的力量。男巫師和女巫師的魔法也相差甚大，男巫師的魔法具有條理、邏輯與語言等陽性特質，女巫師則傾向直覺、靈感與非語言。但相同的是，巫師們都能使用自身能力，有意識地將大自然與我們體內的陰陽能量相互調和。（註：「地球母親」是對孕育與滋養人類的「地球」的暱稱。書中儀式裡用「母神」，像母親的神來稱呼她。）

古時候因為生活環境單純，且與大自然的連結相當密切，因此古時候人類的生活同時也與本身直覺、自然環境所透露的訊息有很大的關聯，他們也自然而然的發展出許多跟地球萬物合作的智慧； 反之，現代人因為生活忙碌，與本該貼近的自身直覺力相當疏遠，跟周圍環境的人、事、物的關係更遠。如果想要練習魔法，除了培養自身直覺力與多親近大自然外，還能養成正面思考的能力，並藉此夢想成真。

本書的魔法配方重點放在「意念」，人類的意念就是「魔法」的基礎。佛家說：「一切唯『心』造。」意念就是我們的想法。自從牛頓發現了「萬有引力」後，也開啟了人類對於「像磁鐵一般」的「吸引力法則」的好奇。人的「意念」是我們用以顯化願望，並達成願望的「工具」，萬事萬物都有「因果」，現在的一切所得是果，以往所做的事即是種下的因，種因得果。因此，想要未來更美好，「現在」就要 do something，才能有意識地改變未來。

你可以很自我的宣告：世界都「繞著我轉」，從靈性的角度來看是對的，因為環繞在你身邊所發生的每一件事，的確都是宇宙或是你的「高我」，要讓你自己體驗人生課題的方式，這些經驗都是你吸引來到自己身邊的。所謂的「高我」，它位於我們頭頂，距離約是你的手臂垂直向上伸所能觸碰的高度，也有人稱之為「第八脈輪」，它是我們跟「宇宙源頭」（書中稱之宇宙父親）連結的能量點，本質是超越人類肉眼所見的觀點，又稱為「超意識」。

是的，「事出必有因」，你買了這本書，正是你想要做出正面改變的最好證明。請一定要幸福喔！事實上，你已經正走在掌握幸福的路上了。

## 03 | 來自印度的人體七脈輪

「人體七脈輪」是源自於印度的傳統人體構成概念，不僅盛行在印度的瑜珈術中，並散播至世界各地。它說明了環繞人體的靈妙體（非肉體）的能量狀態。

在此特別提到脈輪，是因為魔法需要「觀想」讓事情由「想像」發生在「現實」層面，所以「觀想」與願望相對應的脈輪顏色，將有助於施行魔法。以下讓我跟大家說明七脈輪的對應位置及其意義。

海底輪（紅）：主掌肉體、生命力與生存。海底輪處於活躍狀態，你會感覺到有安全感與穩定感。但過度活躍會有過多的貪慾，活力不足時則會沒安全感、沒企圖心，也會感到疲倦甚至是恐懼。這個脈輪堵塞時，你可能會有自律神經失調、氣血不佳、容易疲倦等相關問題。

膀輪（橙）：主掌情感與性慾。膽輪處於活躍狀態的話，你在情緒表現上處於開放態度，對於性與親密關係的態度也會是開放的。如果膽輪太過活躍，會導致你過於情緒化，反之，不足則會對於他人呈現出冷漠、不關心的態度。這個脈輪堵塞時，你可能會有腸胃、性冷感、或是子宮等的問題。

太陽神經叢（黃）：主掌人如何在團體裡、社會中呈現自己。這個脈輪活躍時，你會很有自信，顯示出你和周遭互動良好，並且正面的自我表達；當這個脈輪能量低落時，你會顯得自卑且猶豫不決。但若是太過活躍，你會出現過度掌控欲，還會顯得激進與自私。當這個脈輪堵塞時，你可能會有胃的毛病出現。

心輪（綠）：主掌愛與被愛的能力，「無條件的愛」也是從這裡開啟。這個脈輪活躍時，你可以信任他人，還能愛人與感覺被愛；活躍力不足時你會對人猜忌、感覺麻痺，還會關閉自己的心門，不接受他人關心。若是太過活躍時，人容易濫情，且會模糊與人的界線。當這個脈輪堵塞時，你可能會有心臟方面的問題。

喉輪（藍）：主掌表達自我的能力。這個脈輪活躍時，你可以自在地跟人溝通，也表示你比較有創意；當活躍力不足時，你會壓抑自我。如果太過活躍，你在表達上會咄咄逼人，甚至自以為是。當這個脈輪堵塞時，你可能會有喉嚨、甲狀腺等問題。

眉心輪（靜）：主掌直覺、第六感的能力。這個脈輪活躍時，你會容易相信直覺，並且具有想像力、洞見，也容易將夢想轉化為現實；若活躍力不足，容易缺乏對事物的聚焦能力，或是缺乏分析事物的邏輯。當這個脈輪堵塞時，你可能會有頭痛、容易做噩夢等問題。

頂輪（紫）：主掌我們的靈妙體，以及跟宇宙連結的能力。當這個脈輪活躍時，你容易信任自己和你內在靈性的指引；當活躍力不足時，你會沒有自信，容易受到「自我」的左右，也輕易就開始擔憂、恐懼，甚至是緊張。當這個脈輪堵塞時，你可能會有失眠、判斷力不佳、不易信任人的問題。

脈輪在正常狀況下是像彩虹一樣擁有七種顏色，從腳底到頭頂，依序是：紅、橙、黃、綠、藍、靛、紫。但是在現代忙碌的工商社會裡，我們不一定每一輪都是平衡的，也許你的工作量與壓力最近比較大，所以你的海底輪可能會失衡、甚至縮小，它的能量也會影響你與金錢的關係；如果最近你的情緒比較多，無故擔憂恐懼，甚至壓抑情緒，你的膺輪可能就是失衡狀態，腸胃或子宮也可能受到影響。因此，當你了解脈輪的重要性，就會更加重視自己身心靈的健康，進而提升你施行魔法的觀想能力。

## 01|準確率極高的塔羅牌

最早的塔羅，據說源自於古埃及，但根據目前考據得到的最早歷史資料顯示，塔羅最早出現於十五世紀，是當時義大利貴族的「紙牌遊戲」。在西方人眼中，他們文化的源頭：希臘和埃及，都是古老、並擁有神秘奧妙知識的國度，非常令人嚮往。

現今流通程度最廣，也是較為有名氣的有兩副塔羅牌，一是由萊特·偉德在1910年公開的版本，人稱「偉特塔羅」，也是本書所使用的塔羅牌；另一副是由艾力斯特·克勞立設計，稱為「托特塔羅」。

塔羅牌的圖樣裡充滿西方文化象徵，像是卡巴拉、鍊金術、聖經故事、埃及神話、希臘神話，還有數字學、占星學等，西方的古代文化資產，透過塔羅牌的流傳保留了一部分下來。

也因為塔羅含有深刻的文化與神秘學意涵，如果要將塔羅作為占卜工具，操作的占卜師就需要具備許多相關知識，以及相當豐富的經驗、對人有足夠的瞭解，才能應用自如。而在二十一世紀的現在，塔羅牌占卜已經成為大眾所熟知的占卜方式之一。塔羅牌驚人的準確度，真的可以說是現今仍遺留於世的古代「魔法」的一種！

我與塔羅的接觸約在我二十多歲，在國外讀書的時期。有一回逛書店，我替自己買了一副手繪版本的「大阿爾克那」22 張塔羅牌自學，從此以後就對塔羅著迷不已。當年台灣還不流行塔羅牌占卜，但塔羅牌在西方已經是一種日常的、固定行事的占卜方式。

塔羅是個非常容易入門的工具，卻也不容易成為「專家」，因為解讀牌卡的功力，需要靠占卜者的人生經驗累積與對牌卡的熟練度。在國外，人們相信年紀越大的占卜師就能算得越準確，在國內反而有年輕化的傾向。

我非常鼓勵讀者們都要學習這套集神秘學為基礎的學問，以偉特塔羅牌的創辦人偉特為例，他隸屬於一個英國神秘教派「金色黎明會」（The Golden Dawn），這個教派的教義是整合所有的神秘學為基礎，像是神秘的埃及、希伯來的傳承（喀巴拉）、和許多跟此秘密組織有關的象徵圖案，和西方聖經裡的故事、希臘的生命靈數、中古世紀的聖杯傳奇（Holy Grail）等，非常值得學習與研究。

最神奇的，是這副 78 張牌卡竟然可以跨越時空，占卜現代人的問題，仍然得到跟古代同樣的答案！可以說，塔羅牌本身的準確度就是一種「魔法」，令人驚嘆不已！

# 06 | 四大元素

施行魔法時，魔法師很常使用到四大元素的力量（請看「魔法為何有效」p.58）。四大元素除了是地球基本元素，在占星學的領域也很常使用。了解四大元素與我們的關係，可以讓我們對自己的優缺點更加理解，知道如何平衡滿足自己與他人的需求。

中古世紀有一位集自然魔法者為大成者：帕拉塞爾蘇斯（Paracelsus）認為，每一個元素後面都有一位精靈在守護著，祂們都有各自的運作方式，他還教導我們如何與這些精靈的元素特質共處，用何種態度可以讓此種元素更穩定，而且還可以學習如何「掌控」各種元素精靈特質。以下是四大元素的介紹：

# ▽ 土元素：

守護神為地精靈。

對應星座為：摩羯座、金牛座、處女座

用一顆開放的心可以穩定住土象能量。因為土象人天性保守拘謹，如果土象人能夠擁有開放的心，一定能讓生命更加活躍！

# ▽ 水元素：

守護靈為水神，這很像希臘神話的海神波賽頓。

對應星座為：雙魚座、巨蟹座、天蠍座。

此種水能量必須以堅定的態度，讓自己像「水的波動」的能量穩定。當水象人情緒波動時，不妨以堅定的態度自處。

# ▽ 火元素：

守護神為火精靈。

對應星座為：牡羊座、獅子座、射手座。

此種火能量必須以平靜的心面對世界，因為他們通常很激動興奮，尤其是面對當下發生的事情。

# ▽ 風元素：

守護神為空氣中的女精靈。

對應星座為：水瓶座、雙子座、天秤座。

此種風能量必須以定力將其想法落實下來。當他們「想得多、做得少」時，用果決的態度即可讓他們回到現實。

塔羅、占星、魔法都重視四大元素，在這裏，身為多年塔羅的學習與教學者，我要跟大家分享塔羅中四大元素各自代表的意義。

代表像土一樣固定的，象徵著能夠被觸摸與擁有的東西，像是錢與實際的事物。當一個人在牌陣中抽到過半數的土元素牌，表示這人有著很務實的人格特質。

像樹木一樣的權杖形狀的火，象徵著陽性的行動、冒險、樂觀、開始。當一個人在牌陣中抽到過半數的水元素牌，表示這個人有著很衝動但是樂觀的人格特質。

最適合「豐收」、「感恩」魔法。

許願時要小心意念，絕對要正面！

這是介於滿月到新月期間。

適合「釋放負面信念」魔法。

# 實施魔法的傳統節日介紹

一般來說，實施魔法平常只需要注意月相和星期的變化就可以找到魔法成功的頻率。但當你越來越熟悉魔法儀式，你會開始注意到適當的季節性時機。

所謂季節性時機，說明的是在地球上四季變化的能量轉變。你可以看農民曆，上面都有詳細記載。

| 節氣               | 說明                                                                                      |
|--------------------|------------------------------------------------------------------------------------------|
| 春分（每年三月二十一日左右） | 春分是大地經過冬天的沈睡、休養之後，再度甦醒過來。萬物欣欣向榮，這正是「新的開始」。你可以做一些跟「開始」有關的魔法儀式。可以用季節性鮮花、水果獻祭，向父神、母神表示你的感謝。 |
| 夏至（每年六月二十一日左右） | 夏至是一年中大地要轉變成炎熱的開始，傳統上，魔法師認為這天是「至陽之日」。夏季人體會藉由炎熱的熱氣，排出身體多餘的毒素，因此，很適合做「釋放」有關的魔法儀式。這個儀式就如同我在書中常提起的火能量釋放方法，用火燒掉你的負面思想或是煩惱，是個很適合的時機。 |
| 秋分（每年九月二十一日左右） | 秋分是大地豐收的時刻。傳統上，魔法師認為這天日、夜均等，但是太陽會逐漸減弱。在大自然中，四處可以看到成熟的果實，大地充滿豐富的農產。很適合在豐盛的大自然中做「感恩豐收」魔法。 |
| 冬至（每年十二月二十一左右） | 冬至是大地即將進入休息的狀態，在運作了三個季節後，大地和土壤也需要好好為來年做準備，傳統上，這天被魔法師們認為是「至陰之日」。這時候，可以點燃蠟燭，跟父神、母神連結，在燭火中感受到即將來臨的春日的溫暖。 |
