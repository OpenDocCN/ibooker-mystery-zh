# 前 言

翻开此书，我希望你能够对法术有真正的了解，认识一下我们传统文化中法术认知，无论时代变化如何，都不能忽视前人先贤为我们创造的灿烂文化。

法术，对普通人来说是如此奇妙的存在，似乎遥远不可及。然而事实上，法术一直在我们身边，小到我们的一举一动、大到空气与海洋，到处都充盈着构成矛盾和危险的事物。

这本书只会为你铺平一条通往法术深处的道路，它讲解的是基础，或许不会涉及高深的理论与知识，但脚踏实地才能跳的更高。

所谓法术之功效，是为身体在咒语仪式的诱导下进入某种异常而神奇的状态下，达成某种目的只是对身体暂时的或长期的改造。对人体的潜能进行开发。做用部位也只是自身，或者通过自身与外界进行某种沟通。总之法术没有大家想象中的神奇，它只是利用人体潜能的一种捷径。

慕言秋水老师通过几十年的修行总结了本书中的一些简单易行的法术，目的就是帮助人们化解现实的生活中的矛盾和危险，以求人们平安、幸福的生活。

# 一、个人、工厂、商店求财法术

用品：用五色石块、12根辐条、十二块硬币（壹元的），五谷（大豆、高粱、玉米、谷子、小米、小麦各一斤，共五斤）

具体做法：

在夜间11点钟，各钟物摆在宅神前（也就是灶王爷）上香，摆放供品（鸡、鱼、馒头五个、苹果五个、香蕉五个、桔子五个），供上白酒三杯，求灶王爷保佑发财，门口烧纸，烧香烧纸时心里怎么想就怎么说。一定要在烧纸时往地上敬撒白酒。

然后青色石块埋在东方、红色石块埋在南方、白色石块埋在西方、黑色石块埋在北方、黄色石块埋在中央外加十二块硬币。十二个车辐条还有五斤五色粮。

如果是楼房可用黄色布包上放在对应的四个方向和中央。

在上供时将斩煞除虚耗符贴在灶王爷跟前。如果有条件的可供个灶王爷牌位。每年的腊月二十三必须送走，正月初九请回来，如果不供灶王爷，符可直接贴在灶台旁即可。

## 另有一种求财法术

取住宅的生气方的土5斤（以大门为主查生气方），取东方的河水（即长流水也叫青龙水），放上黄酒一瓶用河水和成泥，做成5个泥人，泥人干后贴上5个符（地神助财通律令罡符），埋在院子中心和院子的四正位，用5个龙龟（铜质更吉），龙龟在前，泥人在后4个方位立着埋。其余四个龙头龟头朝着院子中心龙龟、中心的龙头龟朝向主人住的房间。如在屋里龙头龟头朝主人卧室。所用之符在上边。

# 四、借风水发财方法

例如卖服装的，可用花瓶一个，装满九枝牡丹，花瓶里放穿地龙，叫人抱着花瓶到其它生意好的地方去玩，多玩一会。和别人说话的同时不要让别人知道或看出来。转三圈或九圈，然后抱着花瓶回来，放在自己开的店里或收银台上即可。这样就把别人的风水借来了。等别人店里的顾客多时再去，隔几天走一趟，一段时间后自己店的生意就好了。

# 六、长远求财符

两道符共八道，用黄纸墨书。

多准备寿金、金元宝、银元宝、香。晚上在十字路口摆上供桌，摆上鸡、鱼、猪头、水果、酒。事先用金色纸糊两条裤子（以前穿的裤子也可以）或金柜，装上金元宝、银元宝同符子在十字路口一起烧掉。时间是晚上十一点。

咒语：

念七遍咒语，烧七道符一道画符，摆供品，上香，敬三杯酒。发财后，每个月2日、16日到十字路口烧纸敬酒。答谢神灵。
