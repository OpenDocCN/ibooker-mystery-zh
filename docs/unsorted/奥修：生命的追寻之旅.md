# 序

人生来就带著一种未知、不可知的潜能。当他来到这个世界上的时候，他最初（original）的臉孔还未出现。他需要寻找它。那会是一种发掘，而这也是它所具有的美。同时，这也是一个存在体和一个东西、物體之间的差别。一个东西是没有潜能的，它是它所的樣子。一张桌子就是一张桌子，一张椅子是一张椅子。椅子不会变成其他任何东西，它没有潜能；它只有一个事實。它不是什么種子。

人不是 一个东西。这一點带来了所有的麻煩和所有的喜悅，所有的挑戰以及所有的困擾。孩子来到这个世界的時候是空白的，他身上没有書写著任何东西，甚至也没有什么事情能夠顯示出他將會是什麼樣子，他所有的向度都是敞开的。这是你需要了解的一个基本事實：孩子不是 一个东西，他是一个存在體。他尚未完成，他還在過程裡。他是一个過程，也因此你無法預測他最終会 是什麼樣子；你無法預測他所有生命經歷最終的結果是什麼，是苦惱、焦慮還是狂喜；你不知道他最終的結果是什麼，孩子来到这个世界的時候並沒有帶著任何地圖。所以，所有的星象都愚弄了你，那些看手相的人也愚弄了你，而他們之所以能夠愚弄你是因為你給了他們機會。每一個父母都關心孩子會成為了什麼樣子。他們出於愛而關心，所以父母會被各式各樣的騙子所欺騙。那些騙子會說：「他將來會是這樣，或是那樣。」這些騙子不會造成什麼傷害；他們只會占一點小便宜而已。他們的預測從來不會成真。孩子真正的麻煩來自於傳教士、政客以及中小學老師。政客感興趣的不是孩子真正的潛能。他感興趣的是孩子是否會成為他力量門爭裡的一份子。他在孩子身上投資了一些東西，因為每個孩子都是他潛在的 朋友或是敵人。所以招攬的工作進行的越早越好。也因此，在孩子能夠自立之前，已經有人讓他走上了岔路，以便滿足那些政客的欲望，但是，這卻會扼殺孩子所具有的種子。傳教士也對孩子有興趣，他也在孩子身上有所投資。如果這個世界上有較多的天主教徒，那麼教宗會是最大的一个教宗。如果天主教士消失了，教宗還有什么意義呢？誰會在意他呢？所以每一個孩子生來都擁有政客和傳教士想要利用的力量。

問題 我曾經聽你說過生命本身是如此地令人滿足，如此地豐富，如此地喜樂，那到底是什麼讓人感到痛苦（miserable）呢？

生命是豐富的，生命是喜樂的，但是人類已經失去了與生命的聯繫。人類已經變得太過於自我意識（self-conscious）。這份自我意識變成了一種障礙，以至於他雖然還活著，卻不是真正地活著。自我意識是一種疾病。

鳥兒是快樂的，樹木是快樂的，雲朵與河流也是快樂的，但是它們沒有自我意識。它們就是純粹地快樂。它們不知道自己是快樂的。

佛陀是快樂的，克里希納是快樂的，耶穌是快樂的，但是他們是純粹的意識。他們快樂，但是他們不知道自己是。

不知道自己在說些什麼。人類之所以痛苦是因為人類必須是痛苦的。痛苦不是你的錯，那不是你有什麼問題。身為人類就是會感到痛苦，因為身為人類就是懸在中間——不在這裡也不在那裡，而是懸在半空中。出於這種緊張的狀態，煩惱出現了。人類已經失去了一個家園——那個鳥兒仍然在歌唱，動物仍然四處遊移，樹木仍然純放的地方——伊甸園。人類已經失去了那個家園。亞當已經被驅逐出門；亞當已經變成了一個人類。當亞當還在伊甸園的時候，他是一個動物，他不是一個人類。當神把他趕出伊甸園的時候，那個驅逐成就了人類。人類從一個家園裡被驅逐，好讓他能夠尋找另外一個家園——一個更大、更高、更深、更偉大的家園。他已經失去了一個家，所以他對過去留戀不已；他想要成為動物。要遺忘伊甸園是一件很困難的事；因為它是那麼的美。所以在某些時候裡，我們會變得像動物一樣，在深沉的憤怒裡，在暴力中，在戰爭裡，我們會變得像動物一樣。那是憤怒所帶來的一種享受。

當你生氣時，你為什麼感覺這麼快樂？當你摧毀某些東西時，為什麼你會充滿了能量？為什麼在戰爭時期，人們看起來比平時更有朝氣、更健康、更敏銳、更聰明？就好像生命變得不再無聊一樣。到底發生了什麼事情呢？那是因为人類倒退了。即使只有幾天、幾個月的時間，但是人類再度變成像動物一樣。這時候，他再也不知道什麼法律，什麼人性或是神的事情。他就是放掉了自我意識，他變得無意識，然後他開始殺戮、謀殺、強暴。在戰爭裡，所有一切都是被允許的。那就是為什麼人類一直需要戰爭。每隔十年就需要有一場大戰發生，而小型的戰爭更是一直持續不斷地發生。否則，人類有困難持續生活著。 人們變成酒鬼，人們毒品上癮。透過毒品，人們試著找回那失去的家園、失去的天堂。在LSD這種毒品的影響下，你重回伊甸園，只是你從後門進入。LSD是伊甸園的後門。這時候，生命再一次似乎變得如夢似幻，充滿色澤；樹木再一次看起來充滿了光輝，就像是亞當和夏娃曾經見過的一樣，就像現在那些鳥兒、老虎和猴子所見到的一樣。樹木的鮮綠裡有著一種光亮。所有一切看起來是如此地美好。你不再是一個人類，你已經倒退了。你已經強迫自己的存在倒退；因此，毒品和酒精飲料對人們有著巨大的吸引力。在《吠陀經》裡，他們把這稱為蘇麻液（soma），現在的人把它稱為L S D，但是它們其實沒有什麼不同。過去有些時候，那是大麻（ganja）、印度大麻（bhang），現在被叫做大麻煙（marijuana）或其他名稱，但是大麻（ganja）、印度大麻就化學而言，倒退是可能的，但是你沒有辦法真的倒退。倒退是不可能的事，時間不會允許這種事情發生。一個人只能前進。

你沒有辦法在時光裡倒退。倒退檔是不存在的。當福特製造出他的第一輛車子時，它是沒有倒退檔的。一直到稍後，他們發現回家變得很困難。人們需要繞上很大一團，繞很多不必要的路途；才能夠回頭。所以倒退檔是後來才加上去的。但是在時光裡，沒有倒退檔可言；你沒有辦法倒退。

人類一直夢想著這種事情，幻想著這種事情。在某些科幻小說裡，人們夠回到過去的時光裡。喬治·威爾斯（H. G. Wells）想像出時光機的概念，在那裡，你坐在機器裡，使用倒退檔，然後你可以開始倒退。你原本是個年輕人，然後你會變成孩子，然後你會變成一個嬰兒，然後你會在子宮裡。你會開始往回倒退。但是時光機並不存在。它只存在於詩人的頭腦裡、幻想裡。

倒退是不可能的。你只有一種可能性，那就是前進。

人類會一直感到煩惱和痛苦。可行的方法只有兩種，一是讓倒退變得可能，再不然就是超越人類。人類是一座橋樑。没有辦法在這座橋樑上蓋房子。你需要通過這座橋。它不是讓你在上面生活的。

回教的蒙兀爾皇帝阿克巴（Akbar）建造了一座特別的城市叫做法特普希克里（Fatchpur Seekeri），當時他要求智囊團找到一句格言，以便放在銜接著這個城市的橋樑上。他們四處尋找，然後他們找到了耶穌的一句話。《聖經》上沒有這句話；所以它必然來自於其他源頭，來自於蘇非。因為阿克巴的宮廷裡有許多蘇非行者。那句格言是：一個世界就像是一座橋樑，不要把你的房子蓋在上面。這句格言現在仍然還在那座橋樑上；它非常的美。因為事情就是如此。

在印度有一個主要城市博帕爾，我很驚訝地發現那裡的人甚至會在客廳裡放一個小小的牌子，上面寫著：一請勿吐痰一。那個城市是印度唯一一個人們會隨地吐痰的地方。甚至連在你的客廳裡，那都很常見。那些標示沒有辦法制止人們。當我看到那個標示的時候，我告訴當時和我在一起的醫生說：一我從來沒有在印度其他任何地方看過這種標示。而在這裡，在你的城市裡，在每個房子裡都有……這是怎麼開始的？一他說：一我不知道。一

我說：「一定有人先開始放出標示牌；然後才有這種隨地吐痰的現象。」

### 葛吉夫的觀點

在亞當的花園裡，有這麼多的樹木……而神卻特別指著其中某一棵樹說：「這是智憲之樹，永遠都不要吃那上面的果實。」根本不需要什麼蛇來說服夏娃，神自己就已經說服她了；神就是那兩蛇。然後，神給予的處罰更是令人難以置信。甚至到現在，人類還因為亞當和夏娃當時的違背行為而受苦著！

光只是透過禁止，宗教就在人們的頭腦裡創造出了各式各樣的罪行。他們也創造出了各種概念，阻礙人們進行任何追尋。他們說：「就是相信。」但是相信是廉價的，因爲你不需要做任何事情。結果是葛吉夫必須嚴厲到這種程度，嚴厲到他甚至開始說：「你並不擁有一個靈魂。那是一個錯誤的概念，是宗教在你內在植入了一「每個人都有一個靈魂」的概念。」

他說出這種話語，就只是爲了喚起你的追尋，去尋找那到底是事實；否則每個人都非常完美地昏睡著。你爲什麼需要追尋呢？你已經擁有它了；神就在你的內在！所

### 精神追尋

在印度有一個主要城市博帕爾，我很驚訝地發現那裡的人甚至會在客廳裡放一個小小的牌子，上面寫著入一點。除非你發現了自己存在本性的寶藏，否則你没有發現任何事情。

這個伐木者說：～現在，我可以了解為什你總是坐在这裡，坐在这棵樹下，從來不曾離開，也不曾在意過那些鑽石。現在我會坐在你身旁，坐在这棵樹下，直到我找到你所說的這個寶藏。這很難，其他事情都很容易，只要往外再多走一點，再多走一點就可以。可是現在，你改變了整個方向。

這個伐木者寧靜地坐在森林裡，坐在这個神秘家的身旁，他吸收著神秘家所具有的存在感、他的寧靜、那份充滿愛意的存在，然後一整天過去了。當太陽落下時，這個伐木者非常地訝異，他張開眼睛說：～你應該早點告訴我的，我之前每天都經過這裡。你不是什麼慈悲的人。我花了好幾年的時間砍柴，肩膀上扛著重重的木柴，而你就只是坐在这裡享受這份內在的感受、這份喜悅，你從來沒有告訴過我。

這個神秘家說：～當時的你不會傾聽我所說的話。你首先會需要鑽石。現在你可以回家了，因為你的寶藏就在你的內在。只要記得一件事情：持續再持續，不要停止。

沒有什麼會全然的停止，因為全然的停止就是死亡。

我們或許可以創造出某種更好的世界，比這個世界上所有烏托邦夢想者所想像的還要美好，但是烏托邦永遠都在地平線的那一端，它看起來如此地近，好像你觸手可及，線的時候，它不斷地後退，你和地平線之間的距離是永遠不變的。而這就是整個成長和演進的秘密。

奇蹟是可能的，但是沒有任何奇蹟會是完美的。完美是不可能的，也幸好完美是不可能的。這讓你持續不斷，這讓你富有生命力，讓你不斷地流動。

可能的。這讓你持續不斷，這讓你富有生命力，讓你不斷地流動。我完全支持烏托邦，而我也知道這個烏托邦的概念會改變。但是不論我們達成了多少的成就，那都不會是烏托邦，那些尚未達成的部分才是烏托邦。除此之外，再也沒有其他的方式了。人類的演進是多重向度的，人類的意識可以無限地成長，而它也確實該要無限地成長下去。

該要無限地成長下去。


## 奧修給我們的忠告

+   當你快樂時，你是平凡的
+   苦悶只會發生在人類身上
+   學著享受生命中的快樂與痛苦
+   當不執著於頭腦時，才能更有效率地使用它
+   開始享受你已經是的，平凡的生命將變得不凡

我覺得生活非常的無聊。我該做些什麼？
什麼是重要的？在平凡的生命裡真的能夠找到幸福快樂嗎？

本書是奧修提供給我們在生命旅程中，追求內在喜樂所必備的重要心靈指南，
他談到我們在生命中所遇到的關鍵困境及本質問題，並為我們一一解惑。

奧修說，當你自己，放掉所有的「應該」，享受你已經是的。享受生命是唯一的正確方式。生命是由許多小事所組成的，你需要透過覺知、觀照和警覺，把每一件小事蛻變成美好的行動。然後，那些平凡的事情將會變得不凡，你平凡的生命也將變得不凡。

人類與動物的最大差別，在於人類擁有提升意識、達到神聖的潛質。這是人類擁有的優勢，但它也造成人類充滿不安、衝突，並經常徘徊在自私與慷慨、愛與恨、脆弱與勇敢、希望與絕望之間。《生命的追尋之旅》幫助我們深入看待這些問題，並引導我們，如何擁抱並接受生命中的種種，讓你的生命變成一趟充滿冒險的美妙旅程，這就是我們存在的最重要目的。
