# 創造世界的人，正是你自己

為什麼我可以把「肉眼看不到的無形能量」拍成人人都看得到的照片呢？原因在於，對於無形事物的捕捉，我具有紮實的能力根基。我出生於神社家庭，因此我始終相信「八百萬神」這句話的涵義——萬事萬物皆有靈。

自古以來，日本人奉太陽為神祇，頂禮膜拜。那是因為日本人真切地感受到，是陽光孕育了植物，我們才得以生活在這個世界上。此外，產出各種食物的山野、培育種子的大地，對於賜與我們種種恩惠的大自然全被我們當成神祇尊崇。

存在於大自然的萬事萬物均為神靈的觀念，對象包括了植物、動物、礦物、自然現象……甚至「人類」本身也是神祇。

因為人類本身也有「神靈」棲宿，所以必須以尊敬的態度來對待自己及他人，這是我自小被灌輸的教育。

同時，也因為神靈棲宿於我們每一個人的身上，讓我們建立了自己所屬的世界，並透過觀照自我的意念，而創造了世界。

由於擁有這種認知，所以我能感受到肉眼看不到的事物，而且我一直想將這無形的事物，拍成人人都看得到的照片。

# 幼年時期的聯覺

請容許我稍為詳細地介紹自己幼年時期的故事。

我自幼生長在名為「金澤黑沼神社」（p.8照片）的神社裡。金澤黑沼神社位在福島縣內空氣清新、有神靈鎮守的森林之中。

這座神社尊奉神祇至今已經有1300年的歷史，並被認定為重要民俗文化遺產慶典的「羽山籠」，現今仍然流傳，今年已經迎向1088屆。

在這神社長大的我，對於幼時的記憶，迄今仍然記憶猶新。

記得0歲的某一天，我睡在社務所隔壁的祖父母家，當時社務所正進行改建工程。參與改建施工的人們，包括木工師傅、建材行的師傅、玻璃店的師傅，彼此交談的聲音及光景，我到現在仍然歷歷在目。

連剛玩耍回來的姊姊，跑到我睡覺的地方來，還捏捏、摸摸我的臉頰，我也記得一清二楚。

那時，從姊姊說話的口中，我看到了朦朧的綠色物體。我一邊看著朦朧的綠色物體，一邊感受姊姊傳達給我的意念「我回來了」的親切能量，然後我用「謝謝」的意念能量，回應給她。

![](img/6eb238093f0e7a97182acdb03f84947c_9_0.png)

就像這樣，雖然我和姊姊不是用言語對話，但是內心似乎湧現了「我知道了」、「好喔」的感覺。

幼年時期的這種感覺，後來我才知道那是「聯覺 (synesthesia)」。

具有聯覺的人對事物會有特殊的感官知覺。例如：在文字或聲音裡看到顏色、在圖形裡聞到味道。對於某種刺激，有聯覺的人不只有一般人的感覺，還會有它種不同的感受，普遍認為這是少數人才具備的特殊知覺現象。

我以前最擅長的聯覺是「用心靈之眼觀看事物」。

這不是睡夢中的狀態，而是張大眼睛，腦袋像是接收到什麼訊息的感覺，像做白日夢一般，從當下眼前的景象，立即切換到不同的頻道，有別的景象切入視頻的感覺。

在幼年時期的「聯覺」體驗中，我清楚記得一件事。在我出生之後到八個月大的期間，祖母經常背著我去散步。

那時候我有一種被樹木氣味、溫暖意念包圍的感覺……因為神社位在有神靈鎮守的森林裡，所以我感覺到氣的流動、獨特的意念，並在半夢半醒之間親見這一切。當時的感覺像是聽到大自然的旋律，又像是樹木之間的彼此對話。

我更常感覺到的是——和八百萬神靈的意念共處的感覺，那就像是在一個大家族中，大家熱熱鬧鬧談天說地的感覺。

因為我的身軀小，所以曾經覺得待在那裡非常擁擠。但是即使有擁擠的感覺，我卻也真切地感受到——「我來到了如此美麗的大自然，可以感覺到種種不同事物的神奇世界」。

當時的我只要想著「我好想到樹梢上」，我的意念就可以去到那裡，並且看到了樹梢上的景象。「讓意念前往自己想去的地方，那個空間隨即出現。」當時的感覺就像這樣。

# 拍到飛碟後，開始能捕捉能量照

長大之後，我常夢到關於精神層面的夢，同時也有過種種不可思議的體驗。尤其是2011年3月11日的前幾天，我夢到了神靈。（編按：2011年3月11日在日本東部發生了芮氏規模九級的大地震，一般稱為日本311大地震。）

在夢裡，先是出現了有如神仙一般的人，他對我說了一句「待在這裡」，然後就消失不見；後來又出現充滿修行僧氣息的人，他留下一句「千萬小心」，然後一樣消失得無影無蹤……。感覺像是神靈的各方人士，在夢境裡，頻頻傳遞訊息給我。

在那之後，我開始拍攝到「眼睛看不到的事物」。

後來因為出差的關係，我到了名古屋某家飯店的高樓層裡工作。當時映入眼簾的，是窗邊的一般天空景色。但是我感覺到，當時的天空似乎有很強烈、強烈到吸引我注意的能量。

我心裡想著：「到底是什麼呢？」然後朝著天空拍了照片，只拍一張而已。當時的照片裡，我拍到了半透明的飛碟（P.12照片）。

![](img/6eb238093f0e7a97182acdb03f84947c_13_0.png)

拍到飛碟的照片後，某一天，我前往高尾山的「火渡祭」慶典。儀式中，我用雙眼清楚地看見，唱誦著曼陀羅的修行者上方，出現了非常巨大的曼陀羅形影。從那時候開始，我就更能拍到能量及靈球體之類的照片。

到目前為止，我將許許多多各式各樣的能量意念收進照片裡。即使以常識而言，它們屬於無法捕捉到的事物，但我的照片中卻能把它顯像成看得到的形體。

請覺察這些照片裡的能量，並且閉上眼睛，用心靈之眼感受看看。

如果能以本書為契機，讓你和無形的世界有了確切的連結，你將可以突破既往的觀念及制約的框架。未來，將得以感受到人生無限的可能、以及意念的恣意開展。

衷心期盼，本書能幫助你享受無極限的自由人生。

## 01

在大自然中，為你召喚好運的
「精靈的魔法時刻」

不論是存在於樹木、花朵、石頭裡等等大自然的精靈及妖精，都會發出讓我們情緒開心、豁然開朗的能量。

接下來介紹的照片，都是精靈與妖精化為靈球體的照片。

靈球體是能量之光膨脹而投影在照片上的一種現象。

而靈球體有兩種，一種代表「能量」。像是人類的情緒起伏時，或是在能量流動時，瞬間以閃光顯現在當事人周圍的一種能量性靈球體。

另一種則是代表精靈、妖精意念存在的靈球體。

當精靈及妖精棲宿在樹木、花朵、石頭等等大自然物體的時候，或是被大自然吸引而來的時候，也會化為靈球體而顯現在照片上。

帶有精靈、妖精意念的靈球體，顏色有彩虹色、綠色、白色……各色顏色都有，它的特徵是格外耀眼奪目。而靈球體和單純的光暈（halation：物體照到強光時，周圍變亮而模糊的現象）完全不同。

在此介紹一下我遇到精靈的小故事。

那是 2011 年 2 月的某個晚上，當時我睡得很熟，但是突然感到被一陣非常冰冷的風吹過臉龐，我心想：「這是什麼？」當下醒了過來。

然後，我看到了有如小指般大小的閃亮亮圓形虹光。它隨著細雪，以漩渦狀的形式飛到我的臉上。仔細一看，在圓圓的虹光裡，有一個白髮蒼蒼、長著白白長鬚，看起來像是神仙的老人。

那位老人是存在於光之中的雪精靈。

雪精靈站在我的枕頭上，朝著我的臉，「呼～呼～」地從嘴裡吹出雪和風！因為非常冷，所以我想移開身體，但是我動不了，就這樣被精靈吹了一會兒。大概經過2、3分鐘的時間吧。

「好啦，我知道了～」我在心中笑了笑，向他傳達了感謝的心意。

於是精靈的身影隨即消失，我的身體也可以動了。後來我起身，開了窗戶看見那一年的初雪紛飛，地上已經有了些許積雪。

「喔～雪精靈是來告訴我初雪的消息啊，謝謝了！」

我再一次表達了感謝之意。

像這樣，精靈和妖精有時會以如此幽默的方式出現及傳達訊息給我。請你也從照片中感受看看精靈、妖精的能量。

## 來自未來世自己的訊息

前面我介紹了各種能量照，各位覺得如何呢？
到目前為止，我拍了各式各樣的照片。如同〈前言〉裡的介紹，自從我拍到了飛碟照片，接下來就能將我感受到無形世界的一切，如實地拍下來。後來我開始欣賞、感受各種「看不到的無形事物」。

他告訴我——
「你抓住了未來世界的訊息。」
「未來世界的你，觀察著現在世界的你所經歷的過程。」

和巴夏實際進行意念溝通的達里爾·安卡，他和他的下一世，也就是巴夏，彼此有約定，對這一世進行意念靈訊的溝通。
