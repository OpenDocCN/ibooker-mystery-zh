# 作者序

瑞·强德兰(Rae Chandran)教导手印工作坊已有多年的经验，因此他将这些传讯得来的手印付梓成册，是一件自然而然的事。其实在这本书发行之前，瑞所接收到的手印，已超过本书所介绍的手印两倍之多了。

我曾随同瑞前往东地中海区域的三个国家旅游，书中的手印有一部分是瑞在尼罗河岸饭店的阳台上接讯得到，另一些是他在前往希腊时，坐在高速公路的巴士上所得到的传讯手印。至于本书最后一部分，则是瑞在美国麻萨诸塞州大巴陵顿( Great Barrington, Massachusctts)的公寓中所接收到的手印。在我们下一本著作《与天使为盟》(Partnering with Angels)中，我们将会结合呼吸技巧和声音来呈现更多的手印。

在这本书中，有一些手印会搭配音频来使用，另一些则不会。此外有一些是静态的手印，另一些则又包含了动作。在这些手印当中，有一部分是为了疗愈地球，其他的则是为了孩童或个人扬升而得到的手印。我本身曾在一些工作坊中教导这些教材，我发现教导这些手印的一个有效方法，就是先不要把这些手印的名称告知所有参与学习的人。藉由这个方式，所有人都必须透过自己的身体，而不是他们的头脑，去真实地感受这些手印的能量。

# 前言

这本《手之舞》是在群策群力下所完成的著作。书中所有的内容是从许多内在线师如大天使麦达昶 (Archangel Metatron)、大天使麦可 (Archangel Michael)、赛拉比斯·贝大师 (Master Serapis Bey)、图特大师 (Master Thoth)、盖娅女神 (Gaia)、克丽奥佩脱拉皇后 (Cleopatra) 以及阿肯那顿法老 (King Akhenaton) 传讯而来。

从其他星系及银河系来到地球的远古生命，早就知晓并教导以身体创造能量的方法。这本书中的所有教材源自于古老的马雅文明、法老时期的埃及人们，以及印度河流域文明所孕育的人们。这些古老文明深谙使用身体来创造特殊能量的方法，同时他们也使用这些方法，来疗愈身体方面的病痛以及情绪体和精神体方面的问题。

“量子疗愈”是疗愈身体某个部位的基本原则。这意谓着唯有当身体的某一个部位被疗愈时，整体的疗愈才有可能发生。而当我们以手指呈现出某些姿势时，正可以产生能量。藉由身体内在的智慧，我们可以将这股疗愈的能量，传送到身体正在遭受病痛的地方，进而达到疗效。

使用手指呈现出某些姿势来创造疗愈的能量，其实非常简单。大拇指代表“我是临在”的能量或智慧，而食指则反映着个人性格及心智状态。所以当你用大拇指压在食指上的时候，大拇指所传递的较高频率就会被送进你的食指以及心智之中。这些疗愈的能量会在掌心循环后，进而启动位于掌心的“疗愈脉轮”。之后，这股疗愈的能量会流向身体正在遭受病痛的地方，进而达到疗效。此外，当你运用这些手印并配合呼吸及声音的时候，你又可以再度强化这些手印的能量，并将原本的手印提升成为具有更大转换能量和疗愈的治疗工具。

《手之舞》这本书并不是专为灵修人士而书写，它是一本适合普罗大众的工具书，它也是一本跨越宗教的书，这本书的所有内容不仅适用于没有尝试过任何灵修课程的人，也适合任何一位想要致力提升自己生活或生命的人。因此，无论你是位治疗师、老师、大师、灵性极高的灵修人士或是初学者及孩童，都可以受用其中。正如瑜伽一样，瑜伽也与任何宗教无关，瑜伽也并不是专为那些灵修人士所创立，而是为了任何一位想要获得健康身体的人而存在。

这些手印的目的其实很简单，他们可以平衡一个人的身心灵。正如其他的灵性练习一样，打手印将有助于平衡你生活中的每一个面向，并且为你的生活注入喜乐和平安。当然打手印还有其他的优点，但是他们的最终目标就是将你引领至一个喜乐平安的最佳状态。因为当你感觉到喜乐和平安的时候，你就能更容易地触及自己意识的更高层次。

这本书的内容源自许多存有，其中包括大天使麦达昶 (Archangel Metatron)、大天使麦可 (Archangel Michael)、图特大师 (Master Thoth)、阿肯那顿法老 (King Akhenaton)、爱西斯女神 (Isis) 以及瑞·强德兰的高我。你或许会问：为什么这些存有要提供这样的讯息呢？答案其实非常简单，因为这是帮助人类觉醒的另一个工具。

## 你个人的手印练习

关于手印的练习场地并没有任何严格的限制。基本上你可以在任何自己想要施作的地方练习这些手印。因此，无论是在室内或室外的任何地方，都可以练习这些手势。练习手印最重要的不是地点的选择，而是你应该努力维持放松且舒适的姿势，然后在一个安静的环境中练习。至于准备工作的部分，场地仍然不是重点，而是你要让自己与感官系统结合，然后预备自己进入全然放松的状态，以达到最佳的效果。

这些手印最好是作为个人练习之用，但在课堂中被教授时，也可以团体的方式进行。练习一个手印的时间虽然没有限制，但仍建议以三分钟为基准。

至于你该练习多少次的这个问题，则视个人而定。如果你觉得受益，那请持续做下去，并请专注于自己的身体、身体内能量流动的方式、以及个人的感觉来决定练习次数。

在这些手印之中，有一部分的手印必须配合声音施作，而另一部分的手印则没有这样的限制。因为当我们使用声音去搭配某个手印时，声音会强化这个手印的力量，而那些无须搭配声音的手印，本身即已具有高能量，所以并不需要其他的聲音來強化它。言下之意，某些手印需要音频来强化它的效能，但其他已具有高頻能量的手印則無此需求。

当你在施作这些手印的时候并不需要静止不动。孩子们可以在跳舞或玩耍的时候练习这些手印，因为当孩子们在跳舞或玩耍的时候，他们是全然存在于当下的。但成人们却极少可以维持在这种全然存在当下的状态。即使他们在跳舞或玩乐，他们的心思仍然常常飘往他处而无法专心。因此唯有当成年人能保持静止不动时，他们才能比较放松，而且更加掌握自己的意识形态。

有些手印因起源地不同，可能需要施作五个动作才算完成，但在这本书中，我们只选择最适合地球这个星球的手印来作介绍。

另外还有12个DNA手印是用来搭配12股DNA的。这些手印必须被视为一整套而且一起施作。正因为这些手印的用途是为了整体DNA而来，所以请不要单独施作其中任何一个手印。毕竟这一套的12个手印，就是为了启动DNA而产生的。

至于其他的手印则没有这个限制。此外，所有出现在这本书里的手印也没有特别的排序。你可以选择从书中的任何一个地方开始练习，或是挑选任何一个与你共振的手印开始练习。当你施作手印时，请尽量保持身心灵的专注，并全心投入练习之中。

练习这些手印可以为你带来宁静、幸福和更高意识层次的生命。手印的效果虽然因人而异，但假以时日，你会注意到自己正过着平衡且能量提升的生活。此外，你的人际关系也可以因此修复或更加和谐。同时，你也会成为一位更具有弹性及包容力、不再事事批判别人的人。你会感觉自己是完整之中的一部分，而且也是所有地球上弟兄姊妹们的守护者。因为很明显地当你在为疗愈地球而施作手印时，你也会同时带来全球性的疗愈。

这些手印源自远古时代。长久以来，它们被世世代代的人们用来作为疗愈的工具，以创造幸福、安宁和平安。当人们想要精熟使用任何一项疗愈工具时，最好的做法就是去使用它。在你练习的过程中，你会看见自己周遭的能量和气场、以及自己的气和心智感受力的诸多变化。你或许能够了解自己必须要持续地练习，才能够精熟任何一项能力，而这也是我们要给你的最简单的建言：练习、练习、再练习。
