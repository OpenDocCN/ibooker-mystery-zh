

# 茶叶与咖啡占卜

# 关于茶占术的事
# 愿你的茶充满信息！
你熟悉的茶可以是非常强大，近乎“魔法般”的工具，能“溢出”关于当前情况和未来可能的信息，回复你的疑问。

# 你的潜意识
这怎么可能？当然是因为你所需要的一切都在你的潜意识里，而茶占术就像所有其它类型的占卜，可以用于汲取你所需的信息。但是，潜意识是以象征储存信息的，而不是文字——就像你的电脑是以二进制的数字来储存信息的一样——从而，我们必须学会解码，在占卜中将这些象征关联起来，将其转变成一个故事。

# 填补空白
你可以在以特殊方法准备的茶水中找到可识别的形象和象征，但是，有的时候，你不得不用想象来填补空白，以得到具有含义的内容。通常，但不总是，你的感知会告诉你是否做的对。

# 提问
你的问题应当是简单，重要，又明确的问题，问题要有一定的广度，不能只是简单的是与否的问题。例如，“我是否能找到梦想中的男人？” 这种问题最好加上更有用的信息。更好的问题可以是“我应该找怎样的未来另一伴？” 类似的可以是“被我吸引的男人之中，我应该找哪种类型的？” 这样问才能得出有用的信息。因为你未来的男人可能不是你现在梦想中的，而是非常稳定的“邻家男孩”类的，而不是更有情调的，被很多女人环绕的帅哥。用词不要过度明确，那也会过度限制答案的范围。

现在，在纸上写下你的问题，将纸折好，用于占卜。

# 下一步更重要
你接下来必须要为你之前问题创造出你自己的解读。换句话来说，你要创造出一个故事，它讲回答你的问题，无论这个问题是关于你现在的生活（关系，职业，健康，等），你未来的可能，还是你的过去的。

# 不要轻视解读中这种说故事的一面
我们经常用说故事的方式来给自己，给孩子，给朋友诠释生活，甚至是在发生意外的时候也要给警察描述事件，无论你是那些事件的当事人还是观察者。我们给孩子讲故事是为了帮助他们理解情况，而不让他们担心，我给朋友讲故事是告诉他们在做什么，我们是谁，我们的遭遇。故事是我们在生活中找到意义的方式。

# 你需要的工具

茶占术使用方式简单，仅需要投资少量器具。事实上，购买昂贵或异域风情的物件反而会分心你自身的内在能力。记住，力量源自于你，而不是茶杯，茶壶，或茶叶。就像格言，你要“简单就好，甜心。”

你倒是要弄一个传统的白瓷杯子和茶托。你不能用更流行的直线型茶杯，而是要找那种上半部分更大，底部更小的杯子。你也需要用于烧水的茶壶，一个用于倒水的碗，以及每次占卜使用的餐巾或纸巾。

你还需要茶叶，不是那种磨成粉的或袋装的。你可以撕掉茶叶袋用里面的茶叶，但分散的茶叶更好。

舀半勺茶叶到杯子里，倒上热水，将茶托盖在杯子上，等水温到可饮用的程度。不要加糖或牛奶！

# 怎么做

# 加持杯子

在慢慢喝掉四分之三茶水的时候，想着你的问题。喝完后杯子里要留有足够的茶水（茶水要淹没茶叶）。将你的惯用手放在茶杯上一分钟，以加持茶杯和茶叶，让你的灵光随着你的提问影响它们。如果你是为他人占卜的，那么让提问人在思考问题的同时，手放在上面进行加持。如果提问人不在现场，那么用属于他的物件或者照片，或者形成代表那个人的思念体。

将茶托放到一边，在上面放上纸巾或餐巾。你等会儿会用它吸收溢出的水。另外在旁边准备好小碗。

# 分散茶叶

下一步很重要。在倒出剩下的茶水时，你必须要分散在杯内的茶叶。手举着杯子，用你的手腕慢慢地摇晃，倾斜，倒出剩下的水。手拿着杯耳，用你的手腕慢慢地摇晃，让茶水与茶叶在整个杯子的内壁温和搅动。有些占卜师坚持认为你必须要用左手拿杯子，那是你在这一过程中通灵的手。

随着你摇晃，倾斜杯子，让少量水能够从顶部流出，流到小碗中。持续这个过程，直到水没了，茶叶被分散在茶杯内壁上。不要担心在摇晃过程中有少量茶叶随茶水一起流出。需要练习，你才能熟练动作——如果摇晃速度太慢，那么茶叶会挤在一起，或者形成几堆；如果你摇晃速度太快，那么茶叶不会黏在茶杯的内壁上。

# 旋转茶杯

然后将杯子倒放在你之前准备好的放着纸巾或餐巾的茶托上。这会吸收掉杯内剩下的水。

将杯子，仍旧倒放在茶托上，放在提问者面前。

让提问者以顺时针方向旋转茶杯三次。这有两个目的：它会将剩下的水吸收掉，也让提问者的灵光有机会浸入茶叶中。（注意：有些占卜师坚信旋转应当逆时针进行，但我个人偏好顺时针。）

拿起茶杯，放正。现在可以解读了。

# 问问题

# 写下问题

无论是何种占卜，你最好在纸上写下你的问题，而问题自身要足够宽，不是简单的是与否。例如，“我能否找到更好的工作？” 这种问题提供的信息太少了。更好的问题是“我未来的职业生涯有何种可能？” 这种构成问题的方式能让答案给出有用的信息，而不是被问题限制，甚至不相关的。你未来的职业可能是要为自己工作，或者为多个顾客服务，成为自由职业者，或者是与某人进行某种合作，等等。不要让过度明确的用词限制可能。

你，或者提问者，如果对方在现场的话，应当用一只手拿着纸，用另一只惯用手加持杯子。要么沉思疑问，要么大声说出它。

# 为不在现场的人进行占卜

如果问题相关的人不在现场怎么办？这种情况不怎么满足条件，因为无法直接接触到当事人的灵光。如果你必须要这么做，那么就像为你自己占卜那样准备好杯子，再用你的惯用手放在杯子上，聚焦在不在现场的人。一个不错的准备方式是，写下当事人的名字，地址，性别，年龄，等任何你知道的，与情况相关的情况。用你的另一只手拿着纸，大声陈述你的意图是让那个人的能量能够浸入茶叶中，以及你想要知道答案的问题。

# 你能知道什么

# 每次占卜只能一个问题

任何占卜主要有三种功能：占卜当下，未来和过去。在占卜当下中，你能够得到当下情况的明确问题，但每次只能进行一次提问，为未来进行占卜时，你进行的更多是预言，多数是基于你当前情况和道路的未来可能，但占卜仅对更近的未来有意义，因为它会包含影响你当下情况的事件；为过去占卜不适合这个目的，仅因闲置与近期的个人问题。要知道上辈子的信息，最佳的方式是自我催眠和轮回催眠。

但你能习得的最佳课程是你自身的灵性权力。用通灵强化的各种技巧，你的精神得到强化，用我们描述的工具帮助了你将无意识的信息桥接到了你的意识精神中。我们想要它成为你熟悉的末世，无论你选择的工具是什么。

# 准备占卜

在最后顺时针旋转茶杯三次后，拿起杯子，看看黏在内壁上的茶叶。将茶杯的杯耳作为代表你或顾客的焦点，你不仅要看图案（象征），也要看它们所处于的位置

# 位置，位置，位置

你应当顺时针去解读，从杯耳的左边开始。杯耳的最左边是当下，接下来是第三个月，六个月，九个月，以及到了杯耳右边的一年。你可以轻易地将每个“三个月”分成单独的月份。

# 解读时间

靠近边缘的象征代表月份中的初旬，靠近底部的则是月份中的后旬。完全在底部的象征们则是未来一整年的背景。触碰侧边和底部的象征则是从原地到剩下一年的情况。注意这些问题。如果占卜的是明确的问题，那么时间可能不是需要考虑在内的因素，可被忽略。如果你要占卜的是过去的情况，那么请无视上面的时间动态，即便其动态就是类似上图中那样往反方向看的。

# 如果它看上去像只鸭子...

现在，仔细看看杯子的整个内部，看看象征。有些是独立的，有些是群落的，有些会彼此叠加，相互整合在一起。

你看见了什么？即便轮廓很模糊，但你仍旧能感知到（例如）它是个鸭子，那就将它认作是鸭子。不要让外形的模糊影像你的感知，但也不必接受感知到的第一个概念。你可以反复思索下，看看你还能感知到什么。

# 出神一点

# 阅读“整个杯子”

在你给每个象征一个名字之后，注意其位置，将其置于杯子内的整个“大图案”中，现在，是时候让整个杯子来告诉你其故事了，从杯耳的左边开始，顺时针绕圆进行。

# 进入轻度出神

让自己轻度地出神，感受每个象征告诉你的故事中的一面。有时每个象征是一个特别的信息，有时它是一群的象征，它们最终在一起形成了一个故事。它就像是书里的章节一样——每一章都有故事要说，但它们在一起形成了一本完整的书，述说着你或你顾客的人生。

# 写下它

你最好将象征和它们的位置，以及你最终得到的信息都写下。有时候，写下的记录最终会让你理解你是怎样解读的。弄一本占卜记录书是你开发通灵技巧过程中能做的最重要的事了。

# 自己弄本辞典

你可以用象征的辞典——你可以在本章最后找到推荐，但你最好自己写一本。随着你不断尝试占卜，不管是茶叶占卜还是其它占卜方式，你会累积自己的理解。不要让任何辞典中的定义限制你的感知。你应该让它们作为一种种子，让你想象力如花朵般绽放。如果茶叶图案看上去像是一辆车，它可能不是指一辆车，而是指车的功能——它运输你。相似地，飞机也运输着你，但通常距离被车更远。

# 象征的古怪语言

换句话说，事物或人很少被象征代表，代表的通常是他们做什么，比如，在一些情况中，是他们在神话或名著中所代表的。

有一种情况是例外，就是当你的问题是明确关于人或物件的——例如：我什么能得到我最想要的车？或是是否会嫁给老王？

# 复数含义

关于象征，还有一点你需要知道的是，所形成的图案很少只限于指一件事，多数是其它所触及或附近的象征的一部分，构成一个完整的故事。根据故事中的元素的顺序来解读它们。总是要问问自己“为什么这个象征出现在这一时刻上？”

# 象征的简短辞典

下面的辞典是你将会经常遇到的象征解释。注意，象征的含义并非总是指东西自身：铁砧很少是指铁砧自身。随着你占卜次数的增多，你会经历更多的象征，但它们的含义应当都来源同一处——你的潜意识。你要将你潜意识中的信息通灵到你的意识精神中。在辞典里的含义，有时会让你觉得是错的。如果是这样的话，那么你就应该看看你自己感知到的信息是否正确，至少对这次的占卜是如此。这里给“猫”的定义是背叛，可能会让多数爱猫人士感到冒犯，但如果他们有思索下猫的天性的话，它们是否是表达为“喂我，摸我，否则我要干坏事，”那就有可能理解为什么会有那个定义了。

## 咖啡渣占卜

> “咖啡应当像地狱那般黑，像死亡那般烈，像爱情那般甜蜜。”

> ——土耳其谚语

咖啡渣占卜对于西方世界而言是相对较新的概念么，但在中东和地中海区域附近却是传承自古的占卜方式。更古老的概念是通过容器内水剩下的残留物来进行占卜。中国人在用过的铃铛内部找到事件的预兆，而中国式茶杯的样子当反过来的时候，的确像是一个铃铛。罗马人会用高脚杯中红酒的沉淀物，毫无疑问，占卜师都知道类似的例子。

咖啡作为饮料似乎是源自埃塞俄比亚的，然后被阿拉伯人销售到中东，进入欧洲的南方和东方，再进入美国。多数关于咖啡的传说多是关于土耳其咖啡的，因为土耳其帝国是那个时代的文化中心。

## 土耳其或其它重口味的咖啡

咖啡渣占卜基本上是与茶叶占卜类似的，都是在喝的时候为占卜进行准备。虽然多数传统上的指导用的都是土耳其式的咖啡，但现在很多人喝的是星巴克和其它咖啡店煮的的咖啡，有体验过多种不同的咖啡和煮咖方式。

## 土耳其咖啡的准备方式

方法都很直白，但准备方式的确是魔法上的仪式，需要你注意过程中的每一步，因为你是在聚焦意识，以在机械化的过程中加入“魔法”。

## 建立仪式

不管是在为一个人还是多人准备咖啡，聚焦行为自身就是在告知你的潜意识，呼唤元素生命力。因此，很多咖啡占卜师会喜欢将每一步融入一个仪式中——从选择咖啡豆开始，烘焙它们，研磨它们，以及下方会写明的剩余部分。某些真正的咖啡占卜狂热爱好者会做诸如双手捧着豆子，并吟唱传统诗歌的行为。

去，招来，准备好；

所有好的降临，所有恶的出去。

(来源：《咖啡杯内的占卜》索菲亚 著)

索菲亚也有建议用通灵的净化水来配合这个吟唱：

如雪般纯洁，如雨般纯洁；

如海般纯洁，你将是。

## 咖啡渣占卜的准备

+   1. 你需要那种小的咖啡壶，可以在煤气炉/电磁炉或其它炉子上烧的那种。你最好用和茶杯同样斜面的杯子，但传统的小型土耳其式杯子也可以，甚至更好。你也需要杯托。

+   2. 你要弄多少杯咖啡就往壶里倒多少杯冷水，每四杯水额外再加上半杯水。乘着水仍冷的时候，每杯水加入一调羹土耳其咖啡粉，并搅拌。咖啡粉的量取决于口味，但要记住，在喝完咖啡后，杯内应该留下一层厚的咖啡渣。

如果你想要加糖，那你可以这么做了，而不是在咖啡杯加入杯中之后。

+   3. 尽可能慢地加热咖啡壶。火越小，味道越好。注意它不会沸到壶外。这种观察的行为也是意识的聚焦，会给这个过程加上“神秘”。

+   4. 当水煮开后，均匀地在各个杯中平均地加上咖啡。每个杯内的咖啡大约四分之三，以让壶内表层的泡沫倒进去一点，没有泡沫的话，就没有土耳其式的风味了。你可以用调羹挖一点泡沫进去，但小心地倒咖啡会加入其自身的能量。将剩下的咖啡再煮起来，将剩下的咖啡分配到每个杯子中。

因为在这个过程中不会用到咖啡滤纸，你应当几分钟再喝咖啡，从而咖啡渣能够沉淀到杯底。

另外，你可以在倒咖啡前用索菲亚推荐的咒语，用你的左手将杯子顺时针转动，并吟唱：

火焰的力量，水的愉悦；

知识的杯子，土地的豆子；

凭借蒸汽与梦想，诞生智慧吧。

1. 在冥想你的问题或当前的状况时，慢慢地喝下咖啡。你要几乎喝完咖啡，从而底部能留下一点咖啡水和大量咖啡渣，顺时针旋转杯子三次，将你的杯托改在咖啡杯上，再快速将咖啡杯底朝天翻个身。这会慢慢地将咖啡渣沿着咖啡杯内壁掉下，掉到杯托上，形成你解读当前问题所需的图案。

2. 等几分钟让咖啡渣沉淀，再拿起杯子，进行解读。

咖啡渣占卜的规则与茶叶占卜的有所不同，主要表达在每个传统的不同文化背景上。记住，即便你是在一个文化背景下为自己或他人占卜的，占卜传统的来源可能会给你的解读添加上微妙的风格变化。

## 杯和提问者

杯耳代表你，提问者。靠近杯耳的象征代表靠近你家即将发生的事。不管是在哪边的象征，如果它们是指向杯耳的话，那就说有什么东西/事件或人正在接近你；不对着杯耳的象征意味着有人或东西将离开你。

靠近杯子边缘的是未来中的；靠近中间的代表当下；靠近底部的是过去。处于杯子正底部的任何象征可被解读为不幸。在传统中，提问者会被要求把底部的图案弄破，以期破坏掉坏的运气。

## 总体的图像

象征从来不该单独看。你要整个结合起来的图案，再根据它们的大小和位置来看它们彼此之间的关联。

起初，你可能无法理解任何看到的图案。所呈现的图像可能是不完整或模糊的，让你感觉毫无意义的。因此，咖啡渣占卜是比茶叶占卜更难的。你要放松，仔细看几遍，近距离，远距离，不断思考其含义。在索菲亚的书里有一
些简单的指导你可以参考：

聚集在一块的小颗粒可能是指动作

清晰的直线表明必须要引向明确目标的计划

曲线代表不确定

轮廓不清晰的东西表明尚未决定或障碍

你很快会一个接着一个搞清楚图案的含义，在你的想象中会形成一个景象，告诉你一个完整的故事。随着你占卜次数的增多，潜意识精神会回应技巧，你解读出来的故事会变得更丰富，更具细节。

## 你总要有个起点

这是你需要记住的重要一点——就像茶叶占卜，你是在将你当前的情况作为一个起点来讲一个故事的。你可以每次占卜问一个问题，你也可以只是看看在总体上的问题。

## 中东文化

单个的象征一般的释义与茶叶占卜中的相同。但是，如果你参考咖啡占卜起源之地的中东文化来进行解读的话，也会有不错的发现。让那些文化的影响塑形你的想象，但你也要小心不要让自己对另一国家的文化偏见影响你的解读过程。

## 沙漠的现实

出现天使的形象很容易被占卜师认为是好运的，但出现蜜蜂的形象对于沙漠的居住者有不同的意义。在沙漠中，出现蜜蜂说明附近有水源，这意味着生命。

在传统的沙漠文化中，马就像在美国牛仔中的地位一样重要，象征着力量和独立。树木的含义更好，而车轮可能是指命运将发生变化。在伊斯兰世界中，新月标记可能是指宗教层面的呼唤。

## 拓展你的感知能力和潜意识

好吧，这不是电视广告，我们不需要任何促销，除了重复书名中“在茶杯中看见你的未来”，要达到这点，你需要练习这种活其它的占卜体系，从而拓展自身的感知意识和潜意识。

随着你不断运用茶叶和咖啡渣占卜，你会开发自身潜在的通灵能力，变得更容易运用其它类型的占卜。

## 占卜的众多类型

在众多占卜中人们最熟悉的是水晶凝视，灵摆，地占，测字，易经，卢恩，读沙，塔罗，和简单皱纸占卜。

## 无意识的语言

每一种占卜都是揭露性和体验性的。你不仅是“看见”当下的状况随着未来的种种可能得到展开，你也是在体验生命与意识的新维度。所有用到象征的占卜和象征都是无意识的语言。随着你不断探索它们的含义，它们也会
通过另一种方式与你对话——包括你的梦境和直觉。

## 隐藏在各处的信息

在各个地方都有着“隐藏的信息”——在云朵的形状，自然的声音，鸟儿和动物的无缘，“有意义”的巧合，行星的运动和石头的触碰，在含糊不清的言语和对几乎一切难以说清的感觉。就像潜意识精神通过宇宙无意识打开了意识交流一般，能对无法言语的需求和未来的可能做出回应。

## 自然而然的重要性

通常你的注意力会自然而然地被事件、物品或人吸引，当这种情况发生的时候，你要多加注意，寻找可能的象征，让自己接收其含义和信息。

## 左脑，右脑

我们经常被教导要客观，寻找硬证据，要理智思考，逻辑上分析事物，寻找不同。这就是所谓的‘左脑思维’。

相反的，我们也有不怎么开发的‘右脑思维’，由主观和软事实，直觉和创造力构成，以综合‘整体’，找到相似处。

大脑并没有真正的分割——两个半构成一个整体——但是，我们都会根据自身的思维模式做出无意识的选择，因为我们多数人不是右撇子就是左撇子，很少有人能够左右皆宜。但是，我们能够选择去开发被忽视之手的力量，同样地，我们也能够学着更整体地思考，让两种思维模式都得到尊重。

## 整脑 & 思想开放

整体思考和思想开放式的观察就像是整颗心和半颗心之间的区别。当我们全心的地区生活，用开放的思维去看的时候，人生会变得更丰富。

## 不再限制在三维中

占卜就像那样。它让我们的人生充满更大的意义和感受——不再让我们仅限制在三维中，而是四维，甚至五维。随着我们的意识得到拉伸，我们的意识也会对被我们通常称为的“高等世界”敞开。

## 下一步

生命和意识自创造之初就得以存在，无论那创造之初是被称为“大爆炸”还是太初有道“，在那亿万年的进化中，从最极简到复杂和多样化不断演变。然而，这么多年来，很多生物都已经走入了终结，从最小的微生物到巨大的恐龙都有不再存在的生物，但进化仍旧继续，让人类出现，变成了它如今的主要媒介。

在人类的已知历史中，都是关于成长和意识拉伸的，无论是被个人成就还是文化成就所测量的，文明现在已进
入了全球化。随着意识的成长，我们走向了人类进化的下一步。这可能看似不重要，的确，它也不是有意识的选择。但能够认识到这件事能够让我们的日常旅程增加了一点乐趣。
