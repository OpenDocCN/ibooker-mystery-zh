# 第一节 三元九运

三元九运是对时空的一种划分。风水应用这种划分去断吉凶，去建造，去布局，去趋吉避凶。

常常听到这样一种说法，风水学包括两大部分：一是峦头；一是理气。峦头学是形相学，研究山形水态，环境形势，说左青龙，右白虎，前朱雀，后玄武，以其善恶美丑强弱来断吉凶，相对地并不带时间性。理气就不同了，理气是时间空间方位学，不同的时空在同一方位有不同的吉凶，绝对地带时间性。

本来，三元九运是很简单的。三元就是上元、中元和下元。

在每一个元里再分三个小运，“三三归九”，就是三元九运。

每一运是二十年，每一元六十年，三元就是一百八十年。

换言之，上元是一二三运；中元是四五六运；下元是七八九运。每一运二十年，每一元都从甲子年开始。这不是很简单吗？

不过，现时坊间却出现了新的说法，使人糊涂，令人迷惑。

有两种说法。

一种是说没有五运。
那么，五运的二十年跑到哪里去了呢？
他们说，五运的前十年拨入四运去；五运的后十年拨进六运去，就是说，中元的四运和六运都是三十年。

另一种说法是各运并不都是整齐的二十年，有长有短，一运十八年，二三四运二十四年，六七八运二十一年，九运二十七年。

按照第二种说法，八运在一九九六年就到来了，并不是二00四年。

这两种说法是从哪里来的呢？是来自“洛书九宫图”：

> “戴九履一、左三右七、二四为肩、六八为足、五在中宫。”

洛书图再配以先天八卦。
由于只有八个卦，八个方位都能配得一卦，单独位处中宫的五没有卦。

于是，没有配得卦的五，就连运也没有了。要一分为二，把前后十年拨归四运和六运去。

三元九运的计算也因元运配卦而改变了。他们是以卦爻来计算，阳爻九年，阴爻六年。于是：
一运坤卦，三阴爻便是十八年；二运巽卦二阳爻一阴爻便是二十四年……如此类推。

以上是卦爻的数字游戏。
其实，三元九运是太阳系星球运转的产物。

时间是由星球运行而产生，这是常识。地球自转一周是一日；月球绕地球公转一周是一月；地球绕太阳公转一周是一年。

我们用十二生肖配十二年，就是十二年一个周期。为什么十二年一个周期，而不是十三年或者十年呢？

这是因为太阳系最大的星球——木星每十二年绕太阳公转一周。木星每年所转到的方向就是太岁方。木星被称为“岁星”。因为木星的能量对地球有很大的影响，如果“太岁头上动土”就凶祸难测。

体积次于木星的是土星。土星绕太阳公转一周是三十年。每二十年，木星和土星相会一次。这两大行星的会合，对地球的影响是巨大的，深远的。

先贤洞悉这一天机，便以土木两星二十年会合一次为一运，以九大行星配九运。并且，以木星公转十二年和木星公转三十年的最小公倍数六十年为一个甲子，也就是一个元。

元、运、年、月、日、时都是“斗转星移”——星球运行的结果，并不是数字游戏。
我们用玄空飞星替客人看阴阳宅风水，都必要先弄清楚建造的年份。不知时运，就不能起星盘，不能论吉凶，定布局。
人有命运，阴阳宅有宅运。人的命运吉凶可以推算；阴阳宅的宅运同样可以推算。看风水，其实就是算宅运，就像替人算命一样。人算命根据出生的年月日时；看风水宅运就必须根据宅的建造年份。没有正确的时间（元运），就不能看风水，断吉凶，并作出趋吉避凶的指导。

# 第二节 年月九星

计算年家九星何入中宫，钟义明先生在《玄空星相地理学》一书中介绍的方法很好使，在此介绍给读者朋友们。
其法只需将公历年数的数字逐一相加，若总数大于11者再将两数相加，直到其总数小于11为止。最后用11减其得数，剩几就是几入中宫。然后顺飞九宫即可。
如1997年，1+9+9+7=26，大于11，再将其两个数相加，2+6=8，再用11-8=3。即一九九七年的年星为三碧入中宫，顺飞，则四到乾、五到兑、六到艮、七到离、八到坎、九到坤、一到震，二到巽。

九星中，以一六八为吉，九紫平而偏吉，三碧四绿平而偏凶，二五七为三凶星，其中五黄为最凶，故有五黄到处不留情之谓。

计算五黄飞临何方的捷诀，先确定入中之九星，凡与入中之星合十的后天宫位就是五黄所到的宫位了。如1997年三碧入中宫，与三合十者为七，后天数为七的兑宫就是五黄所到之宫位。又如2003年六白入中宫，与六合十者为四，巽四宫就是五黄飞临的宫位。

五黄之方为戊己大煞，其煞气非常之重，择日时山向或修方位临之断不可用，犯之易生祸端，若再有其他凶星介入，则发祸快而猛烈。也有说五黄之方可向而不可坐的，坐之则大凶，但笔者认为坐向都不犯之为宜。

若修方或山向不小心犯着五黄凶煞，可以在其方位上挂五个铜钱或铜制风铃而化解其凶性。

年家飞星，以坐山为主论其吉凶。

九星之用法，年月同忌，但以年家为主，月上次之。一般情况下，月家飞星的吉凶只对年家飞星的吉凶起着损益作用，其吉凶的主体还得从年家飞星上体现。如年家飞星为二黑凶星飞到某方，又正好遇到月星的二五飞到此方而助其凶性，则其凶愈显。

月九星入中之法：
子午卯酉年正月八白入中。
辰戌丑未年正月五黄入中。
寅申巳亥年正月二黑入中。

月星入中，逐月逆数而去。如正月八白入中，二月则是七赤入中、三月为六白入中、四月五黄入中、五月则为四绿入中……

月星入中宫后，与年星一样，顺飞九宫。如八白入中宫，九即到乾、一到兑、二到艮、三到离、四到坎、五到坤、六到震、七到巽。

年月九星飞宫之法，还可以用来断阳宅的年月运气，凡年月同时得吉星飞到厨灶、大门或卧室必有喜庆之事发生；若年月同时逢凶星飞到厨灶、大门或卧室，则必有不吉之事发生。年月飞星逢二黑和五黄重叠加临于厨灶、大门或卧室以及肮脏污秽之地，必有病痛或其他凶祸临身。

# 第五节 兼向与替卦

《玄空学》中把罗盘圆周360度分为三百六十爻，每一爻即为1度。
规定：每卦山正中偏左或偏右4度以内的为正向，排卦称为下卦。
凡偏左或偏右在4度~6度以内的为兼向，用替卦起星。
凡偏左或偏右6度~7.5度者，为卦线空亡。即在两山之间不知道为何山，一般情况下，卦线空亡断为凶。特别是阴阳交接处，最凶。

兼向有的替有的不替，如子、癸、坤、未、戌、乾、亥、辛、酉、午、丁等不替。

替与不替不必特意去死记硬背，因为它与正卦是重复的。比如“子、癸”属坎卦中的二个山向，坎卦壬子癸的卦数是一，替卦的歌诀“子癸甲申贪狼行”中的“子与癸”也是一，因为一与一重复才不用代替的。

关于正向与兼向的飞星该怎么个飞法，我在《玄空飞星速通法》一书中已详解，这里不赘述。

# 第六节 上山与下水

> 《青囊序》曰：“山上龙神不下水，水里龙神不上山”。此语是玄空学论断吉凶之枢纽，祸福之关键。

当旺之星飞到星盘的山盘上，谓“山上龙神”。“不下水”是指山星的当旺之星不可以飞到向盘上，如飞到向盘上则为“下水”；当旺之星飞到星盘的向盘上，谓“水里龙神”。“不上山”是指向星的当旺之星不可以飞到山盘去，如飞到山盘则为“上山”。

立向最忌上山下水之格局，凶祸立见。然亦有犯此而发者，其宅地必龙真穴的，或形局颠倒、又得向首与入中之卦合十，并有一、二节连珠吉水通照始可发。然福来不全，祸来甚速，不能如旺山旺向格局之悠久。如：

# 第七节 收山与出煞

收山出煞：将山上旺气、生气之星置于高处或高山处，将向上旺气、生气之星置于低处或有水处，是谓“收山”。将山上衰死之飞星置于有水处或低处，将向上衰死之飞星置于高处，是谓“出煞”。

# 第八节 反吟与伏吟

山向两星如若五入中宫，顺布八宫为伏吟；逆布八宫为反吟。玄空学论断宅运，甚忌讳犯“反吟”或“伏吟”，犯之者易遭凶害，重者死于非命，祸害之力量作用往往超于上山下水局，可见其严重之危害性。兹对伏吟与反吟细讲述如下：

五黄入中逆飞，飞布之八星必然与元旦盘字字相反，即运星与元旦盘数相冲对峙。如五入中宫逆飞布八宫，4到乾、3到兑、2到艮、1到离、9到坎、8到坤、7到震、6到巽。如1为坎宫数，9为离宫数，则两个数互相飞到对宫相对峙，称为反吟，余类推之。

（2）凡坐山立向，其中天盘一方为五黄者必犯反伏吟。反吟与伏吟比较，反吟比伏吟祸害甚烈。五黄为中央戊己土，为皇极，不为八宫卦属性，但进入每运每宫，就依附每盘五行属性发挥作用，有时互为相冲，有时互为克煞，形成灾煞。如果五黄落到艮宫，必为大灾煞。若五黄进入中宫，回归本位，不再替代依附各宫五行卦气，煞性煞气消减。因此，宅运盘形成的反伏吟都是由五黄引起而起灾难，尤其是立向之方位。

（3）宅运盘出现六宫伏吟，即六宫兼向出现的卦气重叠，也是伏吟，但重叠不会出现反吟。如果属当旺之令星犯伏吟，不会为害，反会为福。如果属生气星犯重叠伏吟，也不会为害，也反会有利。如果属退气、衰气，死气之飞星伏吟，即会加速退气，加大衰气，加倍死气，这就以凶而论断之。

（4）五黄入中逆飞布八宫时，形成某宫位非坐山立向而飞布为犯反吟。因为五黄入中宫飞，其所飞布的某宫位，必是旺星飞到，其宫位所用，必可发福，但其它宫之飞星，可能出现反吟之现象。

（5）另外还有一种反吟卦，即“山星与向星对角合十”，但必同时合成三般卦。

（6）满盘犯伏吟，切不可用。如坐山低陷或空缺，主堕胎。向首有山，或面山有高峻的岩石，主目疾。正合《玄机赋》所云：“坎宫低陷而堕胎，离有恶岩而损目”。

（7）反吟、伏吟之局宜施用平洋，地势平坦空旷，坐后有水流、聚者，若施用山龙，必定发祸。

# 第九节 飞星合十

所谓合十者，即是玄空盘中每宫山星与运星合十，山星与向星合十，或向星与运星合十也。其合十之功能效用在于使宅运全局九宫各卦互相通气，从而使宅局全盘飞星气场流畅。

惟其合十，乃通中五戊己之气，而万物生焉，万物育焉，生生育育，而变化无穷焉。老子号中五为玄牝之门，其中神妙，诚令人不可思议者也。

是以墓宅之山向挨星、飞星满盘合十者，则八宫卦气均与中五相通，即借中宫戊己之力，阴阳二气互为交感，化育无穷，而丁财自然鼎盛矣。故山向星盘满盘合十最吉，仅中宫与山或向合十者亦得吉征。

合局合十也是一种补救措施。在一运和九运中没有旺山旺向，全局合十可以起到旺山旺向之功效。

虽为合十，可为不得旺山旺向之局补救，但应注意该局若犯上山下水或双星会坐，而背后又无水者，虽满盘合十，此宅局亦不可用。

如果该宅局令星到向、向上有水，或有城门诀为可用之宅局，这时若又得满盘合十，此宅局不但能使用，且吉上加吉，此种宅局用后作事顺利，家人和睦，财源不断，广结善缘，主生贵子，升职掌权，富贵双全。

# 第十二节 七星打劫

> 《天玉经》云：“识得父母三般卦，便是真神路，北斗七星去打劫，离宫要相合。”

## 一、何谓北斗
北斗者，乃随宇宙元运运行人中五行之天地立极之气。

## 二、何谓七星
章仲山讲，弄清七星者，首先明白父母三般卦。
章仲山直解“三般”，是从一二三四五六七八九之中。每经四位所抽取出来的三般。即先提出“一”，隔四位取“四”，再隔四位提出“七”。

一为坎，四为巽，七为兑，这样提出来的三个数便组成“一四七”三般卦。同理，若先提出“二”，隔四位提出“五”，再隔四位提出“八”，就组成“二五八”三般卦。于此，若先提出“三”，隔四位提出“六”，再隔四位提出“九”，合组成“三六九”三般卦。故此“一四七、二五八、三六九”就组合成三组三般卦。

玄空运用三般卦按元运生向排入九宫中后，一四七飞落到坎、巽、兑三个宫位，三六九就飞落到震、乾、离三个宫位，二五八就飞落到坤、中、艮三个宫位。

## 三、何谓打劫
何谓打劫，即上元可劫夺中元之元气，中元可劫夺下元之元气，下元可劫夺上元之元气。

## 四、打劫之法
形成七星打劫之法有以下几项构成：
1、必须当运旺星到向首，始可劫夺未来天元之气。
2、要一四七、二五八、三六九之三般卦，要在离震乾三宫飞布，或在坎巽兑三宫飞布。
3、七星打劫是以双令星会向之向方为打劫成局。如七运午山子向下卦双七令星到坎宫方位，与巽宫、兑宫山向形成三般卦，符合七星打劫，则称“坎宫打劫”；七运壬山丙向下卦双令星七到离宫方位，与震宫乾山向形成三般卦，符合七星打劫，则称“离宫打劫”。

4、玄空学中只有为“离、震、乾、坎、巽、兑”者向位，在元运中才有形成七星打劫的机会。而宅向为艮、坤者，则在元运中永远无法形成七星打劫运局。

5、形成七星打劫宅局中的三般卦三宫位中，吉气、旺气、生气是互相流通的，是吉位，旺人丁，可以用来开门路、留气口、铺睡床、安炉灶之用。形成三般卦的七星打劫局，如房屋周围条件许可，同时开门、留气口（窗口、侧门、后门），才能使三般卦七星打劫局在宅运中起到旺丁催财的作用。

## 五、何谓真打劫与假打劫

1、真打劫：凡当运山向旺星飞到离宫向者，乾宫、离宫、震宫三宫山向飞星合成一四七、二五八、三六九运局三般卦者，为真打劫。

2、假打劫：凡当运山向旺星飞到坎宫向者，巽宫、坎宫、兑宫三宫山向飞星亦合成一四七、二五八、三六九运局三般卦者，为假打劫。

凡遇真打劫，自能发福。其使用理气若符合宅运盘中布局，但不符合宅局形势，空与实不合，山与水错位，则无法在当运中去劫夺未来之气，稍有理气不合，形势不符，即易发祸。平洋阴阳宅以南北向最多，如遇此七星打劫运局之造化来催财旺丁，真透出玄空学深奥之秘义矣。

双星会向布盘形成的父母三般卦及七星打劫局，虽是玄空学中的上乘运局，使宅运快速发福，令宅主在短时间内赚到大钱，但美中不足，而使宅主的身体受损，重者可患上癌症或其它绝症。如七运中，选择了一处双七飞星飞临的运局，这种运局出问题的地方就是双七向盘飞临的向方，当然有满意的财富收入，但山星亦飞到向方，身体自然就会出毛病，如果向方有水、水外又有山，毛病还稍轻些，如果只有大水河流低洼处，没有高山或形势，犯的病就重了，这就叫得了财富，毁了身体健康。望风水师给宅主选择立向时，切要注意。双星到向的真打劫与假打劫，均能催富。但真打劫力量比假打劫更有力，都会令身体出问题。如遇七星打劫布局一定要在向方加放大型瓷器或修假山，化解煞气而壮旺山星使身体尽量少犯些毛病，这样既旺财又确保身体健康。

七星真假打劫各有二十四局，除犯反吟伏吟不用外，各有二十一局可用，但务必注意宅局理气与宅屋环境上的生克制化。玄空学在理气风水学中为上乘真品，学到精处，与人心中起经纬。三般父母卦形成的七星打劫局，如若再遇上全局又合十，则是真打劫宅局中的上乘之佳局。北斗七星真打劫局如下：
