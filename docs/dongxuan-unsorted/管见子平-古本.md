# 正官格

正官一位格清高，帶印忠良果俊豪。無煞身強福坦坦，有財氣旺志堅貞。卻嫌分奪喜埋沒，最怕刑沖噬破零。內外合兮官不顯，支干傷尽数難逃。

正官只宜一位，則權專。喜見印綬，稟性忠良英豪淳俊。不宜見殺，無殺則注情舒坦。財乃官之元神，見財則有養翼自得之意。分奪者再入他格，埋沒者減而不見。貪合忘官，柱中忌見刑沖破害，局內最嫌。至於傷官格中忌神，見之者善不永也。

# 偏官格

偏官賦性最剛強，身殺相爭定不祥。殺淺身強權可羨，氣衰殺重命難長。刃星得合功名遠，印綬相生福祿昌。只怕殺官多印少，始終貧困事乖張。

偏官者，陽克陽，陰克陰之類是也。無陰陽配合，故謂之偏。身強有制伏者為偏官，身弱無制伏者為七殺。身弱殺旺，身殺相爭，俱不為佳。身強殺淺，假殺為權；殺重身輕，終須有損，正此之謂也。年刃能合殺，傷官能駕殺，印綬能化殺，此四者見之，主功名遠大，富貴綿長。但忌官煞混雜，又無印化，不但福氣匪薄，亦且難為書生。陰日傷官駕殺，陽日傷官不能駕殺，止可論傷官帶殺矣。

時上偏官一位良，不分明暗喜財強。食神最發於頭見，羊刃偏宜支下藏。性格剛強多智略，襟懷磊落好文章。食神太重無妻要，黃卷青燈空自忙。

時上偏官只喜一位，即時上二位貴，再見者非也。或天干地支有者，俱喜財星生之。且一位偏官，勢不能鼓舞造化，必藉財以生之，是為財滋七煞，英雄獨壓萬人。食神干上以制殺，羊刃支下以合煞，二者宜見。時上偏官，主為人耿介磊落，以文學馳名。如時煞無氣，傷官食神制太過，又無偏正財星生助，乃遇而不遇之人。賦云：偏官時遇刑伏，太過於寒儒，蓋謂此。

# 正財格

正財旺相要根深，內助妻賢鼓瑟琴。柱帶比肩恆不足，運臨七煞福彌深。五行生處多獲寶，三合逢時廣積金。官印兩般格功名，早遂少年心。

正財格必要支下有根為貴。賦云：地支財伏暗生者奇。蓋財者忌也，財單有無，必有貧乏之憂。比肩分財，自然不定。財多身弱者，反不忌之。賦云：命用財，運逢殺吉亦壞隙，故云福彌深也。用財者，要柱中應有生氣，三合局中最妙。財合前數如此者，名積財寶金珠之富。再見官印，此乃三奇全備，定主早登科第，乃富貴雙全。祿嬌沖破，馬忌空亡，不可不知。

# 偏財格

偏財生旺在月支，慷慨風流果出奇。官煞清純財祿盛，比肩混亂利名虧。平生事業由妻掌，半世生涯賴子為。墓庫逢臨沖劫地，壽元福氣兩參差。

月上偏財最宜生旺，為人慷慨風流。或官或煞，止許一位，方為純而不雜，乃有福有祿之人也。比肩認財重見，則混亂。偏財乃眾人之財，能有其機，若被比劫分云，其利若自然廳損。偏財格主人情性疎懶好自在，凡事憑妻子寵愛。偏財如行運到墓庫死絕，卻殺分奪其秀氣，福壽豈不參差矣。

# 時上偏財格

時上偏財一位佳，功名有分仕途賒。他鄉發福高相敬，正室分緣愛妾華。官煞不雜成吉慶，比肩多見謾咨嗟。重財旺身無力，原是名門老故家。

時上偏財與時上偏官同，止喜一位，多則不美。若時財有亂，日主剛強，功名遠大，仕途悠長之命。時上偏財多發福於好郡，主有高人相敬。或官或煞，不可混雜，務要合益。偏財多愛偏妻，時上偏財年提又見，日主無氣，此是財多身弱，正為富屋貧人。賦云：時上偏財別官忌見。又云：時上偏財怕逢兄弟，此之謂也。

# 正印格

正印從來生我身，柱中先要識元神。煞官不見難言貴，偏正財多定是貧。劫重見財還有用，印輕透殺反為親。食神傷官兩相犯，風症纏綿病不仁。

印綬者，生我日干也。官殺乃印綬元神，柱中不可無。財為印綬忌神，柱中不可有。子平云：重者生氣，若無官，當作清高持藝看。官殺不來，無爵祿，縱為拔薦也孤寒。又云：月生日干無天財，乃印綬之名。又云：用印見財，十有九貧。柱中若比劫重重見，反吉。賦云：財星破印，官逢比劫之官。如印綬輕者，喜投星透出，有煞印相生之義，最可說也。印綬見食神，比財尤甚。賦云：印綬重重享見成，食神只恐暗相刑。早年若不歸黃土，孤苦伶仃風疾纏。蓋述此義。

# 偏印格

偏印梟神格上居，無官無殺利名虛。妻星暗見愁無限，天厨明逢病有餘。性孤忙多計較，行藏遲鈍甚趨趨。臨處絕財鄉。

偏印者，陽生陽，陰生陰也。又名倒食者何耶？蓋乃惡鳥也，其聲甚怪，人聞其鳴則凶。柱中無食曰印，有食曰梟，能奪我之食也。如提綱偏印，柱無官殺，為虛名虛利之人。妻星者偏財也，能剋偏印，故主愁苦。天厨者食神也，蓋印忌見天厨，忌見食，食忌見馬。財云：年時月令有偏印，吉凶未萌，大運歲若逢壽星，災殃立至，即此理也。偏印性本奔忙，而幹事多主遲鈍，心雖計較，而謀為卻費趨趨。如行死絕之運，財星又來剋破，官殺不來救解，定為泉路之人。偏印若係偏出奇，主生帶暗疾，役於豪門，女以姪男養子為嗣，義女為妻，往往有驗，字不可輕忽。凡柱中梟食相爭，無官殺解救，決不得於令終也。

# 陽官格有二

五樣陽官論不同，身強財煞是英雄。火土金土金官土貴，水金水無官弱。柱中木火如成格，遵見財官為妻妾。只怕見官財印絕於身，勞碌任西東。

陽官格有五樣：有火土陽官，有土金傷官。二官不宜見官，喜用財，身弱用印，定主富貴。若見官星破了格局，不以貴論。又有金水陽官，喜見財官，二者若無官星，反不為美。又有木火傷官格，最喜見財，財多者見官不忌，無財者見官則凶。凡傷官格身旺無財，身弱無印，終身勞碌貧苦之人也。凡傷官多出枝藝，未免刑傷子息。

其二

傷官伶俐性聰明，傲物高強志尚清。財印要分輕與重，殺官當辨淺和深。柱中原有官鄉重，柱內無官見輕。財印兩明殺刃旺，定知囊貧足平生。

陽官為人性傲，故常以天下之人不如己。伶俐聰明，志尚清高。身旺有財，身弱有印，二者分明，為當貴榮達之命。柱中如殺官並見，當辨其淺深，然後可斷其禍福。凡柱中原有官星，再見官則為禍重；如柱中原無官星，縱運如見官星，禍輕。柱中財印既明，又有手刃七煞以助其威，定主功名顯達。如身弱無印，身旺無財，傷官見官，刑沖破害，主剋陷六親，難為妻子，乃九流之輩，僧道師巫之命。

# 食神格

食神原是星名，最發財扶喜弟兄。明暗要分輕與重，陰陽當辨濁中清。財彰隱形遠壯，印旺官強勢必傾。繼母梟神來入格，天福壽兩無成。

食神名天厨，亦名星喜星。比肩輔助，或干或支，當辨其輕重。書云：陽日食神無傷損，則暗合官星；陰日食神無破虧，契合目視印綬。陽日食神身旺忌見官星，陰日食神身弱忌見印綬。如見此二者，為破格還實不美。財透喜柱藏，以狀其根，本最宜見之。印旺官強，是奪其成，傷其勢，自然傾損。不論陰陽，食神俱以偏印為忌，用食見梟，其壽不延，又安能成其福哉。

# 月令建祿格有二

建祿身強旺本元，財輕印重不堪言。殺官不見同人透，傷食無形縱母存。漢謂歲強遂有禍，年通月利縱為也。不為湖海清閑客，也守空門度且自。

建祿天元坐弱宮，財影官旺福無窮。食神暗隱元無礙，祿馬明運卻有功。制伏還鄉多發達，財官旺地必亨通。不為廊廟青雲客，也是人間富貴翁。

月令建祿，日來旺，印綬太重，財星反輕，不可言福也。柱中原無官殺，卻有比肩遠出支下，又無傷官食神，反有偏印旺甚，無依。縱流年通暢，大運背祿，為福少而為禍多，乃九流僧道之命。若逢祿時坐弱官，又有財官並透，以制其旺氣；傷官食神以洩其旺氣，一制一洩，則中和矣。必廊廟功名之士，富貴享福之人也。

# 羊刃格

格中羊刃誤言凶，七煞傷官最喜逢。刃星喜不剋重逢，財破定難容。長綱然合真最貴，月令逢沖是乏農。無破無傷煞刃顯，邊萬里覓侯封。

柱中羊刃，不可便以凶言。如有傷官洩之，七煞制之，反為貴命。賦云：小人命內亦有正印官星，君子格中也犯七煞羊刃，是也。羊刃不忌多，只怕見財，則有爭奪之禍。如提綱羊刃，即有七煞合之，乃英雄豪傑之士，富貴之人之命。若月令羊刃而被沖破，又無無驚威嚴，止為鄉人而已。如無傷破而有殺星制伏，多出武職，主掌兵符，功名起於邊陲。羊刃剋妻，有羊刃多而反不剋者，是未見財也。若見財星爭奪，恐難容於不傷也。

# 日祿歸時格

日祿歸時格最親，提綱再見不為真。食傷愛有方言貴，偏正財無定是貧。有破有沖心弗遂，無傷無害志遠伸。煞官不見同人少，正笏垂紳拜紫宸。

子平云：日祿歸時沒官星，號青雲得路。此理甚微，必須究所以然之故，方知格局之真，縱後可斷禍福。沒者埋沒也，此沒字最要緊，不可輕忽。且如甲祿到寅，寅中丙火長生，用丙字暗合辛字，甲日得官星。丙戊祿居巳，用巳中庚合乙，戊日得官星。

# 拱祿格

拱祿格中止有五，無沖無破始為良。陰丁巳未宜財助，陽土馬龍得水強。癸日豬牛頭倒用，巳干巳未正相當。煞官不見無填好，與皇家做棟樑。

拱祿格中有五：日時中間虛拱祿星取貴。如丁巳日丁未時，中間虛拱一午字；戊辰日戊午時，中間虛拱一巳字；癸丑日癸亥時，癸亥日癸丑時，中間虛拱一子字；己未日己巳時，中間拱一子字。此之謂拱祿。甚財星忌官煞，刑沖破害，填實不貴，歲運同。

# 拱貴格

拱貴格止有四：甲寅日甲子時，中間拱丑字；壬辰日壬寅時，中間拱卯字；丙戌日丙申時，中間拱酉字；戊申日戊午時，中間拱未字。此皆虛拱日主貴人，故謂之拱貴。喜財食，忌官殺，刑沖填實，才犯者真貴人之命。又忌提綱無氣，方妙。如提綱見官殺，才須以拱貴論。書云：提綱有用提綱重，月令無神用此奇。空亡大忌，藏運亦同。

# 二格合言一旨

兩般虛拱妙難言，最喜其中識本元。貴忌空亡祿忌破，傷嫌重煞嫌刑沖。若有火難兇手刃，如無福始存。月令無神成上格，居官早踏紫微垣。

拱祿拱貴，其理則一。木元者所拱何物？如拱祿怕沖，拱貴怕空。提綱上傷官不宜太重，至月七煞不宜見年，刃卻財並宜去之，則沖破害皆忌之。月令無神者，無官殺也。局中忌神不犯，又有財星滋助，大貴之命。

# 拱祿拱貴

拱祿拱貴格同於一理，故總言之。

# 六壬趨艮

六壬趨艮六甲趨乾，二格同於一理，故總言之。

# 六甲趨乾

六甲趨乾互換合祿，是偶飛趨艮重審名有分。趨乾祿亥福無邊，財官暗見成火陰。沖併明逢是禍符，月令有神成別。

# 陰朝陽格

乙未天元時遇子，取名題貴理幽玄。煞官不見功名備，偏正財逢當貴全。沖絆難沾夫子祿，逢合准許帝王宣。空亡不犯無金剛，難塔題名在少年。

乙巳日得丙子時，用子字遙巳字為祿主。巳中有庚戊，為乙木財官。乙貴在子，故取乙巳鼠貴之名。不宜庚辛申酉丑絆子沖破子格局，是以不貴。喜財星，忌空亡。

# 井欄又格

申子辰全三透庚，井欄又格局澄清。盤中少火方無礙，柱內多金卻有情。時帶子申應減福，運臨辰卯必增榮。丁壬癸亥傷沖不寧，相名亦不寧。

庚子日二個庚字，地支申子辰全為合。此格取申子辰暗沖寅午戌，庚日得官星，乃大貴人之命。忌丙丁巳午寅戌亥壬癸透殺，反不為貴。如子時即丙子時，上見七煞；申時是日祿歸時，壬癸傷官俱不稱美，則減分數。二庚四庚又非此格。或日主一庚遇巳格，細推庚辛干上通宜去，申酉文中不可留。若問功名休踏馬，欲求富貴且騎牛。提綱旺氣通元用，衣紫腰金拜冕旒。

# 子遙巳格

子遙巳格有二：子字遙合巳字，以巳字邀酉字，甲日得官星。賦云：甲子日再遇子時，忌庚辛申酉丑午，要月令旺氣通於日。

# 丑遙巳格

丑遙巳格要追求，財星喜見誠飛砂。官殺愁逢實司憂，子未無情名蹭蹬。酉申有用喜淹留，歲時若也逢蛇鼠。虛利虛名到白頭。

此格以辛癸二日干，要丑字多者為砂，用丑字遙合巳字，取巳中丙戊為辛癸之官星。柱中原無官星，方用此格。若見子絆未沖，則不能遙合矣。賦云：辛癸日多逢丑地，不喜官星。歲時見子巳二官，虛名虛利。

# 飛天祿馬格有五

庚金二日子重重，祿馬飛天用到衡。庚見丙丁難許吉，壬逢戊巳斷言凶。丑終無明通九夏，年沖何路向三冬。五行砂處宜純粹，龍庭活萬戶封。

此格以庚子壬子二日為主，要子字多，暗沖午字，庚壬日得官星，主貴。如柱中見丑字，則合子不能沖午；見午子，則填實子字無用。二子午也，九夏午也，填實羈絆，皆不能為福也。

其一

此論從草格也。如庚辛日干為生，柱要申酉戌全，或巳酉丑全，生於七八月方為正論，生於九月又非也。從草不要官殺，方為貴命。如見官殺，破了格局，不取從草。如合此者，乃登雲折桂之士。如見水者，有金水澄清之妙，金堅受水之相涵，文學堪誇，乃聰明特達之士，吟風弄月之人。子平云：金水聰明而好色，蓋述此也。喜忌合式，乃名公巨卿之命。入從草體全，亦名白虎伏形，取金之象也。青龍木也，玄武水也，柱中之喜者；朱雀火也，勾陳土也，柱中之忌者。庚屬大腸，辛屬肺經，其音商，亦名傳曰金屬肺，內藏魄焉。陰世所謂魂升魄降，蓋陽升陰降之義。五行惟金無餘氣者，何也？金禀肅殺之氣，天地好生之德，不令肅殺之。自餘地亦就水生於亥，水生於申，申乃水長生之地，水能生木，此絕處逢生是也。木之餘氣化為紫蒸，火且以餘氣化為計都，故注於亥絕處。木長生於亥，火亦絕處逢生，火之餘氣化為羅睺。金生於巳，絕於寅，火長生於寅，火生於寅，是以金絕無餘氣也。此乃所以坐金無餘氣之理也。金乃懸針之象，凡金氣太甚，雖云富貴，未免剋子刑妻，親情蕭瑟，乃肅殺之致故也。凡從者，金落之任，從作方圓之體；革者，金造之任，作器皿首飾之具，故曰從革。其性怒，其情貪。天為五車，屬太白；在地為五岳，屬華山；在人為五行，屬金；在身為五臟，屬肺；在理為五常，屬義；在歲為秋；在方為西；在卦為乾；在天干為辛庚；在地支為申酉；在神為白虎；在色為白；在聲為商，故能切。律中夷則，屬殺司令，萬物收藏，天地之形殺也。

## 腎論

治腎用吹，則瀉，吸則補。腎屬水，左腎右命，前對臍，後著於脊，色如縞映紫。凡人至六十，腎氣衰，髮墮齒，氣衰腔脈空虛。人之有此七千，形體皆極，九十如樹之有根也。腎有兩隻，重三兩三。北方黑色，入通於腎，開竅於二陰，左腎為壬，右腎為癸，在形為骨，立久則傷陽。腎脈出於湧泉，兩腳心為湧泉穴。腎脈出於腎者，主蟄封福之本，精之財也。經于上崔，榮于中。

## 膽論

焦衛王卜佳，腎者作強之官，伎巧出焉。腎氣通於兩竅，為唾，唾者腎之液也。腎氣自入則多唾也。六腑膀胱為腎之腑，膀胱與腎合為津液府。五官耳為腎之官，淫聲聞於耳，則懸起而能搖精。聖人教人，非禮勿聽故也。腎氣通則耳聞五音，腎病則耳塞無聞，則腎萎也。腎合於骨，則髮也。腎萎不能起牀者，腎氣先死也。為北方坎水，黑者腎色也，如鴉羽者生，如黑蝶者死。味鹹，其性為入潤下變色也，其莫朽。心邪入腎則惡臭也。其性智，其事悲，水性智故悲泣也。腎之外應北岳，上通辰星之精。冬三月，存辰星於腎中，亦作黑氣存之也。腎神玄冥，字有頂，象如懸石，其神形如鹿兩頭，主智，其神和也。人之和順，而後全其生，則合太清也。腎合於骨，上主於齒，齒痛者腎虛也；又主於耳，人之不聞聲者，腎虛也；人之多呵欠者，腎邪也；久之腰不伸者，腎水竭也；人之色黑者，腎衰也；人之容色紫光者，腎無苦也；人之骨鳴者，腎羸也。夫腎之羽，故病當吹，其氣智能由腎之疾。凡人有積氣衝臟，則強吹；腎氣沈滯，則微通也。天腎主智，使人慧利，神彩殊朗，不滯於事，則固而形全也。且腎者，冬之用，當乾坤氣閉，萬物伏藏，戊寅辰起，凝沖寡欲，養其骨，以恬其神，逆之則傷腎。

## 膽論

治膽用噓，嘻為瀉，吸為補。膽附有肝，色如編青，重三兩三錢，為中青之府，號將軍，決曹吏，能喜剛而能柔和，精為騰蛇，能勇提。膽者，筋之應也，為火府之精，亦為中正之官，決斷出於此。膽神龍曜，字威間。膽雖甲木，實金之精，水之色，是子屬腑。膽者，水也，從陽主喜，泪自出焉爾。肝主怒，為清淨之府，在肝左邊第四葉，象如懸龜，其神形如龜蛇，合於膀胱，上通毛髮。人之勇決，終於膽臟。欲安其神，當息怒忿，窒怒而復理其生，則合太和也。大膽寄於姨宮，使人親智慕賢，屏邪去佞，絕奸防盜，方為直也。膽者生於金，金王武，故多勇，且抑之則吉。天膽者，乘陰之氣，稟金之用，故主愁，又主悲。悲者，金生於水，故目中墮泪。夫心主火，膽主水，亦主火，得水而滅，水得火而煎，陰陽交爭，水膽勝火，故泪從目出也。泪者，類也，因類而出，故曰類也。夫悲。
