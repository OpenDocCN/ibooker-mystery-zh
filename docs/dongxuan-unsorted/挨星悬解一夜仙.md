巽坤離向。巽坤離。山水龍神個個齊。乾坤坎中乾坤坎。一家骨月一家宜。卦分先艮添三合。兩個東分兩個西。此是玄空真妙訣。毋容濫授洩天機。

中陽子曰先天之巽在後天之坤。有未坤申三卦。先天之坤在後天之坎。有壬子癸三卦。先天之離在後天之震。有甲卯乙三卦。皆巽坤離三卦。大要也。其又重複曰巽坤離者非言坐山也。言水城也。言龍也。以先天之巽入後天艮。先天之坤入後天離。先天之離入後天之兌也。巽入艮。坤入離。入兌。而向即是甲癸申子未卯。坤壬乙也。然不但出脈皆巽坤離。而剝換之龍。亦不脫巽坤離也。又不僅言龍與向也。而水之來去城門。亦在巽坤離也。蓋後天之艮山坤向。則坤水要出先天之離。亦可出先天之坤也。後天之離山坎向，則坎水要出後天之巽，亦可出先天之巽也。後天之兌山震向，則震水要出先天之巽，亦可出後天之巽也。是龍亦巽坤離向，亦巽坤離局，亦巽坤離水，亦巽坤離。故曰山水龍神個個齊。此下元之局也。先天之乾在後天之離，有丙午丁三卦。先天之震在後天之艮，有丑艮寅三卦。先天之坎在後天之兌，有庚酉辛三卦。皆乾震坎三卦大要也。其文重複曰：乾震坎者，非言坐山也，言向也。言水城也。以先天之乾入後天之坎，先天之震入後天之坤，先天之坎入後天之震也。乾入坎，震入坤，坎入震，則向上是艮丙辛午酉丑庚寅丁也。不但出脈皆乾震坎也。而水之來在城門，亦在乾震坎也。蓋後天之坎山離向，則離水要出後天之乾，亦可出先天之震。後天之坤山艮向，則艮水要出先天之坎，亦可出後天之乾。是龍亦乾震後天之坤出兌向，則兌水要出後天之艮，亦可出先天之兌。

坎向亦乾震坎局亦乾震坎水亦乾震坎故曰一家骨肉一家宜此上元之局也先天之兑即後天之巽非艮龍不尊先天之艮即後天之乾非兌龍不贅其曰三合又曰兩個東分兩個西也以乾西卦內有申子辰一局巳酉丑一局故曰三合兩個西也巽分東卦有寅午戌一局亥卯未一局故曰三合兩個東也巽水當令則巽水要出先天之坤亦可出先天之艮乾水當令則乾水要出後天之離亦可出先天之兑亥山巳向則龍宜酉丑以扶之酉先天坎丑先天震辰向則龍宜申子以護之申先天巽子先天坤辰山戌向則龍宜寅午以補之寅先天離午先天乾亥向則龍宜卯未以佐之卯先天離未先天巽其水之出口則癸乾巽兩卦之生成正變合局此中元之局各有分屬者也

# 東卦十二山向

臨安于先師曰：東向西，西向東。山之水，在元龍。我今一語傳仙訣。三卦三元一卦通。

解：東者，東卦也。西者，西卦也。山者，論山水者，論水，各有挨星也。三卦者，三元中各有三卦，各有元論也。一卦通者，乃三元卦之中，只用一卦也。這一卦通，則三卦皆通矣。不知用一卦之理，則三卦皆不通，八卦亦不通矣。

又曰：地之吉凶，固在於龍。龍之純駁，全由於水。水之關係，禍福甚大。故有龍穴媳美，而水不合元者，葬之反凶。學者不可不知也。總要龍合向，向合水，水合三吉位。但合元合局，發無疑矣。須知天地與人，三元配合，本相連。山水干支相剝換，陰陽互變，東西全。如上元以白坎為統龍，若用天卦，必須龍向水三處，皆在天卦之內，為一卦通。

●三坎宫壬子癸三山凡上元甲子甲戌二十年正令也坎卦要乾气来龙入坎作坎山离向收离水谓之乾山乾向水流乾然壬子癸三山又有分别之用如戊龙入壬脉结穴者则作壬山丙向然要丙字上水来朝或正出戊或变出丑乃合拨星中四吉水法余皆犯出卦之病如乾亥来龙入子癸结穴者则作子山午向兼癸丁不可兼壬丙而犯阴阳差错之病如亥龙入癸脉结穴者堂前独见丁水则作癸丁向不可兼子午盖子午父母癸丁子息子息不可兼父母也水之来去亦同前论至于右犯壬兼亥左犯癸兼丑则已禄外尤不可用此坎宫卦三爻之分别也

●三坤宫未坤申三山凡上元甲申甲午二十年正令也坤山要震气来龙入坤作坤山艮向收艮水谓之震山震向水流震也然未坤申三山又有分别之用如甲龙入未脉结穴者则作未山丑向收丑水出庚入丙如外龍入坤結穴者作坤山艮向收艮水出兌出離乃合挨星四吉之水餘水皆犯出卦之病不惟不福而反招禍矣如乙龍入申脈結穴者作申山寅向收寅水不可兼坤艮蓋父母息毋容混淆也水之來去亦同前論至於若犯未兼丁左犯申兼庚則謂之祿外尤不可用此坤卦之分別也

●三震宮甲卯乙三山凡上元甲辰甲寅二十年旺令震山要坎氣來龍入震作震山兌向收兌水謂之坎山坎向水流坎而甲卯乙三山又有分別之用法如土龍入甲脈結穴者則作甲山庚向水出戌出丑不得作甲庚兼卯酉凡合挨星四吉之水餘水皆犯出卦之病不惟無福反招禍矣如癸龍入卯乙脈結穴者作卯酉兼乙辛水出艮出乾乃合挨星四吉之水亦不必太拘蓋挨星之用尤要口傳也如癸龍入乙脈結穴者則作乙山辛向收辛水出寅出亥蓋艮即先天震、所謂震山兌向還歸震也、餘水皆犯出卦之病。

三巽宮辰巽巳三山、凡中元甲子甲戌甲申三十年旺令也、巽山要艮氣來龍入巽、作巽山乾向、收乾水謂之艮山艮向水流艮地。

然辰巽巳三山又有分別之用法、如丑龍入辰脈結穴者、則作辰山戌向、收戌水出離出兌、勿兼乾向、蓋辰皆單山類、收不得乾水、不得混兼巽也、以辰為子息、巽為父母也、如艮龍入巽巳結穴者、則作巽乾兼巳亥、收乾亥水謂之乾來乾去一脈通也、餘水不合四吉皆曰屬出卦大凶、如寅龍入巳脈結穴者、則作巳山亥向、收亥水出離出戌吉、出兌亦吉、但乾乃子息之類、不可兼巽、更以向上之水獨在亥字、尤不能舍亥而就乾、致令水空也、經云零神裝在水上、正神裝在山上、非此之謂歟。

以上乃坎坤震巽東四卦十二山向水路龍脈同旺九十年、一定不易。

之成规天心之运之元窍也得来龙合法又得入脉合旺更得水路合局扦之立验。

# 東四卦山水挨星訣

坎宫貪狼管局而壬子癸三山又有貪巨祿三星存焉此中大有講究如太祖乾气龍脉剥换辛艮入壬脉一等結穴者則是主山丙向丙水朝揖合艮丙辛之局矣艮丙辛三位乃挨破軍也似與上元之星不合不知山起挨在山不在水故水从艮丙辛之破軍倒翻在山上起顺挨至丙上乃是巨門豈非仍是上元之星輪在水乎若夫合癸山向之局则龍是酉丑而合午酉丑之右弼起挨则文曲在向上若合癸山丁向之局则龍是酉寅庚而合寅庚丁之左輔挨起而祿存在向上

此坎宫三山之局也龍訣須知從本卦先天開元宫上来剥换

坤宫巨門管局而未坤申三山又有貪巨祿三星存焉此中大有講究如太祖震气起龙脉剥换午酉脉入未一二节结穴者必是未山丑向丑水朝揖合午酉丑之局夫午酉丑三位乃挨右弼倒於山上起逆挨至酉上乃是文曲亦同东四卦之运也岂非仍是东四卦内之吉星乎若坤山艮向之局则龙是丙辛而合艮丙辛之破军起挨则巨门在向上合申山寅向之局则龙是庚丁而合寅庚丁之左辅起挨则禄存在向上此坤宫三山之局也

震宫禄存管局申卯乙三山又有贪巨禄三星存焉此中大有讲究如太祖坎气来龙脉剥换寅丁入甲脉三节结穴者则作甲山庚向庚水朝揖合寅庚丁之局失夫寅庚丁之左辅是与上元之星不合不知山龙起挨在山不在水故从寅庚丁之左辅倒翻於山上起挨至庚上乃是禄存岂非仍是本元之星乎若合卯山酉向之局则龙是午丑而合午酉丑之右弼起挨而文曲在向上若合乙山辛向之局则龙是艮丙而合艮丙辛之破軍起挨而巨門在向上此震宮三山之局也

巽宮文曲管局而辰巽巳三山又有文武二星分局者均不必拘定如巽山乾向則取乾艮離之局蓋乾乃先天之艮離亦先天之乾也而取太祖艮龍入首者以龍亦艮位也所謂艮山艮向水流艮也合乾合離一氣先天後天相一家骨月也故古仙作巽山乾向收乾水者必有艮祖起龍轉離入巽一二節得四綠時三十年旺氣而向上血脈相朝不宜混祿見坎水乃巽山乾向煞也天辰山戌向須要艮祖轉寅午入辰一二節作辰戌向收戌水合寅午戌之局即是大格不必拘泥于文曲也至于巳山亥向亦必艮祖轉卯映到未入巳一二節結穴則作巳山亥向收亥水合卯未亥之局即是大格不必拘泥于武曲也

于仙曰東卦之巽西卦之乾一附上元一附下元附上元之巽乃是中元甲子甲戌甲申三十年旺氣同上元坎坤震一齊大發九十年故一二三四當令則向六七八九以配之六七八九當令則向一三四以配之若當令在山上則心神坐鎮而發福若當令在水上則零神在山反招禍。

# 西卦十二山向

三乾宮戌乾亥三山凡中元甲午甲辰甲寅三十年必令也乾山要元气來龍轉艮剝離入乾一三節結穴者作乾山巽向收巽水出坎出震當六白時大發所謂兌山兌向水流兌也如庚祖入戌脈結穴者作戌山辰向收辰水出坎出震若得兌祖換申峽轉子入戌一三節即結穴者則合甲子辰局若半局則不難也如辛祖出脈剝過換丑入亥一三節結穴者則立亥山巳向收巳水並無兩水來雜謂之巳酉丑全局此兌龍乾宮三山之用也

三兌宮庚酉辛三山凡下元甲子甲戌二十年必令也兌山要離气來龍入兌脈二三節結穴者，作兌山震向，收震水出坤，謂之離向水流離也。蓋震乃先天離，兌乃先天坎，來去之水，皆收離一局也。亦所謂後天來龍先天向也。出坤則合生成出巽，則合本身，緣巽即先天兌也。

當交九一齊發福，然三山又有分別妙用。如丙龍庚脈二三節結穴者，則作庚山甲向，須要甲字上水來，並無甲祿寅水出坤，兩合乃合催官之法。若水出坎，則發在六白之時；若出巽，則發在九紫之時。此丙龍入庚脈之局也。如丁龍入酉辛結穴者，則作酉外兼辛乙，不可兼庚，亦要卯乙字水來出坤，合七赤時催官。餘水皆犯出卦之病。如丁龍入辛脈二節結穴者，則作辛山乙向，收乙水出坤，不可兼酉，為一定不易之法。

蓋一元之中，原分三卦，三卦之中亦分三爻，而用這一卦，則有這一卦之法，用那一卦，則有那一卦之法，用各不同。其中龍穴向水迥別也。學者不可不察。

# 三 艮宫丑艮寅三山八下元甲申甲午二十年旺令也艮山要癸气来龙

入艮脉一二节结穴者作艮山坤向收坤水出震合催官出坎亦合四吉之水阴阳交媾所谓巽山巽向水流巽也然向上左不宜见离水右不宜见兑水如辰龙入丑脉结穴者作丑山未向收未水不宜禄丁向归甲合催官之法归土合四吉之水如巽巳龙入艮寅脉结穴者作艮坤兼寅申收坤水出震合催官或出坎合四吉如巳龙入寅脉一二节结穴者作寅山申向收申水出震出坎不宜禄兑离此和坤申三山之大要也八卦有一卦有一卦分用之妙所谓八卦只有一卦通也

# 三 离宫丙午丁三山八下元甲辰甲寅二十年旺令也离山要坤气来龙

入离作午山子向收坎水谓之坤山坤向水流坤也坤与坎一脉相通癸丙午丁三山又有分别之法如未龙入丙脉一二节结穴者作丙山壬向收壬水左不可兼午右不可兼巳而向上之水亦左不宜见亥右不宜見坎只見壬水曲折前來出辰合催官之法出未得陰陽交媾之妙出甲則不合挨星陰陽水法雖合元亦非全美如坤申龍入平丁脈三P結穴者則作平子兼丁癸收子癸水出巽已合催官出坤申合陰陽交媾亦為全美若出震不得法矣如申龍入丁脈結穴者則作丁山癸向收癸水出已合催官法出申亦陰陽交媾之理若出乙不合挨星水法況離宮三山皆不宜乾艮二水駁雜穿堂犯者九紫之時定損丁財學者慎之

# 西四山龍挨星訣

水在內

乾宮武曲管局而戊乾亥三山又有文武三星分局者不必板求如太祖兌龍剝艮換離入乾脈三P結穴者則是乾山巽向巽水朝揖合巽坤坎之局矣不必挨文亦不必挨武如先祖出脈剝申子入戊三P結穴者是戊山辰向辰水朝揖合甲子辰局不必挨文挨武如兌祖出脈剥換自丑轉亥二三節結穴者則作亥山巳向巳水相朝合巳酉丑局來必換文與武天戌乾亥三山縱要兌龍為祖用其剥換之局以立向不以戊兼辛以乾兼戌以亥兼乾

元旺气，向上挨武，又是西卦之令星，而龙穴向水，合上中下格，非徒在西卦九十年内发达，即历三元之久，亦不败矣。若夫西山卯向之局，则离祖剥换之脉，必是未子，而合子未卯之禄存起挨，而辅星在水上，合辛山乙向之局，则离祖剥换之脉，必是壬坤，而合坤壬乙之巨门起挨，而破军在水上，此兑宫之局也。

艮宫左辅管局，而丑艮寅三山，又有破辅弼三星存焉，此中大有讲究。如太祖巽气起脉，剥换子卯入丑脉，一二节结穴者，则是丑山未向，未水朝揖，合子未卯之局矣。夫子未卯三位，挨禄存也，似与下元之星不合，不知山龙挨星，不在乎水，故从子未卯之禄存起翻倒，于空起挨，逆行至未上，乃是左辅，岂非是本元本卦之令星，轮在水上乎。伊太祖与剥换之龙，皆上元旺气，而到山唇上，向之属辅，向水是未，并不禄丁，亦不禄癸，岂独下元发福，即历遍三元之久，亦不败矣。

合艮山坤向之局，则巽祖剥换之脉，必是壬乙，而符坤壬乙巨门起挨，则破军在水上，合寅山申向之局，巽祖剥换之脉，必是甲癸，合甲癸申之贪狼起挨，而武曲在向上，此艮宫三山之局也。

离宫右弼管局，而丙午丁三山，又有破辅弼三星存焉，此中大有讲究。如太祖坤气起龙，剥换中乙入丙脉，一二节结穴者，则是丙山壬向，壬水朝揖，合坤壬乙之局矣。夫坤壬乙三位，挨巨门也，似与下元之星不合，不知山龙起挨在山不在水，即从坤壬乙巨门翻倒于山上起挨，顺行至壬上，乃是破军，岂非仍是本卦元之令星乎。夫太祖与剥换之龙，虽非离宫本龙之旺，而入脉之丙向上三壬，得阴阳交媾之妙。

又无乾坤艮水禄，岂仅下元发福，即历经三元，亦不败矣。若夫午山子向之局，则坤祖剥换之脉，必是未卯，合于未卯之禄存起挨，则向上必是左辅，丁山癸向，则坤祖剥换之脉，必是甲申，而合甲癸申之贪狼起挨而武曲在水上此离宫三山之局也。

以上二十四山分东西两卦，东卦当令则入脉一二节必要东卦之龙乃谓之旺龙，而向水必要西卦之水乃为吉水，若东卦时混入东卦之水必败。

西卦当令则入脉一二节必要西卦之龙凡谓之旺龙，而向水亦要西卦之水乃为吉水，若西卦时混入西卦之水必败。此水不合而朝指明堂者，主有财无丁，异姓承桃。此山是山是水，毋容混也。

于先生曰：有西卦之时而葬东卦之山，亦发矣，似属不合，却喜穴前见东卦之水掩映天心，不见西卦之水来于何方去于何位，一逢东卦之时，未有不败者。有东卦之时而葬西卦之山，亦发矣，似属不合，却喜穴前见西卦之水融会堂中，不见东卦之水来于何方去于何位，一逢西卦之时，未有不败者。凡东山东水发于西，山西水发于东，人亦莫之辨，学者宜参看之。

如乾龙三丁入主脉结穴处。
