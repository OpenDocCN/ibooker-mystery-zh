- 2、邻屋

- 3、房高，外局的周边环境好时，房高出好。周边环境坏时，高出凶。
  例：西南5黄到，门口山星犯下水，房子高出很多，有三个烟囱。女儿被谋杀。越高出越不好。
- 4、寺庙，周围尽量不要住人，医院周围或对面不要住。如果在小区里看不到医院也无妨。衙门，不要正对
- 5、树，一楼二楼可以把树做山论，楼上则不论，具体要看比例。
- 6、车库，不能以车库为向，车库通常不论风水。以进屋的门为向，以主要的门为向，以宅的风水为主。
- 7、电梯，以动论。楼梯，以动论，不论上楼下楼的楼梯（指的是镂空的楼梯，透气的楼梯）
- 8、门，不管形状，方圆皆可，只论旺衰之气。
- 9、鞋柜，不要放在门与立极点之间，减弱了气。如果空间大，可以鞋柜上放盆景，盆景可引气。
- 10、门顶上不要有柜。不利气，工作压力大，卡气，钱被人管，可用葫芦化解。
- 11、电视机、冰箱、空调，不开时可做为山论，开时做水论。最好放山向二星都旺的方位。钟，动的厉害的才论水，否则仅是指针转动或玻璃里面关着，力量不大。
- 12、剑，只适合于公检法、武职、练武之人。
- 13、鹰，属蛇、鸡、鼠、兔等的人易受伤。也不要挂老虎的图片。
- 14、古董、旧物，来历不明的物品，有阴性信息。羊头、牛头、骨头，不要放。
- 15、挂画，不要有玻璃、镜子，反气。镜子当水论。（健身房等特殊行业例外）
- 16、神位、菩萨，做为山论，放在山星旺方。（向星旺方行不行???）观音菩萨不要放在卫生间、餐厅、夫妻房。
- 17、楼梯，做山论（指的是侧面有板或封死的楼梯）
- 18、假山，做山论，但忌奇形怪状，则为形煞。如果水多则论水，山多论山，山水差不多则山水都论。
