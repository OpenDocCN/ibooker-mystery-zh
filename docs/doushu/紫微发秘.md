# 自序

吾師黃耀德先生於今春舉辦紫微斗數講座，聘請張耀文先生爲講師，講述透露派紫微斗數秘奧。張耀文先生實爲斯界權威，非但立論正確，並且評斷如神。尤以閏月身宮一事，迎刃而解。講授內容，句句是金言，錄於筆記，條條爲玉律。如此至寶，不敢私藏，遂將口述筆錄，重新編成一書，公開於世。盼各位同好，因而有所獲焉。

### ○安身宫法

大抵人身，俱从寅上起正月，顺数至本生月止，又自人生月上起子时，顺数至紫微宫安身。假如正月生紫微在子，就在寅宫安身，紫微在丑，顺去卯安身，紫微在寅，顺去辰安身，余宫仿此。又若闰正月生者，仍在正月安身，凡有闰月，俱要依此为例。凡安命必用节，过立春方属寅，否则去年丑月算。安身则以初一为率，不必过节。盖命为太阳，身为太阴，理所当然。

例一：火局生於万历初九，故紫微在子。

例二：土局生於万历十三，故紫微在午。

例三：木局生於万历二十二，故紫微在亥。

## 定大限诀

凡推大限始行之岁数，但从所生之日起。阳年生男，阴年生女，则顺行，数至未来节。阴年生男，阳年生女，则逆行，数至已往节。皆过节而止，得足数三日，为一岁，三十日为十岁。余一日，作加多一百二十天算。少一日，作扣欠一百二十天算。不多不欠者，作整数论。第一运必为命宫，第二运则顺为父母，逆为兄弟，余做此。

## 定小限诀

- 寅午戌人辰上起一岁。
- 申子辰人戌上起一岁。
- 亥卯未人丑上起一岁。
- 巳酉丑人未上起一岁。

男顺行，女逆行。

# 紫微通变 命宫篇

## ◎紫微在命宫

紫微在子宫，独守庙旺，稍有富贵。炎铃、羊刃、空劫同、孤独刑伤。曲昌、天存、魁钺、左右同、富贵双全。

紫微在午宫，独守庙、鲤登龙门。炎铃同、大富小贵。曲昌同，侯门似海。羊刃同、快人快语。天存、魁钺、左右同、富贵双全。空劫同、有贵不富。

紫微在卯酉，贪狼同，囚色有患。炎铃、羊刃、空劫同、酒色败身。曲昌同，风流才子。

紫微在丑未，破军同，一生平安。空劫同，清寒低吏。

紫微在辰戌，天相相同，衣食丰足。炎铃同，辰凶戌吉。曲昌同，辰吉戌平。羊刃同，独立单行。左右同，富贵双全。空劫同，有贵不富。

紫微在寅申，天府同，富贵双全。炎铃同，寅吉申平。曲昌同，寅平申吉。陀罗同，富贵带刑。天存、魁钺、左右同、巨富大贵。空劫同，虽贵亦贫。

## ◎天机在命宫

天机在子宫，独守庙，机谋多变。炎铃同，子凶午吉。曲昌同，子吉午平。羊刃，空劫同，孤穷下局。天存、魁钺，左右同，财官贵显。

天机在卯酉，巨门同，富天贫寿。炎铃、羊刃、空劫同，平常之命。曲昌、天存、魁钺左右同，小商可营。

天机在丑未，独守失，性急多失。炎铃、羊刃，空劫同，贫苦之命。曲昌、魁钺、左右同，小商可营。

天机在辰戌，天梁同，机谋多变。炎钺同，辰平戌吉。曲昌同，辰吉戌平。羊刃同，性急多慈。左右同，富贵慈祥。空劫同，布施颇多。

天机在寅宫，太阴同，胥吏之命。炎铃、陀罗、空劫同，孤穷下局。曲昌同，怀才不遇。天存，魁钺，左右同，平常之命。

天机在申宫，太阴同，财官贵显。炎铃、陀罗、空劫同，有名无实。曲昌、魁钺、左右同，多学多能。天存同，官商供富。

天机在巳亥，独守失，因酒误事。炎铃、陀罗、空劫同，孤穷下局。曲昌、魁钺、左右同，多学多能。

## ◎太阳在命宫

太阳在子宫，独守失，先勤后懒。炎铃同，羊刃、空劫同，成败不一。曲昌，天存，魁钺，左右同，平常之命。

太阳在午宫，独守庙，文武皆贵。炎铃同，财官昭著。曲昌同，终南捷径。羊刃，空劫同，流於奢侈。天存，魁钺，左右同，巨富大贵。

太阳在卯宫，天梁同，荣华富贵。炎铃，羊刃，空劫同，浪费颇多。曲昌，天存，魁钺，左右同，文武皆宜。

太阳在酉宫，天梁同，飘蓬之客。炎铃，羊刃，空劫同，到处为家。曲昌，天存，魁钺，左右同，稍有名望。

命宮凶而兄弟吉者，兄弟無情，獨立成功。
