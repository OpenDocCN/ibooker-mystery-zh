# 引言

紫微斗数系以星辰推纳吉凶，大凡某种行为与灾祸，必有它的特殊星辰，产生一定的命数，因此在占算命运上，不能漫无条理地针对某星验论，而应将群星形性加以归纳分类，然后综合推理。

本书乃借鉴海外某派的观点，配以实例解说，准验度较高。惟深刻理解各星之星化力及相互交错时所产生的显象，在实践中反复体会，方能批断准确，否则易有“差之毫厘，失之千里”之误。

本书与诸书的不同之处在于：完全弃四化而不用，而引入中限的观念，着重以星化力来推断流年灾祸的各种类型。

> 【特注】：本书所述‘幽冥’、‘拘提’……等说法均为象征性的比喻，以给读者一个形象性的认识。特此说明。

- 一、中限：
人生富贵吉凶，有高有低，随运行走，不能固定在本命盘上推纳，须视行运而灵活通变，如此对人生浮沉才‘有法可依’。故不能根据先天命的格局优劣而铁口直断，须知命之穷通富贵系乎格局，而人生之顺逆起伏随运变化。所以孔子有陈蔡之困，关羽有麦城之厄，武穆有风波亭之惨死：……可见人命林林总总，先贵后贱、先贱后贵者有之，一生安逸享乐

每个问题，都有它的星力，脾气刚烈凶暴者容易起冲突，甚至产生凶杀命案，由于星群复杂的聚合，如果单就某一星辰推占吉凶，实难找出行爲轨迹，譬如‘为何打架’、‘闹事’、‘有无人命’……这些问题都不是传统的算命理论，能够清晰分辨，因此以紫微斗数探求灾祸，应该从星性加以归纳分类。

# 第三章 錢財損失
人生富貴貧賤，不能只在財帛宮上探討，往往財宮吉星聚會，卻兩手如貧，也有劣星糾纏，而富甲一方，這些例子不勝枚舉。

# 第十一章 牢狱、官司

法律，是社會安定鞏固歹徒犯罪，所設立的牢獄刑法。無可厚非，律師可以幫助沒有法律常識的人伸冤，但也形成某些刁民纏訟的案例。吾人期望的法律，是結束歹徒，而不是動則以牢獄刑囚加于人。時下占命，往往把『灾禍』歸諸四化中的『化忌』，這種占法欠缺全面性。事實紫微斗數，完全在于星化組合，本篇即以此研討『牢獄官司』的特殊星化現象。

-   (一)中限命身二宮碰上天刑、天魁、封誥。
-   (二)中限（大限）田宅宮遇到天刑、天魁、地劫、太陰。
-   (三)流年太歲與小限的命身宮見封誥、太陰。

中限命宮見天刑，主是非流連，干犯科法。田宅宮又見暗鬰星（地劫），化為官司囚牢。天刑，系一顆刑傷星，如果飛入行運、身二宮，主身體有毀傷，如刀傷、開刀手術、被虐待、……等，如果田宅宮又碰上暗鬰星（太陰、地劫、陰煞、封誥），則化為官司囚牢。天魁，主仇對之位，行運見之，有喪事、歹人作怪，或被刑警拘留刁難。封誥是最佳吉祥星，能給人崇高榮譽與品德，凡是品學兼優，豐功偉績，都是此星照示使然，然而它是一顆拘束星，人一到高官顯爵，成為模範學生，他的行為與生活環境也會拘限起來。因此它也屬於封閉星，如果行運與刑惡星相會『天刑、天魁』，此星變壞為（牢獄、刑囚），故不能以吉星論。

『地劫、太陽、陰煞』，人一入囚牢，他的生活空間多在黑暗監獄中渡過，是故行運命、身、田宅宮見到，要小心天刑、天魁是否飛入，若有官司、牢獄難免。
