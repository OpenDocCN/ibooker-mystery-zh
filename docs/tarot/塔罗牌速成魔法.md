

# 塔罗牌速成魔法


# 认识塔罗牌

塔罗占卜是非常灵验的预测工具，早在十四世纪时于欧洲已极为盛行，现今最广泛流传的塔罗牌是“伟特牌”。塔罗牌本身不会说话，了解牌义后然后做预测的，是人的“心”。透过塔罗牌的牌义和图像，人可以万念归一，发挥潜意识的能量，这时，人不但可预测未来，为未来作出最恰当的抉择；更可以与人心对话，了解自己的心理状态。

塔罗牌有逾百种占卜阵法，根据不同问卜者的不同问题而选择不同形式的牌阵。

## 揭开78张塔罗牌的神秘面纱

只要明白了各张牌的牌义和主要阵法后，就可以透过不同牌阵来解决不同的难题，并且达到一定的准确度。塔罗牌有分正牌和反牌，如果显示的伟特牌是向着大家，就是正牌，相反就是反牌。要注意：正向或反向的牌义会完全不同。“伟特牌”由78张牌所组成，包含22张主牌，40张副牌及16张宫廷牌：

- ◎二十二张主牌

22张主牌代表问卜者无法控制的部分。当牌阵中大牌出现的比率特别高时，表示问卜者对于这个问题是无能为力的。另外，大牌出现特别多，有时也代表着问卜者在生命中发生了一件影响较深远的重要事件。

二十二张主牌分别是：愚者、魔术师、祭司、皇后、皇帝、教皇、恋人、战车、力量、隐士、命运之轮、正义、弔人、死神、节制、恶魔、塔、星星、月亮、太阳、审判和世界

### ◎ 四十张副牌

40张副牌是补充22张主牌的细节，每副牌都代表着我们在生活上会面对的各种问题。

### 占卜时要注意的地方

占卜前须先洗手，洗牌与排牌阵时要默念问题，不断重复，直到开牌后才停止。以下是“占卜前”、“占卜时”和“占卜后”要注意的事项：

#### 占卜前

- 1. 留意身心状态
在生病、情绪低落、激动、生气或疲倦时，都应该避免占卜。原因是解牌需要耗费精神来思考，身心状况不佳时，会影响解牌的准确度。而且，选出来的牌常反映出问卜者当下的情绪，试想身心状况这么糟的时候，选出来的牌会好到哪里去呢？如果选到坏牌，心情岂不是更糟？如果无法平心静气，客观忠实地解牌，占卜结果的准确度亦会大打折扣。

- 2. 占卜时地的选择
占卜的地点以明亮、干净与舒适的环境为佳。在嘈吵和人流太多的环境下都会影响占卜者的专注力。此外，最好避开风大或潮湿的地方，以免影响心情，令占卜失准。

- 3. 至亲好友才能代人问卜
问卜的人要与当事人有密切的关系，换言之，母亲可以代替孩子问卜，但同事代替同事问卜则是不恰当的。

#### 占卜时

- 1. 不宜预测国家大事
占卜时，只能针对与自己有密切相关的人或事来占卜，不宜用于预测国家大事、球赛胜负等。换言之，如要占卜美国总统的竞选结果，必须找到候选人来抽牌，因为他们才是与问题切身相关的人物。

- 2. 发问的技巧
每次只问一条问题，不要连续问几条问题，如“我在公司的工作前景如何、与老板的关系以及和同事的关系如何？”这样我们难以判定塔罗牌所回应的是何者。

#### 占卜后

占卜后的善后工作要好小心。万一遗失了其中一张牌，就要重新买一套。所以一般玩家占卜完之后，都会用绒布包起，然后放在干燥阴凉的地方。

### ◎十六张宫廷牌

### 玩塔罗牌有什么禁忌

- 所谓的塔罗牌禁忌并非犯了就会遭到厄运，而只是为了提高准确度。在使用塔罗牌时切勿犯上以下错误：
◎不可以在同一天内占卜同一个问题。塔罗牌是我们的伙伴，我们必须给予信任。但可以转换问题形式发问，或用第二副塔罗牌再问。
◎不可以有“不算到好结果决不罢休”的心态，要尊重塔罗的结果，不可以一意孤行，塔罗要做的正是告诉我们真相，而不是按照我们自己的意愿说话。
◎虔诚的宗教信徒最好不要使用塔罗牌。因为塔罗会揭示神谕，这一禁忌或许与宗教信仰有冲突。
◎占卜开始时，应该注意自己的心情，太开心或太低落都不适宜占卜，占卜时应保持轻松的情绪，尽量找一个安静的地方开始。
◎如果时间紧迫，就不宜占卜，应另行约问时再做。
◎尽量避免发问一些关于生死的问题。帮别人占卜时，如果对方得到了“死神”这张牌，就得好好留意如何将它演绎，尝试以一个正面的态度解说。
◎不要过分依赖塔罗牌，例如有关医学诊断、法律诉讼或股票买卖等问题，就要找专人协助。
